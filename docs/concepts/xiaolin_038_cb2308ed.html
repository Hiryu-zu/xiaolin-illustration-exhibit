呼吸のイラスト

竹林の静かな呼吸

Calm bust-up portrait in bamboo light, useful as a standard style baseline.

注目ポイント

  • standard style
  • natural face lighting
  • outfit reference

記事用プロンプト

参考画像の人物、服装、髪型、体格を保ったまま、竹林の中で静かに呼吸を整えるバストアップを描く。激しい動きは入れず、肩の力が抜けた姿勢、柔らかい自然光、わずかに揺れる髪や布で、内側の気配を表現する。目立つ攻撃演出は避け、静から動へ移る直前の落ち着きを主役にする。背景の竹は穏やかに配置し、人物の表情と呼吸の間が伝わるようにする。高品質なアニメ調、柔らかい光、静かな表情、自然な空気感。

ネガティブ指定

低品質、崩れた人体、余分な指、不自然な手足、顔の歪み、左右非対称すぎる目、読みにくい構図、過剰な文字、ロゴ、透かし、背景の破綻、服装の大きな変更、参考画像と違う人物
