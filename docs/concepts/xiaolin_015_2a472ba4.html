Concepts / 画風イラスト

画風イラスト

竹林スタイル比較

Four-panel comparison of similar bamboo stances with different rendering density.

注目ポイント

  • same-pose style comparison
  • background handling
  • line and color density

個別プロンプト

参考画像の人物を使い、同じ竹林の構えを複数の描写密度で比較する。線の細かさ、色の濃さ、背景の描き込み、キャラクターの輪郭の強さを変え、どの画風が武術ポーズに合うかを見比べられる構成にする。ポーズは近い形に保ち、画風差が主役になるようにする。
