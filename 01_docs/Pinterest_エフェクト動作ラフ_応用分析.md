# Pinterest エフェクト・武術参考の応用分析

## 参照元

- エフェクト動作ラフ: `https://jp.pinterest.com/pin/367887863301978590/`
- 武術歩型参考: `https://jp.pinterest.com/pin/223843043966357869/`
- 確認日: 2026-06-17
- 利用方針: `01_docs/Pinterest_個人利用とダウンロード方針.md`

## ダウンロード可否

エフェクト動作ラフのピンでは、Pinterestの「その他のアクション」メニュー内に `画像をダウンロード` が表示された。  
そのため、技術的にはPinterestの公式UIから画像保存できる導線がある。

ただし、これは「Pinterest上でダウンロードボタンがある」という意味であり、著作権・二次利用・学習素材としての再配布許諾とは別問題。詳細は `Pinterest_個人利用とダウンロード方針.md` にまとめる。今回のワークフローでは、以下の扱いに留める。

- 正規UIで取得できる場合のみ、個人用の参考資料として保存する。
- 生成画像へそのまま複製させず、構図・流れ・エフェクト方向の観察メモに変換して使う。
- 成果物フォルダへ入れる場合は、出典URLと確認日を必ず残す。
- 外部公開・配布用の素材としては扱わない。

## エフェクト動作ラフの性質

メインのピンは、小さな人物ラフと風・砂・竜巻・飛翔・斬撃のような軌跡が組み合わさった資料。人体構造の正確なポーズ資料というより、以下を見るためのラフ資料として有効。

- エフェクトの流れ
- 画面内の移動方向
- 人物とエフェクトの大小関係
- 飛翔、回転、突進、砂塵、風圧のシルエット
- 1枚絵としての勢い、余白、視線誘導

小さな棒人間に近い人物が多いため、身体の左右、支持脚、重心、関節角度、顔向きは読み取りにくい。

## そのまま使えるか

### そのまま参考画像にしてよい用途

- エフェクト形状の参考
- 風、砂塵、竜巻、飛翔線の方向性の参考
- 技後の余韻や布・髪の流れの参考
- 戦闘カットのラフ構図参考

### そのまま使わない用途

- 小鈴の三方向GIF用マネキン本体
- 正面、左側面、右側面の厳密な比較
- 支持脚、接地、重心の判定
- 手足の左右指定
- 顔・衣装・髪型の一貫性確認

この資料を直接プロンプトへ入れると、派手なエフェクトだけが強く反映され、身体の姿勢が曖昧になりやすい。マネキン生成では、白い3Dマネキン資料を正本にして、このPinterest資料は「エフェクト層」だけに限定して使う。

## 関連ピンから拾える候補

確認時、周辺には以下の系統が表示されていた。

| 系統 | 採用度 | 使い道 |
|---|---:|---|
| 風・円・螺旋エフェクト | 高 | 円掌受け、旋身換歩、穿掌後の気流 |
| 直線風圧・飛翔線 | 高 | 前手直突き、後手直突き、横蹴りの打点方向 |
| 砂塵・竜巻 | 中 | 退歩、旋身換歩、地面を擦る移動感 |
| 火炎エフェクト | 低 | 小鈴の基礎動作とは相性が弱い。別キャラや敵技向け |
| 小型棒人間のアクションラフ | 中 | 勢いの参考にはなるが、人体正本にはしない |
| 武術歩型表 | 高 | 低い姿勢、接地、膝角度、体重移動の姿勢辞書 |
| 手型・拳型の表 | 中 | 穿掌、掌受け、拳の形の補助資料 |

## 武術歩型参考の使い方

別タブで開かれていた武術歩型ピンは、`mabu`、`gong bu`、`xu bu` などの低い姿勢がまとまっており、小鈴の基礎動作に使いやすい。

使える点:

- 支持脚と曲げ脚の差
- 膝を曲げたときの重心位置
- 低い構えの安定感
- 踏み込み、退歩、虚歩、弓歩の姿勢辞書
- 武術らしい手の構え

注意点:

- 写真は正面・斜め・側面が混在している。
- 衣装や人物の比率は小鈴に直接合わせない。
- 連続動作ではなく姿勢一覧なので、GIFの中割りは別途作る。
- 三方向GIFでは、同じ歩型を正面・左側面・右側面に再構成する必要がある。

## 小鈴モーションへの割り当て

| 動作 | 参考にする要素 |
|---|---|
| 前手直突き | 直線風圧、拳先から前方へ抜ける細い軌跡 |
| 後手直突き | 肩と腰の回転、打点後方から前方へ抜ける風 |
| 穿掌 | 指先から伸びる細い気流、袖の遅れ |
| 前蹴り | 足先の直線軌道、接地脚の安定 |
| 横蹴り | 横方向の風圧、蹴り脚を隠さないエフェクト |
| 円掌受け | C字または円弧の気流、手首から外へ回る軌跡 |
| 退歩 | 足元の小さな砂塵、後方へ薄く流れる線 |
| 旋身換歩 | 腰を中心にした螺旋、裾と髪の回転遅れ |

## プロンプトへ入れる変換ルール

Pinterest資料を使う場合は、プロンプトに以下の役割限定文を入れる。

```text
参照画像は人体ポーズの正本ではなく、エフェクトの流れ、画面内の移動方向、風や砂塵の軌跡、構図の勢いだけを参考にする。人体の体格、関節角度、支持脚、重心、顔と視線の向きは、別途指定する白い3Dマネキンを正本にする。棒人間やラフ内の曖昧な手足、左右、接地状態は引き継がない。エフェクトは人体の輪郭を流し、顔、指、手首、膝、足首、支持脚を隠さない。
```

## 判定

エフェクト動作ラフは、第一成果物の「基礎動作三方向GIF」には直接使わない。  
まずは白い3Dマネキンで正面・左側面・右側面の姿勢を揃え、GIFで身体の一貫性を確認する。  
その後、小鈴へキャラクター化し、必要な動作だけにPinterest由来の風・砂塵・螺旋エフェクトを薄く追加する。
