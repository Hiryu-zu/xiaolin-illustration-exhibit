# リーファン 戦闘構図設計

リーファンの戦闘イラストを、基礎武術から独自技へ段階的に展開するための設計資料。

## 正本ビジュアル

戦闘イラストでは、次の画像を最優先の外見参照とする。

- [リーファン 立ち絵](../../../../リーファン/リーファン　立ち絵.png)

顔、髪型、体格、衣装、装飾はこの立ち絵を基準とする。
旧全身立ち絵や既存戦闘画像と差異がある場合も、この立ち絵を優先する。

### 固定する外見

- 黄緑からオリーブ色の短い髪。
- 左右の控えめな中サイズの植物編み込みお団子。
- 両方のお団子に大きな桃色の蓮華。
- 黄緑から金色の瞳と、柔らかく可憐な顔立ち。
- 袖のないピンクのチャイナ風ドレス。
- 肩、上腕、前腕を覆う布袖は付けない。
- 両上腕に蓮華飾りを装着する。
- 上腕の蓮華から、細長い桃色のリボンが複数垂れる。
- 腕には細い蔦装飾を絡ませるが、筋肉や関節を隠さない。
- 金色の縁取り、チャイナボタン、腰横の結び飾りを維持する。
- 脚の蔦、足首の花飾り、桃色の靴を維持する。

### 袖とリボンの扱い

- 旧衣装の分離袖、長袖、フレア袖は使用しない。
- 腕の動きは素肌の肩、肘、手首で明確に見せる。
- リボンは袖の代わりに運動方向を示す補助線として使う。
- リボンの流れは、腕や身体の進行方向と逆側へ遅れてついてくる。
- リボンを腕、指、蔦、別の手と融合させない。
- リボンは一つの腕から2から4本程度に抑え、過密な線を避ける。

### 靴のプロンプト制御

リーファンは戦闘時も立ち絵と同じ桃色の靴を着用する。蹴り足を強調する構図でも裸足へ変更しない。

```text
Both feet wear the established closed-toe pink Chinese-style low-heeled shoes with gold ornaments.
Keep the shoes on both the supporting foot and the kicking foot.
The kicking heel, sole direction, and ankle angle must be expressed through the shoe silhouette.
No bare feet, no exposed toes, no missing shoe, no mismatched footwear.
```

- 「靴を履く」だけでなく、軸足と蹴り足の両方へ個別に指定する。
- 前景へ出す足にも靴を維持し、足裏や指を直接描写する指示を避ける。
- 前蹴では、靴底または靴の踵で前方を押す形にする。
- 横蹴では、靴の踵を接触点として、つま先を踵より後方へ収める。
- 靴は正本立ち絵の桃色、金色の飾り、低い踵、閉じたつま先を基準とする。
- ハイヒール、ブーツ、スニーカー、左右で異なる靴へ変更しない。

## 衣装と露出対策

戦闘動作では前掛けがめくれやすいため、正本の印象を保ちながら以下を固定する。

- 正面の前掛けは膝上から膝付近までの十分な長さを保つ。
- 前掛けを極端に短くした戦闘用ミニ衣装へ変更しない。
- 両脇のスリットは脚の可動を見せるが、骨盤付近まで過剰に開かない。
- ドレスの下に、衣装と同系色の不透明なフィットインナーを着用する。
- インナーは桃色または淡い桃色のショートパンツ型とし、下着として描かない。
- 蹴り、跳躍、ローアングルではインナーを必須とする。
- 前掛け、左右パネル、インナーの順序を固定し、布や脚を融合させない。
- 胸元、骨盤、臀部を強調するカメラ位置や性的な構図を避ける。
- 技の判読に必要な股関節、膝、軸足は、布で完全に隠さない。

## 戦闘表現の軸

- 太極拳を基礎にした、円運動と重心操作。
- 力を正面から受けず、流す、逸らす、巡らせる、崩す。
- 筋力で打ち抜くのではなく、足から掌までつながる全身運動。
- 基礎型ではエフェクトを弱くし、身体の動きを優先する。
- 光華拳では白から薄金、淡い翠色の圧縮された光を使う。
- 爆発、派手な粒子、過剰な破壊表現は避ける。

## 共通画像仕様

- `16:9` 横長。
- 均一な白背景。
- 敵、標的、人型の影を描かない。
- 全身をフレーム内に収める。
- 目線に近い高さを基本とし、広角歪みを抑える。
- 衣装の前掛け、裾、上腕の蓮華リボン、髪で運動方向を補助する。
- 左右のお団子、蓮華、チャイナ風衣装、蔦装飾を維持する。

## 遠近と画面設計

戦闘構図の遠近表現は、[牙焔 戦闘構図ガイド](../../牙焔/02_prompts/牙焔_戦闘構図ガイド.md)で有効だった技法を参照する。
ただし、牙焔の重く直線的な力ではなく、リーファンの円運動と柔らかな重心移動へ置き換える。

### 基礎型の遠近

- 四方向資料では、目線に近い高さと控えめな遠近を使う。
- 手足を極端に拡大せず、肩、股関節、膝との接続を優先する。
- 正面、側面、背面で人物の大きさとカメラ距離をなるべく揃える。
- 基礎型は構造確認用なので、前景物で身体を大きく隠さない。

### 独自技の遠近

- 人物を奥側または画面上側へ置き、掌、足、光の流れを前景へ伸ばす。
- 奥、小、中、大の順で段階的に拡大し、急な寸法変化を避ける。
- 手前の掌や足は画面中央から少し外し、顔を完全に隠さない。
- 技の軌道は一つの連続した円弧または緩いS字で示す。
- 光、花弁、リボンを同じ運動方向へ揃え、別々の流れを作らない。
- 前景の光や気流は画面端へ抜いてよい。全周の円を無理に閉じない。
- 画面外へ逃がすことで、長い気流や植物の蔓の分断、分岐、融合を防ぐ。

### 奥から手前へ迫る構図

```text
奥のリーファン
-> 肩または股関節
-> 前腕または大腿
-> 掌または足
-> 光華拳の流れ
-> 画面下または画面横へ抜ける
```

- 身体から技の先端まで、一本の力の経路として指定する。
- 突きでは `軸足 -> 骨盤 -> 背骨 -> 肩 -> 肘 -> 拳` を維持する。
- 掌法では `軸足 -> 丹田 -> 肩 -> 肘 -> 掌 -> 光` を維持する。
- 蹴りでは `軸足 -> 骨盤 -> 股関節 -> 膝 -> 踵` を維持する。
- 前景の部位だけが身体から切り離されて見えないよう、途中の関節を残す。

### 横方向と回転

- 人物を中央から少し外し、技が進む側へ余白を取る。
- 前掛けとリボンは、身体の回転より少し遅れて逆側へ流す。
- 腰、肩、腕、視線、光の円弧を同じ回転方向へ揃える。
- 太極拳らしい円運動は一つの大きな弧で見せ、複数の輪を重ねない。

### ローアングル

- 独自技、踏み込み、跳躍では使用可能。
- 基礎四方向資料では原則使用しない。
- ローアングル時も前掛けの長さと不透明インナーを維持する。
- 脚を前景へ出す場合、足裏よりも踵と膝の運動方向を優先する。
- 顔、蓮華、お団子を前景の手足やエフェクトで完全に隠さない。

## 四方向の定義

同じ技を、以下の四つのカメラ位置から生成する。

1. **正面**
   - 顔、両手、両脚の関係を確認する。
   - 技の意図と表情が最も伝わる主構図。
2. **左側面**
   - 重心移動、歩幅、腰と肩の回転を確認する。
   - 画面上でリーファンが右方向へ技を出す。
3. **右側面**
   - 左側面の単純反転ではなく、衣装、髪飾り、手足の接続を再確認する。
   - 画面上でリーファンが左方向へ技を出す。
4. **背面**
   - 肩甲骨、骨盤、支持脚、後ろ足の向きを確認する。
   - 顔は横顔が少し見える程度にし、完全な無表情の後頭部資料にしない。

## 第1段階 基礎型

### 01 弓歩直突

前足を曲げ、後足で地面を押しながら、腰の回転を拳へ伝える基本の突き。

**構図**

- 主構図は正面寄りの三四分視点。
- リーファンを画面中央より少し後方へ置き、突く拳の側に余白を取る。
- カメラは胸から腰の高さ。拳だけを過度に大きくしない。
- 前脚、骨盤、肩、拳を対角線上へ並べる。
- 引き手、後脚、上腕のリボンを拳と逆方向へ流し、螺旋を見せる。
- 四方向資料では遠近を抑え、独自技への発展版では拳を前景へ出す。

**静と動**

- `静 4 : 動 6`
- 脚と骨盤は安定した静、拳とリボンは前進する動として対比させる。
- 突進ではなく、重心が拳へ静かに乗り切った瞬間を描く。

**注意点**

- 牙焔のような直線的な全力突進にしない。
- 突く拳で顔や胸郭を完全に隠さない。
- 前脚の膝と足先を揃え、後ろ足から拳まで力をつなげる。
- 肩を上げず、肘を伸ばし切らず、手首を折らない。
- リボンが腕や拳から生えたように見えないよう、上腕の蓮華との接続を残す。

- 前脚の膝と足先を同じ方向へ向ける。
- 後ろ足の踵を浮かせすぎず、足裏から力を伝える。
- 突く拳の手首を前腕の延長上へ置く。
- 反対の拳は脇腹付近へ引き、肩を上げない。
- 牙焔のような直線的な突進ではなく、柔らかな螺旋を含む。

生成数: 正面、左側面、右側面、背面の4枚。

### 02 穿掌

開いた掌を前へ差し込み、接触点から相手の重心をずらす基本の掌打。

**構図**

- 主構図は正面斜め45度。掌を画面の左または右前景へ置く。
- 人物は反対側の奥へ配置し、掌から顔、胸、後ろ足まで一本の奥行きを作る。
- 掌は画面中央から外し、顔と五本指を同時に見せる。
- 反対の掌を丹田付近へ置き、前後の掌で緩いS字を作る。
- 淡い気流を使う場合は、後ろの掌から前の掌へ一本の曲線でつなぐ。

**静と動**

- `静 6 : 動 4`
- 差し込む速度より、接触直前の精密さと静かな圧力を見せる。
- 指先と視線だけに緊張を集め、衣装やリボンの流れは控えめにする。

**注意点**

- 掌を真正面へ向けた「停止」のジェスチャーにしない。
- 四本指と親指一本を明瞭にし、掌の周辺に花弁や光を重ねない。
- 肘を伸ばし切らず、肩から指先まで緩い曲線を維持する。
- 前景の掌を大きくしすぎて腕との接続を失わない。
- 反対の手を残像や第三の手に見せない。

- 指はそろえすぎず、自然に伸ばす。
- 四本指と親指一本の計五本。
- 掌は真正面ではなく、わずかに斜めへ向ける。
- 肘を伸ばし切らず、肩から掌まで緩い曲線を保つ。
- 反対の掌は胸または丹田付近へ置く。

生成数: 正面、左側面、右側面、背面の4枚。

### 03 前蹴

膝を先に引き上げ、足裏または踵で前方を押す基本の蹴り。

**構図**

- 主構図は側面寄りの三四分視点。
- リーファンを画面奥へ置き、蹴り足を画面横または斜め下の前景へ伸ばす。
- 踵、膝、股関節、軸足を一つの平面上で見せる。
- 両掌は胸前で中心線を守り、顔の周辺に小さな静の三角形を作る。
- 前掛けは蹴り足の反対側へ流し、股関節と軸足を隠さない。

**静と動**

- `静 4 : 動 6`
- 上体と両手は静、蹴り脚とリボンは動。
- 足を振り回すのではなく、踵で相手を押し戻す瞬間へ寄せる。

**注意点**

- 前蹴を横蹴や回し蹴りへ変化させない。
- 蹴り足のつま先を不自然に内側へ向けない。
- 軸足と蹴り足の両方に桃色の靴を維持し、足裏や足指を露出させない。
- 前景の接触面は裸足の足裏ではなく、靴底または靴の踵で示す。
- 軸足の膝を軽く緩め、身体の真下へ置く。
- ローアングルでは前掛けと不透明インナーを必ず維持する。
- 足裏を過度に巨大化せず、膝と股関節への接続を残す。

- 股関節、膝、足首を同じ進行面へ置く。
- 軸足の膝を軽く緩め、足先を極端に外へ向けない。
- 蹴り足の足首を不自然に内側へ反転させない。
- 上体を反らしすぎず、両手で中心線を守る。
- 衣装の裾で脚の接続部を隠さない。

生成数: 正面、左側面、右側面、背面の4枚。

### 04 横蹴

身体を横へ開き、踵を主導して押し出す基本の蹴り。

**構図**

- 主構図は蹴る方向の斜め前から見る横長構図。
- 人物を画面の片側へ寄せ、蹴り足を反対側へ水平に伸ばす。
- 軸足から骨盤、蹴り脚まで一本の長い横線を作る。
- 上体は蹴り足と逆側へわずかに傾け、両腕を前後に分ける。
- リボンと前掛けは回転方向へ大きな一弧を描く。

**静と動**

- `静 3 : 動 7`
- 基礎型の中では最も動寄り。
- 蹴り足の鋭い直線と、衣装・リボンの柔らかな円運動を対比する。

**注意点**

- 踵を技の進行方向へ向け、足裏を内側へ反転させない。
- 軸足と蹴り足の両方に桃色の靴を維持し、蹴り足だけ裸足にしない。
- 靴の輪郭で踵主導を示し、足指や裸足の足裏を描かない。
- 軸足を回し、膝と骨盤だけが取り残されないようにする。
- 蹴り足で顔や上体を隠さない。
- 前掛けやリボンを脚へ融合させない。
- 完全な水平開脚や不自然な柔軟ポーズにしない。

- 蹴る方向へ踵を向け、つま先を軽く下げる。
- 膝、踵、股関節を同じ高さの運動面へ置く。
- 軸足を回し、骨盤だけが逆方向へ残らないようにする。
- 外へ蹴る足の裏が内側へ反転しないことを重点確認する。
- 両腕は前後へ分けて均衡を取る。

生成数: 正面、左側面、右側面、背面の4枚。

## 第2段階 受けと歩法

基礎4型を確認した後、必要に応じて追加する。

### 05 円掌受け

片方の掌で円を描き、正面から来る力を外側へ流す。

**構図**

- 正面寄りの三四分視点で、受ける掌を顔の前方、反対の掌を腹部へ置く。
- 両腕で縦長の円を作り、その円の中心から顔を見せる。
- 片足へ重心を置き、もう片足は円を描く方向へ軽く開く。
- 淡い気流は腕の外側へ一本だけ沿わせる。

**静と動**

- `静 7 : 動 3`
- 相手の力が触れる直前の静けさを中心にする。
- 動きは掌、手首、リボンの小さな円に限定する。

**注意点**

- 腕を交差させすぎて手や肘を融合させない。
- 防御壁や硬いガードに見せず、外へ流す隙間を残す。
- 円形エフェクトを完全な輪として閉じない。
- 掌と手首の向きを明確にする。

### 06 双掌押

両掌を身体の中心から前へ押し出し、流れの圧力で重心を崩す。

**構図**

- 正面ローアングルを弱く使い、両掌を左右の前景へ配置する。
- 顔を両掌の間に見せ、掌、顔、後ろ足へ三角形の奥行きを作る。
- 両肘は完全に伸ばさず、肩幅より少し狭い間隔を保つ。
- 足元から両掌へ淡い光の流れを一本ずつではなく、一つの面としてつなぐ。

**静と動**

- `静 5 : 動 5`
- 押し出す動作と、腰を落とした安定感を同量で見せる。
- 爆発の瞬間ではなく、圧力が面として届く瞬間を描く。

**注意点**

- 両手を同じ形の複製にせず、遠近に応じた大きさと角度差を付ける。
- 指の本数と左右の親指位置を確認する。
- 胸元を強調する構図にせず、掌と重心を主役にする。
- 光で両手の輪郭を隠さない。

### 07 退歩引化

一歩下がりながら、両腕で力を引き込み、側方へ逃がす。

**構図**

- 横方向の構図で、身体を画面奥へ退かせる。
- 前方の掌から胸前、後方の掌へ緩いS字の動線を作る。
- 後ろ足を画面奥へ踏み、前足は接地したまま軽く残す。
- 前掛けとリボンを前方へ遅れて流し、後退方向を逆説的に示す。

**静と動**

- `静 5 : 動 5`
- 足は後退する動、上体と視線は相手を捉える静。
- 派手な速度ではなく、力が空振りするような滑らかさを重視する。

**注意点**

- 前進技に見えないよう、骨盤と後ろ足の移動方向を明確にする。
- 両掌を同じ高さへ置かず、奥行きを作る。
- S字の動線を複数の光帯に分裂させない。
- 足が交差する歩法にしない。

### 08 旋身換歩

軸足を中心に身体を回し、正面を入れ替える方向転換。

**構図**

- やや高めの三四分視点で、両足の配置と回転軸を見せる。
- 軸足を画面中央に置き、遊脚、骨盤、肩、腕を一つの螺旋へまとめる。
- 顔は振り返り、次に向く方向へ先行させる。
- 前掛けと両腕のリボンで大きな一弧を描く。

**静と動**

- `静 3 : 動 7`
- 軸足のみ静、身体と衣装全体は動。
- 回転途中の流れを見せ、完成姿勢で止めすぎない。

**注意点**

- 両足を同時に浮かせず、軸足を明確に接地させる。
- 骨盤と肩を逆方向へねじ切らない。
- リボン、光、衣装を異なる回転方向へ流さない。
- 円形エフェクトを何重にも重ねない。
## 左右差の扱い

四方向資料が完成した後、必要な技だけ左右の手足を入れ替えた差分を作る。

- 右突きと左突き。
- 右前蹴りと左前蹴り。
- 右横蹴りと左横蹴り。
- 右受けと左受け。

最初から全技を左右で倍増させず、四方向で身体構造を確認してから差分化する。

## 独自技への発展案

独自技は、基礎型を置き換えるのではなく、基礎動作へ光華拳の流れを加えて作る。

### 光華拳・穿華

基礎の穿掌から派生。
掌へ圧縮した光を集め、接触点から相手の重心と気の流れをずらす。

**構図**

- リーファンを画面奥の片側へ置き、穿掌の掌を斜め下の前景へ差し込む。
- 掌、顔、後ろ足を一本の対角線へ並べる。
- 掌の直前に小さく圧縮された白金の光点を置き、画面端へ細い波紋を抜く。
- 光は掌より大きくせず、触れた一点から流れが変わる表現にする。

**静と動**

- `静 7 : 動 3`
- 発光の派手さより、指先と視線の集中を主役にする。
- 静かな接触の中に、相手の流れだけが動く印象へ寄せる。

**注意点**

- 光弾やビームにしない。
- 掌と光点を分離させず、指先または掌根の直前へ置く。
- 前景の掌で顔を隠さない。
- 指の周辺へ花弁を重ねず、五本指を確認できるようにする。

### 光華拳・環流

円掌受けから派生。
受け流した力を翠色の円環へ乗せ、別方向へ巡らせる。

**構図**

- 正面寄りの三四分視点で、両腕を上下へ分けて大きな開いた円弧を作る。
- 人物を円弧の中心ではなく少し内側へずらし、顔と手の前後差を残す。
- 淡い翠色の気流は一周させず、片方の掌からもう片方の掌へ約4分の3周だけ描く。
- 気流の出口を画面横へ逃がし、受けた力が別方向へ流れることを示す。

**静と動**

- `静 6 : 動 4`
- 身体は静かに構え、気流だけが滑らかに巡る。
- 表情は穏やかだが集中している。

**注意点**

- 完全な魔法陣や防御バリアにしない。
- 気流を複数の円へ分裂させない。
- 両腕を円環へ融合させない。
- エフェクトで手首と指を隠さない。

### 光華拳・蓮衝

双掌押から派生。
両掌の間で光を圧縮し、蓮が開くように前方へ解放する。

**構図**

- 正面の弱いローアングル。
- リーファンを画面奥へ置き、両掌を左右の中景へ押し出す。
- 両掌の間に小さな閉じた光の蕾を置き、手前へ向かって薄い花弁状の圧力面を広げる。
- 花弁状の光は画面下または左右へ抜き、完全な巨大蓮を描かない。
- 顔は両掌の間から見せる。

**静と動**

- `静 4 : 動 6`
- 腰と脚の安定は静、開く光とリボンは動。
- 爆発ではなく、圧縮された面が静かに押し広がる瞬間へ寄せる。

**注意点**

- 両掌の指を光で隠さない。
- 光の蓮を別の生物や巨大な花の召喚にしない。
- 胸元よりも、両掌と丹田からの力の流れを主役にする。
- 手前の光を濃くしすぎて人物を消さない。

### 光華拳・返葉

退歩引化から派生。
後退しながら相手の力を引き込み、葉が風に返るように方向を反転させる。

**構図**

- 横長の側面寄り構図。
- リーファンを画面奥へ引き、前方の掌から背後の掌へ気流をS字に導く。
- S字の後半だけ向きを反転させ、画面上または横へ抜く。
- 数枚の葉を気流に沿わせるが、葉は軌道確認用に限定する。
- 前掛けとリボンは前方へ残り、身体の後退との時間差を見せる。

**静と動**

- `静 4 : 動 6`
- 身体は滑らかに後退し、気流は途中で向きを変える。
- 避ける弱さではなく、相手の力を支配する落ち着きを見せる。

**注意点**

- S字を二本以上作らない。
- 葉を大量に散らして気流を隠さない。
- 前後の手を残像や多腕に見せない。
- 後退足と前足を交差させない。

### 光華拳・旋脈

旋身換歩から派生。
身体の回転で足元と周囲の龍気を円環状に巡らせ、乱れた流れを整える。

**構図**

- やや高い三四分視点で、軸足と回転する全身を見せる。
- 軸足を中心に、前掛け、リボン、淡い翠色の気流を一つの螺旋へまとめる。
- 螺旋は足元から腰、肩、上げた掌へ向かって上昇する。
- 上端の気流は画面上へ抜き、閉じた円柱や竜巻にしない。
- 顔は次の正面方向へ振り返らせる。

**静と動**

- `静 2 : 動 8`
- 独自技の中で最も動寄り。
- 軸足と視線だけを静の核にし、その他を一方向へ巡らせる。

**注意点**

- 両足を浮かせず、軸足の接地を見せる。
- 気流、リボン、前掛けを同じ回転方向へ揃える。
- 竜巻や破壊的な風属性攻撃にしない。
- 螺旋エフェクトで腰、膝、足首を隠さない。

## 推奨制作順

1. 弓歩直突の四方向。
2. 穿掌の四方向。
3. 前蹴の四方向。
4. 横蹴の四方向。
5. 各画像の肩、手首、骨盤、膝、足首を比較。
6. 必要な型だけ左右差分を追加。
7. 受けと歩法を追加。
8. 基礎型を参照して独自技を生成。

## 生成後チェック

- [ ] 左右のお団子と蓮華が維持されているか。
- [ ] 髪が長いツインテールへ変化していないか。
- [ ] 布袖、分離袖、フレア袖が追加されていないか。
- [ ] 両上腕の蓮華と細長いリボンが維持されているか。
- [ ] リボンが腕、指、蔦、別の手へ融合していないか。
- [ ] 手は五本指で、親指の位置が掌の向きと一致するか。
- [ ] 突きの手首が折れていないか。
- [ ] 肩、肘、手首が一本の運動線としてつながっているか。
- [ ] 骨盤と胸郭のひねりが逆方向に破綻していないか。
- [ ] 膝と足先が同じ方向か。
- [ ] 蹴り足の踵、足裏、つま先が技の方向と一致するか。
- [ ] 軸足と蹴り足の両方が、正本と同じ桃色の靴を履いているか。
- [ ] 前景の足が裸足になり、足指や生身の足裏が描かれていないか。
- [ ] 軸足が接地し、重心が成立しているか。
- [ ] 前掛けが短くなりすぎていないか。
- [ ] スリットが骨盤付近まで過剰に開いていないか。
- [ ] 蹴り、跳躍、ローアングルで不透明な同色インナーが維持されているか。
- [ ] 前掛け、左右パネル、インナーの前後関係が成立しているか。
- [ ] 衣装の裾が股関節や脚の接続を完全に隠していないか。
- [ ] 光や花弁が手足の形を隠していないか。
- [ ] 基礎型ではエフェクトが主役になっていないか。
- [ ] 遠近を使う場合、前景の手足が身体から切断されたように見えないか。
- [ ] 前景部位やエフェクトが顔を完全に隠していないか。
