# 小鈴 マネキンアクションシート 動作体系

## 目的

`03_named_images/小鈴/01_mannequin_body/小鈴_旧マネキン基礎動作8種シート_参考01.png` のレイアウトを参考に、小鈴用の白い3Dマネキンで基本動作を体系的に作る。

完成キャラクターを描く前に、支持脚、重心、骨盤、胸郭、手足の向き、動きのつながりを確認するための資料とする。

## 参考画像から採用する要素

- 16:9の横長シート。
- 上段4パネル、下段4パネル。
- 左上から右へ01から04、下段左から右へ05から08。
- 全パネルで同じマネキン、体格、縮尺、床面、照明。
- 薄いグレー背景、弱い接地影、柔らかなスタジオ照明。
- 人物を頭頂から足先まで収める。
- 各パネル左上に小さな番号だけを置く。

## 参考画像から変更する要素

- 男性体型ではなく、小鈴用の21歳成人女性型マネキンにする。
- 7.3頭身前後、身長165cm相当、しなやかな武術体型にする。
- 強いボディビルダー筋肉や男性的な肩幅を使わない。
- 一覧だけでなく、一つの動きを8段階で追う連続動作シートを主力にする。
- 動作の連続性を確認するシートでは、カメラ方向を途中で変えない。

## シートの種類

### 1. 動作一覧シート

異なる8動作を一枚に並べ、作るべき資料の索引として使う。

**動作一覧シートは非連続であり、GIF作成、歩行ループ、キャラクター連番の入力に使用しない。**

|番号|基本移動一覧|打撃一覧|
|---|---|---|
|01|自然立ち・待機|基本ガード|
|02|歩行|前手直突き|
|03|早歩き|後手直突き|
|04|走行|掌底・穿掌|
|05|急停止|肘打ち|
|06|方向転換|膝蹴り|
|07|横移動|前蹴り|
|08|後退|横蹴り|

一覧シートはポーズ辞書であり、GIFの連続フレームには使わない。IDには `CAT` を付け、連続動作の `SEQ` と区別する。

### 2. 連続動作シート

一つの動作を01から08まで時系列で並べる。

- 全フレームで同じカメラ方向、同じアイレベル、同じ人物サイズを維持する。
- 床面上の移動距離をパネルごとに自然につなげる。
- 直前フレームから突然手足が入れ替わらないようにする。
- 歩行と走行は08から01へ自然につながるループにする。
- 突きと蹴りは01構えから08到達・余韻までを作る。
- 突きと蹴りの戻りは別生成せず、`08 -> 07 -> ... -> 01` の逆順再生で確認する。

IDには `SEQ` を付ける。GIFへ使用できるのは連続動作シートだけとする。

### GIF入力ゲート

- ファイル名または資料名に `一覧`、`カタログ`、`CAT` がある場合は使用禁止。
- 一つの動作だけで構成されていること。
- 全フレームでカメラ方向と進行方向が同じこと。
- 歩行では左右の接地、荷重、通過、持ち上がりが交互に存在すること。
- 足元基準線と骨盤中心を揃えてからGIF化すること。

## 第一優先で作る連続動作

|ID|動作|主な確認点|再生方法|
|---|---|---|---|
|L01|歩く|接地、重心移動、腕振り、骨盤の上下|01から08をループ|
|L02|走る|接地、沈み込み、蹴り出し、滞空|01から08をループ|
|S01|前手直突き|後足から拳までの力線、守り手|01から08、戻りは逆順|
|S02|前蹴り|膝上げ、支持脚、伸展、踵または靴底|01から08、戻りは逆順|
|S03|横蹴り|軸足回転、骨盤、膝、踵・足刀|01から08、戻りは逆順|

## 第二優先で追加する動作

- 自然立ちから構えへ移る。
- 早歩き。
- 全力疾走。
- 急停止。
- 90度方向転換。
- 前進、後退、左右のサイドステップ。
- 後手直突き。
- 掌底、穿掌、円掌受け。
- フック、肘打ち、膝蹴り。
- 回し蹴り、後ろ蹴り。
- しゃがむ、跳ぶ、着地する。
- 転倒、受け身、起き上がる。

## 歩行8フレーム

右向きの完全な側面カメラを基本とする。

|番号|段階|姿勢|
|---|---|---|
|01|左足接地|左踵が前方へ接地、右足は後方、右腕が前|
|02|荷重|左脚へ体重が移り、身体が少し沈む|
|03|通過|右足が支持脚の横を通り、身体が真上に乗る|
|04|持ち上がり|左脚で床を押し、右膝が前へ進む|
|05|右足接地|01の左右反転。右踵が前方へ接地|
|06|荷重|右脚へ体重が移り、身体が少し沈む|
|07|通過|左足が支持脚の横を通る|
|08|持ち上がり|右脚で床を押し、左膝が前へ進み01へ戻る|

## 走行8フレーム

右向きの完全な側面カメラを基本とする。

|番号|段階|姿勢|
|---|---|---|
|01|左足接地|左足が身体のほぼ真下へ接地、右脚は後方|
|02|吸収|左膝と足首が曲がり、重心が少し沈む|
|03|蹴り出し|左脚が床を押し、右膝が前へ出る|
|04|滞空|両足が床から離れ、右脚が前へ移る|
|05|右足接地|01の左右反転|
|06|吸収|右脚で衝撃を受ける|
|07|蹴り出し|右脚で床を押し、左膝が前へ出る|
|08|滞空|両足が床から離れ、左脚が前へ移り01へ戻る|

## 前手直突き8フレーム

左脚前、右脚後ろの半身構え。左前手で突く。

|番号|段階|姿勢|
|---|---|---|
|01|構え|両手で中心線を守る|
|02|始動|後足で床を押し、重心が前へ動き始める|
|03|骨盤移動|骨盤と胸郭がわずかに前へ入る|
|04|肘の前進|左肘が拳の後ろを通り、腕が半分伸びる|
|05|伸展|左拳が相手の中心へ進み、右手は守りに残る|
|06|到達直前|肩、肘、拳が一直線に近づく|
|07|到達|拳が胸から顔の高さへ届き、手首はまっすぐ|
|08|余韻|身体全体の力線を保ち、肘は過伸展させない|

## 前蹴り8フレーム

右脚を支持脚、左脚を蹴り脚とする。

|番号|段階|姿勢|
|---|---|---|
|01|構え|両手で中心線を守る|
|02|荷重移動|重心を右支持脚へ移す|
|03|膝上げ開始|左踵が床を離れ、左膝が前へ出る|
|04|膝上げ|左膝を高く引き上げる|
|05|伸展開始|左膝を支点に下腿が前へ出る|
|06|到達直前|踵または靴底が標的方向へ向く|
|07|到達|左脚を前へ伸ばし、右支持脚を接地|
|08|余韻|蹴りの方向と全身のバランスを保つ|

## 横蹴り8フレーム

右脚を支持脚、左脚を蹴り脚とする。

|番号|段階|姿勢|
|---|---|---|
|01|構え|半身で両手を守りに置く|
|02|荷重移動|重心を右脚へ移す|
|03|膝上げ|左膝を胸側へたたむ|
|04|軸足回転|右足を回し、骨盤を横へ開く|
|05|伸展開始|左踵を横方向へ押し出す|
|06|到達直前|左膝、踵、標的方向をそろえる|
|07|到達|踵または足刀で横へ蹴る|
|08|余韻|支持脚上に重心を残し、上体と両手で均衡を取る|

## 品質チェック

- [ ] 全8体が同じ成人女性型マネキンか。
- [ ] 頭身、肩幅、骨盤、手足の長さが全フレームで同じか。
- [ ] カメラ方向、人物サイズ、床の高さが固定されているか。
- [ ] 前後フレームで手足が突然入れ替わっていないか。
- [ ] 支持脚と接地影が一致しているか。
- [ ] 骨盤と胸郭の動きが手足の動作につながっているか。
- [ ] 歩行と走行は08から01へ自然にループするか。
- [ ] 突きと蹴りは01から08へ力が増しているか。
- [ ] 手足がパネル境界で切れていないか。
- [ ] 髪、衣装、色、文字説明が混入していないか。
