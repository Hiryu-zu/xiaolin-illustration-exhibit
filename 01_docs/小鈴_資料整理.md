# 小鈴 資料整理

## 目的

小鈴（シャオリン）の立ち絵・設定・プロンプトを、マネキン経由の戦闘イラスト制作へ使える形に整理する。

このキャラクターでは、最初から派手な完成戦闘絵を狙わず、以下の順で一貫性を作る。

1. 正本立ち絵から外見の固定要素を抽出する。
2. 顔角度リファレンスを作る。
3. 体格マネキン四面図を作る。
4. 基本構えを正面・左側面・右側面で作る。
5. 基礎動作を正面・左側面・右側面の3方向連番で作る。
6. 採用した動作だけを小鈴へキャラクター化する。
7. 必要に応じて固有技、表情、エフェクトを追加する。

## 小鈴の固定コア

- 21歳の女性。
- 身長165cm、7.3頭身前後。
- しなやかで均整の取れた健康的な武術体型。
- 黒髪ショートボブ。あご下から肩に付かない程度。
- 前髪あり。両サイドに顔を縁取る毛束。
- 琥珀色から黄褐色の瞳。
- 可愛さと凛々しさの中間の顔立ち。
- 普段はやや鋭い目つき。
- 白黒ベースの中華武術服。
- 凛々しく清潔感があるが、話すと気さくな雰囲気。

## 画像から固定する視覚要素

正本立ち絵候補から、以下を優先して固定する。

- 白い立ち襟トップス。
- 前中心に黒いチャイナボタンまたは組紐留め。
- 黒い縁取り。
- 肩から脇、体側へ入る黒いサイドパネル。
- 七分袖から長袖寄りの袖。袖口は黒い縁取り。
- 腰から前へ垂れる白い前垂れ。裾に黒い縁取り。
- 左右に短い黒い腰布またはサイドパネル。
- 黒いレギンス。
- 白黒配色の中華武術シューズ。
- 金色装飾は少量のみ。主役にしない。

## 生成時に崩れやすい点

- 白黒衣装がメイド服、制服、ドレス、現代スポーツウェアへ寄る。
- 前垂れがスカート化し、股関節や脚の前後関係を隠す。
- レギンスが素足やタイツ風の曖昧な脚へ変わる。
- 靴が裸足、サンダル、ハイヒールへ変わる。
- 鋭い目つきが怒りすぎ、冷酷すぎ、または幼い丸目へ寄る。
- ショートボブがロングヘア、ツインテール、ぱっつん姫カットへ変わる。
- 戦闘動作で袖や前垂れが関節を隠しすぎる。

## 小鈴向けの制作順

### 1. 正本立ち絵の確定

現在の `小鈴_正本立ち絵候補_01.png` は設定との一致度が高い。

追加で必要になる可能性が高い資料:

- 正面直立の全身立ち絵。
- 背面立ち絵。
- 左右側面、または四面図。
- 表情差分。
- 靴と袖口の拡大資料。

### 2. 顔角度リファレンス

戦闘ポーズで顔がカメラ目線へ戻ったり、髪型や目つきが別人化したりするのを防ぐため、5×5顔角度シートを作る。

使用資料:

- `02_prompts/小鈴_顔角度表情シートプロンプト.md`
- `C:\ぐるぐる\tomari-guruguru\docs\01_画像生成用テンプレ.png`

作る候補:

- 通常の鋭い真顔。
- 戦闘集中顔。
- 軽い笑顔。
- 目閉じ。
- 照れ顔。

### 3. 体格マネキン四面図

衣装や髪を外し、小鈴の身体比率だけを白い3Dデッサン人形へ抽出する。

参考にする観察メモ:

- `01_docs/Kindleアクションポーズ資料_マネキン応用メモ.md`

確認点:

- 7.3頭身前後か。
- 肩幅は標準からやや華奢か。
- 腰、骨盤、脚のラインが自然か。
- 鍛えられているが筋肉質すぎないか。
- 手足の長さが正本立ち絵候補と近いか。

### 4. 基本構え三方向

小鈴の基準姿勢は、軽やかな中華武術の半身構えとする。

- 左脚前、右脚後ろを基本。
- 拳よりも掌、手刀、受け流しへ展開しやすい構え。
- 低すぎる重量級構えではなく、足運びの軽さを残す。
- 顔と視線は相手方向へ向ける。
- 正面、左側面、右側面で同じアイレベル、床面、足幅、手の高さを維持する。
- 斜め方向や背面は、第一成果物では必須にしない。

### 5. 実戦基礎動作

第一成果物では、以下の8種類を推奨する。

|番号|動作|目的|
|---|---|---|
|01|前手直突き|最も基本の直線攻撃|
|02|後手直突き|後ろ足から拳への力線確認|
|03|穿掌|小鈴らしい掌技の核|
|04|前蹴り|支持脚、膝上げ、上体バランスの確認|
|05|横蹴り|脚線、股関節、足刀の確認|
|06|円掌受け|中華武術らしい受け流し|
|07|退歩|後退重心の確認|
|08|旋身換歩|回転と足位置入れ替えの確認|

リーファンの基礎動作セットと相性がよいが、小鈴では「三方向の連番」と「GIF往復確認」を優先する。

専用資料:

- `01_docs/小鈴_成果物目標_基礎動作三方向GIF.md`
- `02_prompts/小鈴_基礎動作三方向GIF_マネキンプロンプト.md`

Kindleアクションポーズ資料からの追加方針:

- 前蹴りは、支持脚へ重心を落とし、上体を蹴り方向と反対へ少し逃がす。
- 横蹴りは、軸足、骨盤、蹴り脚、腕で身体全体の釣り合いを取る。
- 直突きは、後ろ足から拳までの力線を明確にする。
- 回転動作は、軸足、骨盤、胸郭、肩、顔と視線を同じ回転方向へつなげる。
- 遠近がある場合は、アイレベルと消失点を意識して手足の前後差を作る。

### マネキンアクションシート

旧マネキン基礎動作8種シートの4列×2段レイアウトを利用し、基本移動と打撃を体系化する。

- 動作一覧シート: 異なる8動作を並べるポーズ辞書。
- 連続動作シート: 一つの動作を8段階で追うモーション資料。
- 第一優先: 歩行、走行、前手直突き、前蹴り、横蹴り。
- 歩行と走行は順方向ループ。
- 突きと蹴りは構えから到達までを生成し、戻りは逆順再生する。
- 旧画像の男性体型は使わず、小鈴用の成人女性型7.3頭身マネキンへ統一する。
- `CAT-MOVE-01 / C01基本移動一覧` は異なる8動作のポーズ辞書であり、歩行GIFには使用しない。
- 2026-06-21作成の `02 -> 03 -> 04 -> 08` GIFは、歩行、早歩き、走行、後退が混在するため不採用。
- 歩行GIFは `SEQ-WALK-01 / L01 歩行8フレーム` をマネキン段階から新規生成する。
- SEQ-WALK-01候補01は半周期の形は改善したが、左右脚と左右腕の交換を確認できず不採用。
- 次回は左手足に青リング、右手足に赤リングを付けた `SEQ-WALK-01-DIAG` を先に作る。
- `SEQ-WALK-01-DIAG` 候補01を前半01から04、後半05から08に分けて生成し、4列×2段の診断シートとGIFへ結合済み。

専用資料:

- `01_docs/小鈴_マネキンアクションシート_動作体系.md`
- `01_docs/小鈴_C01基本移動シート_歩行GIF失敗評価.md`
- `01_docs/小鈴_SEQ-WALK-01_マネキン候補01レビュー.md`
- `02_prompts/小鈴_マネキンアクションシート_基本動作プロンプト.md`

添付8ポーズマネキン画像の扱い:

- 01は正面ガード構え、02は斜め正面の直突き、03は側面の掌打または穿掌、04は背面寄りの突き、05は背面ガード、06は膝上げ、07は側面の前蹴り、08は斜め正面の横蹴りとして読む。
- 男性マネキンの筋肉量や体型は引き継がず、姿勢、カメラ方向、重心、手足の向きだけを使う。
- 三方向GIFの連続モーション正本ではなく、単発ポーズや一枚絵の構図補助として使う。
- 専用資料: `01_docs/小鈴_添付8ポーズマネキン_シャオリン化メモ.md`
- 専用プロンプト: `02_prompts/小鈴_添付8ポーズマネキン_シャオリン化プロンプト.md`

GIF確認の基本フレーム順:

```text
01構え -> 02途中 -> 03到達 -> 02途中 -> 01構え
```

引き戻し動作は別生成しない。
戻りは逆再生のようにフレームを再利用して確認する。

## キャラクター化の基本方針

参考画像の役割を明確に分ける。

- 正本立ち絵: 顔、髪、目、衣装、靴、配色の正本。
- 顔角度シート: 顔の向き、目線、髪型の角度変化の正本。
- マネキン: 姿勢、支持脚、重心、関節、カメラ方向の正本。
- 文章プロンプト: 技の意味、視線、衣装の揺れ、禁止事項を補う。

完成イラストは一枚ずつ生成する。

- 一枚につき小鈴一人、一つの動作だけ。
- グリッド、分割パネル、一覧画像にしない。
- 白背景を基本にする。
- 敵、武器、派手な背景は最初は入れない。

## 一枚絵成功要素

`GPT成功イラスト` と `ChatGPT-小鈴一枚絵プロンプト.md` から、完成一枚絵に必要な要素を別資料へ抽出した。

専用資料:

- `01_docs/小鈴_一枚絵成功要素_抽出.md`
- `02_prompts/小鈴_一枚絵生成プロンプト.md`
- `05_reference_paths/xiaoling_success_sheet.jpg`

一枚絵では、マネキン生成よりも以下を強く見る。

- 小鈴一人、一つの動作、一つの瞬間に絞る。
- 黒髪ショートボブ、琥珀色の瞳、白黒中華武術服、黒レギンス、白黒武術シューズを維持する。
- 道場、武館、石畳、夕暮れの稽古場など、武術の文脈が伝わる背景を使う。
- アクションでは、支持脚、骨盤、胸郭、肩、腕または蹴り脚までの力の流れを見せる。
- スピード線、風圧、砂埃、衣装や髪のなびきは、技の方向を補助するために使う。
- 掌底の一人称構図では、敵キャラを画面内に出さず、肩越し構図、第三者視点、観察者視点を避ける。
- 手や足の遠近感は迫力を出すために使うが、成人女性として自然な大きさに抑える。

比率の目安:

|用途|推奨比率|
|---|---|
|note見出し|1.91:1|
|本文用シーン絵|16:9|
|日常ポートレート|4:5|
|全身立ち絵|2:3|
|戦闘アクション|16:9 または 3:2|

## 小鈴用チェックリスト

### 外見

- [ ] 黒髪ショートボブか。
- [ ] 琥珀色から黄褐色の瞳か。
- [ ] やや鋭い目つきだが、冷酷すぎないか。
- [ ] 白い立ち襟トップスと黒い縁取りが残っているか。
- [ ] 黒いサイドパネルが残っているか。
- [ ] 黒レギンスか。
- [ ] 両足とも白黒の中華武術シューズか。

### 動作

- [ ] 支持脚と重心が成立しているか。
- [ ] 胸郭と骨盤の向きが動作と一致しているか。
- [ ] 顔と視線が攻撃または受け流し方向を向いているか。
- [ ] 手は五本指で、掌や拳の向きが自然か。
- [ ] 股関節、膝、足首、靴の方向が自然につながっているか。
- [ ] 前垂れや腰布が関節を完全に隠していないか。

### 保存判定

- **採用**: 外見と動作が両方成立。
- **条件付き採用**: 小さな視線、指、布の問題だけ。
- **再生成**: 別キャラ化、衣装崩れ、重心破綻、動作違い。
