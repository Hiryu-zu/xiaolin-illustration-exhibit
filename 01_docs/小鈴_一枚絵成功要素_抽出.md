# 小鈴 一枚絵成功要素 抽出

## 目的

`GPT成功イラスト` と `ChatGPT-小鈴一枚絵プロンプト.md` から、一枚絵イラストで効いた要素を抽出し、今後の完成イラスト生成に再利用できる形へ整理する。

この資料は、三方向GIF用のマネキン生成ではなく、採用済みモーションを小鈴の完成イラストへ変換するときの判断基準として使う。

## 成功画像から見える固定要素

- 黒髪ショートボブ、前髪、顔まわりのサイドロックを維持する。
- 琥珀色から黄褐色の瞳。目つきは鋭いが、冷たすぎない。
- 白黒の中華武術服を主役にする。白い立ち襟トップス、黒い縁取り、黒いチャイナボタン、黒いサイドパネルを残す。
- 黒レギンスと白黒の中華武術シューズを維持する。
- 体型はしなやかな武術体型。細すぎず、筋肉を強調しすぎない。
- 背景は道場、石畳の稽古場、夕暮れの武館など、武術の文脈が伝わる場所が合う。
- 完成絵では、衣装の裾、髪、袖、砂埃、流線で動きの方向を補うと読みやすい。

## 一枚絵で必要な基本要素

### キャラクター固定

- 21歳の女性。
- 身長165cm、7.3頭身前後。
- しなやかで均整の取れた健康的な武術体型。
- 黒髪ショートボブ、前髪あり、顔を縁取るサイドロック。
- 琥珀色から黄褐色の瞳。
- 可愛さと凛々しさの中間の顔立ち。
- 普段はやや鋭い目つきだが、怒りすぎない。
- 白黒ベースの中華武術服。

### 画風

- アニメ調、ライトノベル表紙風、精密なデジタルイラスト。
- 写実、3D、実写、厚すぎる油彩、極端な劇画には寄せない。
- 手だけ、足だけが過度にリアルにならないよう、顔と同じ絵柄に揃える。

### 構図

- 一枚につき小鈴一人、一つの動作に絞る。
- グリッド、分割パネル、連続写真、説明図にしない。
- 動作の核となる手足、顔、重心が一目で分かる構図にする。
- アクションでは、支持脚、骨盤、胸郭、肩、腕または蹴り脚までの力の流れを見せる。
- 背景は補助にし、主役は小鈴の姿勢、表情、技の軌道にする。

## 比率ルール

|用途|推奨比率|使い方|
|---|---|---|
|note見出し|1.91:1|顔と動作の核を中央寄せにする|
|本文用のシーン絵|16:9|道場、夕景、背景込みの完成絵|
|日常ポートレート|4:5|半身、立ち絵、静かな表情|
|全身立ち絵|2:3|正本確認、ポスター風|
|戦闘アクション|16:9 または 3:2|蹴り、突き、踏み込み、横移動|

## 日常一枚絵の成功条件

- 稽古後、道場の縁側、茶を飲む、夕暮れに構えるなど、静かな時間が合う。
- 凛々しさを残しつつ、少しだけ柔らかい表情にする。
- 武術服と姿勢で小鈴らしさを維持し、私服や現代服に寄せすぎない。
- 日常でも背筋、手の置き方、足の向きに武術家らしさを残す。

## 戦闘一枚絵の成功条件

- 低重心、踏み込み、腰の入り、胸郭と肩の連動を明示する。
- 攻撃部位だけを大きくするのではなく、身体全体から力が出ているようにする。
- スピード線、風圧、砂埃、衣装や髪のなびきは、技の方向を補うために使う。
- 足元、支持脚、体幹が見えないと動作の説得力が落ちるため、必要な範囲で残す。
- 敵キャラ、武器、派手なエフェクトは、動作確認段階では入れない。

## 掌底アクションの改善要素

### 一人称視点の指定

「相手目線」だけでは、肩越し構図や第三者視点にずれやすい。以下を明記する。

- 攻撃を受ける側の一人称視点。
- 画面内に敵キャラクターを描かない。
- 相手の頭、肩、背中、腕、シルエットを入れない。
- 肩越し構図、第三者視点、観察者視点にしない。
- カメラは相手の胸元から顔の高さ。
- 小鈴が視聴者をまっすぐ見据える。

### 掌底の身体操作

- 小鈴は低い中腰から前屈みに踏み込む。
- 後ろ足で床を押し、腰が前へ入り、胸郭と肩が前に通り、その流れで掌底が出る。
- 掌底は相手の顎、顔面下部、または胸元へ、下から上へ迫る。
- 真正面へ平たく押し出すだけの手のひらにしない。

### 手の制御

- 打撃面は手のひらの付け根部分。
- 親指は必ず見せる。
- 指は自然に揃える。
- 手首は適度に締める。
- 手のサイズは成人女性として自然な大きさに抑える。
- 老けた手、皺の多い手、節張った手、男性的な手にしない。
- 指の付け根、親指の付け根、掌底部分、手の中心のくぼみに柔らかい陰影を入れる。
- 手を細長くしすぎず、掌には自然な丸みと厚みを持たせる。

## 失敗しやすい点

- 手や足だけが巨大化し、身体とのつながりが消える。
- 掌底が「止まれ」の手や、防御ポーズに見える。
- 親指が消える、または手のひらに貼り付く。
- 手だけが写実的で、顔や身体のアニメ調と合わない。
- 「相手目線」が肩越し構図になり、敵キャラの後頭部や肩が入る。
- 背景の文字や看板が崩れ、主役より目立つ。
- 白黒武術服がメイド服、制服、ドレス、スポーツウェアに寄る。

## 採用判定チェック

### 外見

- [ ] 黒髪ショートボブ、前髪、サイドロックが維持されている。
- [ ] 琥珀色から黄褐色の瞳で、鋭いが冷たすぎない。
- [ ] 白黒中華武術服、黒縁取り、黒チャイナボタンが残っている。
- [ ] 黒レギンスと白黒武術シューズが残っている。

### 一枚絵

- [ ] 小鈴一人、一つの動作に絞られている。
- [ ] 顔、手足、重心、技の方向が読みやすい。
- [ ] 背景は動作を補助しており、主役を奪っていない。
- [ ] 分割パネル、一覧画像、説明図になっていない。

### アクション

- [ ] 支持脚、骨盤、胸郭、肩、腕または蹴り脚が自然につながっている。
- [ ] 手足の遠近感が強すぎず、成人女性の比率として成立している。
- [ ] 技の軌道がスピード線、砂埃、衣装のなびきと一致している。
- [ ] 掌底の場合、親指、掌底の厚み、手のひらの立体感が成立している。
