- シート: `03_named_images/小鈴/01_mannequin_body/小鈴_マネキン連続_SEQ-WALK-01歩行8フレーム_候補01.png`
- GIF: `03_named_images/小鈴/01_mannequin_body/小鈴_マネキン連続_SEQ-WALK-01歩行8フレーム_候補01.gif`
- 生成目的: 完全側面・画面右向き・その場歩きの8フレームループ。

## 結論

前回のCAT-MOVE-01より歩行周期へ近づいたが、採用不可。

01から04は接地、荷重、通過、振り出しの半周期として概ね読める。05から08は同じシルエットの反復になっており、右脚と左脚が正しく入れ替わったことを視覚的に確認できない。

白一色の左右対称マネキンを完全側面で描いたため、手前側と奥側の腕・脚の所有関係が消えている。これは生成結果だけでなく、検証用資料の設計にも問題がある。

## 腕の評価

歩行では、前へ出る脚と反対側の腕が前へ出る必要がある。

今回のシートでは、全フレームで画面手前に見える腕の配置がほぼ同じ。01から04と05から08で左右腕が交換されたことを確認できない。

- 01から04: 腕の振り幅変化が小さい。
- 04から05: 前後腕が交換されるべき境界だが、同じ腕配置に見える。
- 05から08: 01から04の反復に見える。

脚以上に腕振りの一貫性が弱い。

## 良かった点

- CAT一覧ではなく、一つの歩行だけに限定できた。
- 正面、背面、走行、停止、後退は混入していない。
- 全フレームがほぼ同じ完全側面カメラ。
- 01から04は接地、荷重、通過、振り出しの流れがある。
- 少なくとも一方の足が床へ接しており、走行の滞空はない。
- 同じ女性型マネキン体格が概ね維持されている。

## 出来なかった理由

### プロンプト側の問題

1. 左右を文章だけで指定した
   - 白一色の完全側面では、左脚と右脚が重なり、モデルがどちらを描いたか判別できない。

2. 「05から08は左右反転」と指定した
   - 同じ方向を向いた側面歩行では、左右反転した位相はほぼ同じシルエットになる。
   - 生成モデルは手足の所有を交換せず、上段の形を再利用しやすい。

3. 一度に要求する制約が多い
   - 8位相、左右脚、腕振り、骨盤固定、床固定、体格固定、4×2レイアウトを一度に処理させた。
   - 大きな構図は守られたが、左右の意味とピクセル位置が弱くなった。

4. 診断用マーカーがなかった
   - 人間にも生成モデルにも、どちらが左手足かを追跡する視覚情報がない。

### イラスト生成機能側の課題

1. 永続的な骨格IDを持たない
   - パネルをまたいで「この脚が同じ左脚」という内部的な保証がない。

2. グリッド内を別々のイラストとして処理しやすい
   - 同一人物でも、位置、歩幅、手足の奥行きがセルごとに再解釈される。

3. 完全側面の重なりに弱い
   - 手前と奥の腕・脚が重なり、左右の所有関係が消えやすい。

4. 座標拘束ができない
   - 「骨盤X固定」「床Y固定」は意味として理解できても、ピクセル単位の拘束にはならない。

5. 画像生成はアニメーション・運動学エンジンではない
   - 単発ポーズの生成より、連続した中割りとループ閉鎖の方が難しい。

## 次回の対策

### 1. 診断用の左右マーカーを付ける

白いマネキン本体は維持し、以下だけ色分けする。

- 左手首と左足首: 細い青リング。
- 右手首と右足首: 細い赤リング。
- 左右リングは全フレームで消さない、入れ替えない。

これにより、右脚、左脚、右腕、左腕をフレーム単位で追跡できる。

### 2. 4フレームずつ生成する

- Aシート: 01から04、左脚接地から右脚振り出し。
- Bシート: 05から08、右脚接地から左脚振り出し。

先にAを確定し、Aと左右マーカーを参照してBを生成する。一度に8位相を生成しない。

### 3. 腕振りを独立した検査項目にする

- 前脚と同じ側の腕が前へ出ていないか。
- 01と05で前腕の色が交換されているか。
- 04と08で前へ振り出す腕が交換されているか。

### 4. 足元・骨盤は後処理で整列する

生成後に以下を機械的に補正する。

- 接地足の最下点を共通Yへ合わせる。
- 骨盤中心を共通Xへ合わせる。
- 人物の高さを許容範囲内で統一する。

ただしポーズや左右脚の誤りは自動整列では修正しない。

### 5. 採用ゲート

- [ ] 左青リング、右赤リングが全フレームで維持されている。
- [ ] 01と05で前方の足リング色が交換されている。
- [ ] 03と07で支持脚リング色が交換されている。
- [ ] 04と08で振り出し脚リング色が交換されている。
- [ ] 前脚と反対側の腕が前へ出ている。
- [ ] 01から04、05から08、08から01が連続する。
- [ ] 骨盤X、足元Yを整列後にGIF化している。

すべて満たした場合だけ、小鈴のキャラクター画像へ変換する。
