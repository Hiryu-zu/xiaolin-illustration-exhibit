## 結論

今回のモデルイラスト生成ワークフローへ応用できる。

ただし、役割はそのまま全身戦闘ポーズを作るものではなく、以下の補助資料として使うのがよい。

1. 顔角度の一貫性確認。
2. 目・口・表情差分の固定。
3. 生成画像をセル単位で切り抜く運用。
4. 複数フレームをGIF化してズレを確認する運用。

小鈴では特に、戦闘ポーズごとに顔つき、目線、髪型が変わる問題を抑えるための「顔角度リファレンス」として有効。

## 使える要素

### 1. 5×5角度シート

ぐるぐるでは、5列×5行で顔の左右角度と上下角度を作っている。

- 列: 左横顔、左斜め、正面、右斜め、右横顔。
- 行: 強く上、少し上、水平、少し下、強く下。

小鈴への応用:

- 戦闘カットの顔向き指定に使う。
- 「カメラ目線へ戻る」「後方構図で首だけ振り返る」問題の補助資料にする。
- 小鈴の鋭い目つき、琥珀色の瞳、黒髪ショートボブの維持に使う。

### 2. 表情差分生成

ぐるぐるでは、AからFまでの6状態を作る。

|状態|目|口|
|---|---|---|
|A|開け|とじ|
|B|開け|中間|
|C|開け|開け|
|D|閉じ|とじ|
|E|閉じ|中間|
|F|閉じ|開け|

小鈴への応用:

- 戦闘用には全6状態は不要。
- まずは以下の5表情があれば十分。
  - 通常の鋭い真顔。
  - 集中した戦闘顔。
  - 軽い笑顔。
  - 照れ顔。
  - 目閉じ、息を整える表情。

### 3. 切り抜きツール

`tools/slice_character_sheets.py` は、単純な900pxセル切りではなく、透明部分の連結成分を検出して切り抜く。

利点:

- 隣セルにはみ出した髪や服を誤って混入させにくい。
- 透明背景のグレー残りを除去できる。
- 切り抜き後の中心位置と足元位置を揃えられる。

小鈴への応用:

- 5×5顔角度シートを作る場合に有効。
- 将来、全身ポーズの一覧シートをセル分割する場合にも応用可能。
- ただし、現状のスクリプトは「5×5」「A-F表情差分」「頭部またはアバター用」を前提にしているため、全身戦闘ポーズ用には調整が必要。

### 4. 背景除去ツール

`tools/prepare_character_sheets.py` は、ChatGPT Imagesで市松模様や薄い白灰色背景が焼き付いた場合に、外周からつながる背景を透明化する。

小鈴への応用:

- 白背景や薄灰背景で出力した素材の切り抜き補助に使える。
- レギンスや黒髪など暗色部分は残しやすいが、白い服と白背景の境界は要注意。
- 小鈴は白トップスなので、完全自動ではなく目視確認が必要。

### 5. GIF確認

`tools/make_demo_gif.py` はWebアバターのデモGIF用。
今回の `04_gif_tools/make_preview_gif.py` は、戦闘カットを順番に並べる用途として既に十分。

応用するなら:

- フレームごとの位置合わせ。
- 背景色指定。
- 文字やUIなしの軽量GIF。
- 透明PNGから白背景GIFへの変換。

## 小鈴での推奨追加フロー

現在の制作順に、顔角度資料を追加する。

1. 正本立ち絵を確定する。
2. 小鈴の5×5顔角度シートを作る。
3. 必要な表情差分を作る。
4. 体格マネキン四面図を作る。
5. 基本構え八方向を作る。
6. 実戦基礎動作をマネキンで作る。
7. キャラクター化時に、正本立ち絵、顔角度シート、マネキンを役割分担して添付する。

## キャラクター化時の参考画像役割

```text
参考画像1: 小鈴の正本立ち絵。顔、髪、目、衣装、靴、配色の正本。
参考画像2: 小鈴の5×5顔角度シート。顔の向き、目線、髪型の角度変化の正本。
参考画像3: 白い3Dマネキン。姿勢、支持脚、重心、関節、カメラ方向の正本。
```

この分け方により、マネキンのポーズへ合わせても顔だけ別人化しにくくなる。
