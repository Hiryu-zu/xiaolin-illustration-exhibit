## 切り抜き候補

以下は、見た目上は一連のモーションとして完全にはつながっていないが、切り抜けば別アニメーションの部品として使える可能性がある。

|画像|切り抜き後の主な用途|注意点|
|---|---|---|
|基本構え正面|待機、構え、会話中の武術姿勢|床影が残る。必要なら影を別途除去|
|前蹴り膝上げ|前蹴りの途中、膝蹴り予備、バランス動作|軸足と上体は使いやすい|
|横蹴り到達|蹴りの決めポーズ、攻撃カット|脚が高く、前蹴りではなく横/高蹴り寄り|
|背面膝上げ|背面回転、振り向き、旋身換歩の途中|第一成果物の三方向には直接入れず、回転動作の補助にする|
|左背面構え|背面待機、方向転換、後退開始|正面構えとの連続性は弱いが単体部品として有効|
|正本立ち絵|会話立ち絵、待機、表情差分の土台|白衣装と背景の境界は丁寧に抜く必要あり|

## 切り抜き可否

確認結果:

- 元PNGはすべて透明ではない。アルファなしの通常PNG。
- 小鈴の全身画像は、外周から白〜薄灰背景を除去する簡易切り抜きが可能。
- キャラ本体はキャンバス端へ接していないため、素材化しやすい。
- ただし、足元の床影も一部残る。
- 白い衣装と背景が近い部分は、完全自動だと境界が少し甘くなる可能性がある。

今回の切り抜き方法:

- 最初の確認では一時Pythonスクリプトを使用した。
- その後、同じ処理を `04_asset_tools/cutout_background.py` として正式ツール化した。
- 画像外周からつながる白〜薄灰色、低彩度の背景だけを flood fill で透明化する。
- キャラ内部の白い服は外周とつながっていないため、おおむね残った。
- 床影は背景色と違うため一部残った。
- `03_named_images/小鈴/07_cutout_tests/小鈴_ツール切り抜き一覧.jpg` でツール実行結果を確認できる。
- この検証は「切り抜けるか」の確認であり、最終素材化ではない。

ツール実行例:

```powershell
python "C:\モデルイラスト生成ワークフロー\04_asset_tools\cutout_background.py" `
  --inputs "03_named_images\小鈴\06_cutout_candidates\*.png" `
  --out-dir "03_named_images\小鈴\07_cutout_tests" `
  --suffix "_tool_cutout" `
  --preview "03_named_images\小鈴\07_cutout_tests\小鈴_ツール切り抜き一覧.jpg" `
  --csv "03_named_images\小鈴\07_cutout_tests\小鈴_ツール切り抜きログ.csv"
```

今後の背景指定:

- マネキン検証では、床面と接地影を残して重心確認を優先する。
- キャラクター化後に素材として使う画像では、完全な無地白背景、床なし、影なし、グラデーションなしを指定する。
- 透明PNGが確実に出せる環境なら透明背景を試す。
- 詳細は `02_prompts/切り抜き用背景指定_共通ブロック.md` を参照する。

## 現状のシャオリンのモーション評価

現状の5枚は、完全な連続モーションではない。

- 構え正面。
- 膝上げ。
- 高い蹴り到達。
- 背面膝上げ。

このまま `01 -> 02 -> 03 -> 04 -> 05` のように並べると、カメラ方向と身体向きが飛びすぎる。
ただし、各画像は切り抜き素材としては使える。
