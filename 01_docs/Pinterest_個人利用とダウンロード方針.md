# Pinterest 個人利用とダウンロード方針

確認日: 2026-06-17

## 結論

今回のプロジェクトでは、Pinterest画像は「個人作業用の参考資料」としてのみダウンロード可にする。  
ダウンロードした画像を、成果物・配布物・公開資料・生成画像の直接素材としては使わない。

つまり、許可するのは「見る、分類する、動きや構図を言語化する」範囲まで。  
禁止するのは「画像そのものを流用する、模写に近い形で再現する、外部公開物に入れる、素材として再配布する」こと。

## 根拠メモ

- Pinterest利用規約では、投稿コンテンツの権利と責任は投稿者側にある。
- Pinterestに投稿されたコンテンツは、Pinterestや他のユーザーがサービス内で保存・表示・利用できる形になっている。
- ただし、Pinterestヘルプでは、特定の画像・動画の使用許可が必要な場合は著作権保有者へ問い合わせる必要がある、と説明されている。
- Pinterestは、特定ピンの著作権保有者や連絡方法を把握・提供できないとしている。
- 日本の私的使用のための複製は、個人的または家庭内など限られた範囲が前提。仕事・公開・配布・商用利用へ広げると別問題になる。

参照:

- Pinterest 利用規約: `https://policy.pinterest.com/ja/terms-of-service`
- Pinterest 著作権ヘルプ: `https://help.pinterest.com/ja/article/copyright`
- CRIC 著作権Q&A 私的使用のための複製: `https://www.cric.or.jp/qa/hajime/hajime7.html`
- 文化庁 私的使用目的の複製に係る権利制限について: `https://www.bunka.go.jp/seisaku/bunkashingikai/chosakuken/hoki/h30_06/pdf/r1411529_06.pdf`

## このプロジェクトで許可する使い方

### OK

- Pinterest公式UIの `画像をダウンロード` から保存する。
- 保存先は `05_reference_paths` または専用の非公開参考フォルダに限定する。
- ファイル名またはメモに、元URL、確認日、用途を残す。
- 動き、構図、エフェクト方向、身体の重心などを観察メモに変換する。
- プロンプトでは、画像そのものではなく「観察した条件」を文章化して使う。
- 個人作業中の比較、分類、検証に使う。

### 条件付きOK

- 生成AIへ参考画像として入れる場合:
  - 個人検証の範囲に限定する。
  - 画像の絵柄、線、構図をそのまま再現させない。
  - 「人体正本ではなく、エフェクトの流れだけを参考にする」など役割を限定する。
  - 公開・販売・納品予定の成果物では、権利確認済み素材または自作資料に置き換える。

### NG

- ダウンロード画像を成果物フォルダの完成素材として扱う。
- 画像を切り抜いて小鈴の素材やGIF部品に混ぜる。
- 画像をそのままSNS、ブログ、ポートフォリオ、配布資料に掲載する。
- 画像を素材集として再配布する。
- 元画像に似せた完成イラストを作る。
- 商用・公開用途で、権利者不明のまま使う。

## ダウンロードする場合の保存ルール

推奨フォルダ:

```text
05_reference_paths/pinterest_downloads/
```

推奨命名:

```text
Pinterest_用途_ピンID_確認日.png
Pinterest_風螺旋_367887863301978590_2026-06-17.png
Pinterest_武術歩型_223843043966357869_2026-06-17.png
```

各画像ごとに、同じフォルダへ `sources.csv` を置く。

```csv
file,pin_url,checked_date,use,rights_note
Pinterest_風螺旋_367887863301978590_2026-06-17.png,https://jp.pinterest.com/pin/367887863301978590/,2026-06-17,エフェクト方向の観察,個人参考のみ・公開素材化しない
```

## ワークフロー上の扱い

1. Pinterestから正規UIで保存する。
2. `sources.csv` に出典と用途を記録する。
3. 画像を見て、使える条件を文章化する。
4. マネキン生成では、文章化した条件だけ使う。
5. 完成素材に進む段階で、Pinterest画像は参照から外す。
6. 公開・販売・納品する場合は、自作マネキン、自作エフェクト、権利確認済み素材に置き換える。

## 今回の判定

今回の小鈴モーション制作では、Pinterest資料をダウンロードしてもよい。  
ただし、用途は以下に限定する。

- エフェクトの方向、密度、余白の観察
- 武術歩型の低い姿勢、支持脚、重心の観察
- プロンプト化するための言語メモ作成

小鈴のGIF、切り抜き素材、完成イラスト、配布可能な資料には、Pinterest画像そのものを含めない。
