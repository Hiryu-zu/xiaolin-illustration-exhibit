# テーマ別プロンプト素材インデックス

記事テーマを決めるための索引です。1枚の画像は複数テーマに所属します。

## 使い方

- 記事テーマを先に決める
- 該当画像を3〜12枚程度選ぶ
- `strengths / composition / motion / lighting` を記事本文に変換する
- `selected` は一時置き場なので、library登録後は削除してよい
