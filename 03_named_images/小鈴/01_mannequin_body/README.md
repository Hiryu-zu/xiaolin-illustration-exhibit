# 小鈴 マネキン資料

小鈴の体格、基本動作、連続アクションを検討する白い3Dマネキン資料を置く。

## 現在の参考画像

- `小鈴_旧マネキン基礎動作8種シート_参考01.png`
  - 4列×2段、8パネル、薄灰背景、床面、接地影のレイアウト参考。
  - 男性体型と筋肉量は小鈴へ引き継がない。
  - 8種類の単発ポーズ一覧であり、連続モーションの正本にはしない。

## 生成済みアクションシート

- `小鈴_マネキン動作一覧_C01基本移動8種_01.png`
  - CAT-MOVE-01 基本移動8種一覧。非連続のポーズ辞書。
  - 01自然立ち、02歩行、03早歩き、04走行、05急停止、06方向転換、07横移動、08後退。
  - 歩行GIFや連続モーションには使用禁止。
  - 2026-06-21生成。

- `小鈴_マネキン連続_SEQ-WALK-01歩行8フレーム_候補01.png`
  - SEQ-WALK-01歩行8フレーム候補。
  - 完全側面、画面右向き、同一カメラ、同一床面。
  - 01から04と05から08を左右脚の対応位相として生成。
  - 白い左右対称マネキンでは近側・遠側の脚の交代が判別しにくいため、GIFで要確認。

- `小鈴_マネキン連続_SEQ-WALK-01歩行8フレーム_候補01.gif`
  - 上記シートの01から08を120ms/コマで確認する無限ループGIF。
  - マネキン段階の採否確認用。合格前に小鈴へ変換しない。
  - レビュー判定: 不採用。左右脚・左右腕の交換を白一色の側面像から確認できない。
