# 小鈴 基礎動作三方向 参考資料候補

確認日: 2026-06-17

## 目的

白い3Dマネキンを毎回ゼロから作る手間を減らすため、Pinterest上で見つかる既存の武術・格闘ポーズ資料を、各モーションの下敷きとして使う。

ただし、Pinterest画像は個人作業用の参考資料として扱い、完成素材・公開資料・配布物・小鈴GIFの直接部品にはしない。  
使うのは「姿勢」「支持脚」「重心」「腕や脚の軌道」「ステップ順」の観察結果。

## 動作別の使い分け

### 01_前手直突き

主資料:

- 武術歩型一覧: 前脚重心、弓歩、足幅
- パンチ参照: 肩、肘、拳の直線
- 前屈立ち: 前後の体重配分

見る点:

- 前脚に乗るが、膝が内側へ潰れない。
- 前手は肩から拳まで一直線。
- 後ろ手は胸元または腰元で防御として残す。
- 側面では拳、肩、前膝、前足が同じ方向に伸びる。

### 02_後手直突き

主資料:

- 武術歩型一覧: 後ろ足の蹴り出し
- パンチ参照: 後ろ肩から拳までの回転
- 長拳連続表: 腰と胸郭の回転

見る点:

- 後ろ足、骨盤、肩、拳の順に力が伝わる。
- 後手だけが伸び、前手は顔前の守りに残る。
- 横から見たとき、上体が前へ倒れすぎない。

### 03_穿掌

主資料:

- Chāquán基本手技
- 八卦掌の基本掌位
- 長拳連続表

見る点:

- 指先が伸びるが、手首を反らせすぎない。
- 掌は刃物のように細く前方へ通す。
- 肘、肩、肩甲骨が自然につながる。
- 小鈴らしい中華武術の個性を出す核にする。

### 04_前蹴り

主資料:

- 前蹴りステップ順
- 武術歩型一覧
- 前屈立ち

見る点:

- 01構え、02膝上げ、03蹴り到達の3段階が明確。
- 02膝上げで支持脚の真上に重心が残る。
- 03到達で上体は少しだけ後ろへ逃がす。
- 蹴った足を戻す絵は作らず、GIFで逆順再生する。

### 05_横蹴り

主資料:

- 横蹴り参照
- 高い横蹴り参照
- 武術歩型一覧

見る点:

- 02で膝を横へ畳む。
- 03で足刀を横へ押し出す。
- 軸足、骨盤、蹴り脚が同一面に乗る。
- 側面では蹴り脚が胴体に隠れすぎない。

### 06_円掌受け

主資料:

- 八卦掌の基本掌位
- Chāquán基本手技
- エフェクト動作ラフ

見る点:

- 腕だけで円を描かず、肩甲骨と胸郭も回す。
- 掌は顔前から外側へ流れる。
- 円弧の開始点と終了点を決める。
- エフェクトを使う場合も、顔、手首、肘を隠さない。

### 07_退歩

主資料:

- 武術歩型一覧の虚歩、坐盤、低い構え
- 長拳連続表
- エフェクト動作ラフの砂塵

見る点:

- 攻撃を引く動作ではなく、重心を後ろへ移す歩法として作る。
- 前足は軽くなり、後脚が体重を受ける。
- 上体は逃げすぎず、相手を見続ける。
- 砂塵は足元だけ。接地を隠さない。

### 08_旋身換歩

主資料:

- 長拳連続表
- 武術歩型一覧
- エフェクト動作ラフの螺旋

見る点:

- 視線、肩、骨盤、足の順で向きが変わる。
- 軸足が突然入れ替わらない。
- 02途中で片脚が浮く場合、接地脚を明確にする。
- 03到達で次の構えとして成立させる。
- 第一段階では背面を必須にせず、必要なら追加資料として扱う。

## ダウンロード運用

ダウンロードする場合は、Pinterest公式UIの `画像をダウンロード` を使う。保存先は以下。

```text
05_reference_paths/pinterest_downloads/
```

保存後、`05_reference_paths/pinterest_downloads/sources.csv` に以下を記録する。

```csv
file,pin_url,checked_date,use,rights_note
```

YouTube動画を使う場合は、以下へ記録する。

```text
05_reference_paths/youtube_motion_refs/
```

```csv
file_or_folder,video_url,checked_date,time_range,target_motion,use,rights_note,status
```

## 判定

最初にダウンロードするなら、以下の3つで十分。

1. 武術歩型一覧
2. 前蹴りステップ順
3. 長拳連続表

この3つで、8動作のうち「拳・掌・蹴り・退歩・旋身換歩」の大枠をかなり埋められる。  
不足したら、横蹴り、掌位、パンチ資料を追加する。
