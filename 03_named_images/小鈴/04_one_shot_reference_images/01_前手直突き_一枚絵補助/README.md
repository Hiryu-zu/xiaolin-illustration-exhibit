# 01_前手直突き 一枚絵補助

## 目的

前手直突きのモーション連番には使いづらいが、一枚絵の拳、腕、打撃到達姿勢、パースの観察には使える画像を保存する。

## 移動理由

元の保存先:

- `03_named_images/小鈴/03_basic_motion_three_views/01_前手直突き/00_reference_images`

移動理由:

- 構え、途中、到達の動きの繋がりが読みにくい。
- 三方向GIFのマネキン正本として使うと、フレーム間の一貫性確認に向かない。
- 一方で、完成一枚絵の拳腕パース、前手と後手の役割、到達姿勢の迫力には使える。

## 注意

- 動きの連続性を判断する資料にはしない。
- 三方向GIF用プロンプトでは、メイン参考として指定しない。
- 完成一枚絵に使う場合も、ポーズをそのまま再現せず、拳腕の方向性、迫力、パースだけを観察する。
