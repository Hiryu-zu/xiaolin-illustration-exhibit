# モデルイラスト生成ワークフロー

このフォルダは、ChatGPT画像生成でキャラクター戦闘構図を安定して作るための実験知見置き場。
リーファンと牙焔で試した「デッサン人形 -> キャラクター化 -> 採用/修正判断 -> GIF確認」の流れをまとめる。

## フォルダ構成

- `01_docs`
  - ワークフロー解説、キャラ別の構図設計資料。
- `02_prompts`
  - 実際に使ったプロンプト集。キャラ別フォルダからコピーしたもの。
- `03_named_images`
  - `GPT出力イラスト_○○` の画像を、内容が分かる名前にしたコピー。
  - 元画像は削除せず、元フォルダにも同名コピーを置いて検索しやすくする。
- `04_gif_tools`
  - 連続動作チェック用のGIF作成ツール。
- `04_asset_tools`
  - 切り抜き、透明PNG化、素材化確認用のツール。
- `05_reference_paths`
  - 参照元フォルダ、立ち絵、出力先などの一覧。

## 基本思想

1. 最初から完成絵を狙わず、白い3Dデッサン人形で構図を固める。
2. 人形は「体の向き」「重心」「足裏接地」「手指」「カメラ角度」を見るための資料とする。
3. キャラクター化では、立ち絵を外見の正解、人形画像をポーズの正解として分離して指示する。
4. 戦闘絵は一枚ずつ生成する。八分割シートは資料には便利だが、完成絵では顔・指・衣装が崩れやすい。
5. 背景は白背景を基本にする。切り抜きやすく、ポーズ資料として比較しやすい。
6. 長い鎖やリボンなどは破綻しやすい。一本の流れ、C字、画面外へ逃がす遠近などで制御する。
7. 複数枚が動きとしてつながる場合はGIF化し、流れと破綻を確認する。

## 実験で得たルール

- 人形シートは一枚にまとめてよい。完成キャラ絵は一枚ずつ出す。
- 指は毎回明示する。開き手、掌底、拳、鉤手などを指定し、指の本数と曲げ方向をチェックする。
- 靴は明示する。リーファンでは裸足化しやすいため「両足ともピンクの靴を履く」を固定文に入れる。
- ショートパンツや前掛けの長さは比率で指定する。「太ももの上半分を覆う」など。
- 顔の向きは「視線は技の方向」だけではぶれる。頭部の向き、目線、胴体の向きを別々に書く。
- 牙焔の炎や鎖は、関節や指を隠しすぎると確認不能になる。エフェクトは体の外周を流す。
- GIFは滑らかさより「動きがつながるか」「別カットとして使えるか」の確認に使う。

## 推奨フロー

1. キャラ立ち絵の四面または八方向の人形資料を作る。
2. 基本構えを八方向で作る。
3. 実戦的な基本動作を八種類作る。
4. 良い動作だけをキャラクター化する。
5. 採用候補を命名して `03_named_images` に保存する。
6. 連続性があるものは `04_gif_tools` でGIF化する。
7. 最後に固有技、エフェクト、武器、技名を足す。

## 今回の採用資料

- 小鈴
  - 正本立ち絵候補01: 白黒基調の中華武術服、黒髪ショートボブ、琥珀色の鋭い目。
  - キャラ設定: 21歳、165cm、7.3頭身前後、しなやかな武術体型。
  - 出力画像整理: ルート直下の `ChatGPT Image 2026年6月16日 22_*` は `03_named_images/小鈴` 以下へ分類し、重複元ファイルは削除済み。
  - 切り抜き確認: `04_asset_tools/cutout_background.py` で透明化テスト済み。白〜薄灰背景の小鈴全身画像は切り抜き可能。床影は要調整。
  - 背景運用: マネキン検証は薄い床面と接地影あり。素材化するキャラクター画像は純白背景、床なし、影なしを指定する。
  - 顔角度資料: ぐるぐるの5×5角度シート手法を応用。
  - マネキン補強: Kindleアクションポーズ資料から、アイレベル、消失点、重心、力線、軸足の観察を反映。
  - 第一成果物: 基礎動作8種を、正面・左側面・右側面の3方向連番で作り、GIFで一貫性を確認する。
  - GIF方針: 引き戻し動作は生成せず、`01構え -> 02途中 -> 03到達 -> 02途中 -> 01構え` で確認する。
  - 次工程: 顔角度シート、体格マネキン四面図、基本構え三方向、基礎動作三方向GIF。
- リーファン
  - 基礎01-08: 突き、穿掌、前蹴、横蹴、受け、双掌、退歩、換歩。
  - 光華拳01-05: 独自技候補。02と05が特に良好、03は条件付き、01と04は修正候補。
- 牙焔
  - 基本格闘構え01-08: 八方向の構え資料。
  - 実戦基礎動作01-08: 打撃、掌底、フック、肘、膝、前蹴、回し蹴り。
  - 01-07はGIFで連結確認済み。連続アニメというより、戦闘カットの流れとして有効。
