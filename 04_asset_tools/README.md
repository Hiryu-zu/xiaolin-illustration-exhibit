# アセットツール

画像素材化、切り抜き、透明PNG化などの補助ツール置き場。

## 背景切り抜き

`cutout_background.py` は、白〜薄灰色の単純背景を外周から検出し、透明PNGに変換するツール。

現状の用途:

- 小鈴の白背景・薄灰背景画像を、アニメーション素材や合成素材として使えるか確認する。
- 出力画像をまとめて透明化する。
- チェック柄プレビューとCSVログを作る。

## 使い方

```powershell
python "C:\モデルイラスト生成ワークフロー\04_asset_tools\cutout_background.py" `
  --inputs "03_named_images\小鈴\06_cutout_candidates\*.png" `
  --out-dir "03_named_images\小鈴\07_cutout_tests" `
  --suffix "_tool_cutout" `
  --preview "03_named_images\小鈴\07_cutout_tests\小鈴_ツール切り抜き一覧.jpg" `
  --csv "03_named_images\小鈴\07_cutout_tests\小鈴_ツール切り抜きログ.csv"
```

## 主なオプション

- `--min-brightness`
  - 背景候補にする明るさ。初期値 `198`。
  - 背景が残るなら下げる。服や肌が消えるなら上げる。
- `--max-delta`
  - RGB差の許容値。初期値 `28`。
  - 低彩度の白〜灰色背景だけを抜くための値。
- `--fringe`
  - 背景除去範囲を数px広げる。境界の白フチ除去に使う。
- `--feather`
  - 境界をぼかす。素材確認では基本 `0`。
- `--trim`
  - 透明化後、人物の外接矩形へ切り詰める。
- `--padding`
  - `--trim` 時の余白。
- `--preview`
  - チェック柄背景の一覧画像を作る。
- `--csv`
  - 切り抜きログをCSVで保存する。

## 注意

- このツールは、外周につながっている白〜薄灰色背景を抜く。
- キャラ内部の白い服は、外周とつながっていなければ残りやすい。
- 床影は背景と色が違うため、残る場合がある。
- 小鈴は白い衣装なので、最終素材化では目視確認と手動補正が必要。
- マネキン検証画像は床面と接地影が必要なため、切り抜き前提にしない。
