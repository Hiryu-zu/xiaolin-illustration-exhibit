# YouTube Motion References

YouTube動画から、個人作業用のモーション観察メモ、フレームシート、確認GIFを作るための候補フォルダ。

動画素材そのものは、完成素材・配布素材・公開資料として使わない。  
用途は、モーションの流れ、重心、支持脚、カメラ方向、手足のタイミングを観察し、プロンプトへ言語化することに限定する。

## 運用ルール

- 優先順位は低め。Pinterest静止画、Kindle観察メモ、既存マネキンで不足した場合だけ使う。
- 動画全体を保存・再配布しない。
- 切り出したフレームやGIFは、個人作業用の観察資料としてのみ使う。
- 完成イラスト、公開資料、配布物には動画由来のフレームを含めない。
- プロンプトへ入れる場合は、画像そのものではなく「観察した動作条件」を文章化する。
- 切り出しを行った場合は、`sources.csv` にURL、時間範囲、対象モーション、用途を記録する。

## 作業手順

1. 動画を視聴して、使えそうなモーションの時間範囲をメモする。
2. 対象モーションごとに短い範囲だけを切り出す。
3. 切り出した範囲から、3〜7枚程度の主要フレームを選ぶ。
4. 1枚のフレームシートに並べる。
5. 必要なら確認用GIFも作る。
6. 観察結果を、支持脚、重心、骨盤、胸郭、手足の軌道、顔と視線として文章化する。
7. マネキン生成時は、文章化した条件だけを使う。

## 小鈴モーションへの使いどころ

|小鈴モーション|使える可能性|
|---|---|
|04_前蹴り|膝上げ、支持脚、上体の逃がし方|
|05_横蹴り|股関節の開き、軸足、蹴り脚の面|
|08_旋身換歩|回転時の視線先行、足の入れ替え|
|07_退歩|重心を後ろへ移す歩法|
|03_穿掌 / 06_円掌受け|中華剣術や手技の体幹回転を間接参考|

## 判定

この動画は第一優先資料ではない。  
まずは静止画資料と白3Dマネキンで基礎動作を作り、破綻した動作や中割りが足りない動作だけ、この動画から補助観察する。
