# GPT Request Template: Martial Visual Prompt Design

## Purpose

I will provide reference images. Please help turn them into a reusable prompt system for martial arts visuals.
This is not for a note article yet. The goal is classification, prompt design, and reusable wording.

## Important premise

- Preserve the person, outfit, hairstyle, and body type from the reference image.
- Do not over-specify the character identity.
- Focus on martial arts composition, motion, lighting, shadow, manga-like staging, and impact.
- One image may belong to multiple themes.

## Theme labels

- `summary_sheet`: まとめイラスト
- `manga_composition`: 漫画構図
- `style_study`: 画風イラスト
- `eye_study`: 眼のイラスト
- `shadow_study`: 影のイラスト
- `screen_break`: 画面破壊のイラスト
- `breathing_study`: 呼吸のイラスト

## Output format requested

```text
Theme name:
Goal:
Composition guidance:
Lighting and shadow guidance:
Motion or stillness guidance:
Common failure points:
Reusable prompt:
Short prompt:
Variation ideas:
```
