# Prompt Design Notes

## Core policy

These images are research assets for future prompt requests. The prompt should preserve the reference image's character, outfit, hairstyle, and body type, while changing or strengthening the visual expression.

## Layer structure

1. Reference handling
2. Visual theme
3. Composition
4. Motion or stillness
5. Lighting and shadow
6. Negative control

## Theme directions

- summary_sheet: pose catalog, character reference, comparison sheet
- manga_composition: panel layout, cut-ins, sequential motion
- style_study: linework, paint density, background rendering
- eye_study: gaze, iris highlight, expression through eyes
- shadow_study: face shadow, giant wall shadow, psychological pressure
- screen_break: fist/kick foreground, cracks, debris, impact lines
- breathing_study: still training, breath, wind, negative space
