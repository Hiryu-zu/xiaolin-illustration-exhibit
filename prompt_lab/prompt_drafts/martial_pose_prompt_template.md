# プロンプト案: 武術ポーズ

## 基本テンプレート

```text
Use the reference image as the main guide for the character, outfit, hairstyle, body type, composition, and mood.
Preserve the person, outfit, hairstyle, and body type from the reference image.
Create a high-quality anime-style illustration set in [scene].
The pose/action is [pose], expressing [intent].
Composition: [composition].
Emphasize [strengths].
Keep [motion], [lighting], and [mood].
Avoid extra characters, unwanted weapons, text, logos, distorted hands, broken fingers, incorrect limb joints.
```

## 設計方針

人物そのものをプロンプトで固定しすぎず、参考画像を維持する。プロンプトでは、構図・光・動き・見どころを補強する。
