# 小鈴 マネキンアクションシート 基本動作プロンプト

## 目的

小鈴用の同一体格マネキンを使い、歩行、走行、前手直突き、前蹴り、横蹴りを8段階の連続動作シートとして生成する。

## 添付する参考画像

1. 小鈴用の承認済み体格マネキン四面図。
2. レイアウト参考: `03_named_images/小鈴/01_mannequin_body/小鈴_旧マネキン基礎動作8種シート_参考01.png`

レイアウト参考画像からは4列×2段の構成、薄灰背景、床面、番号配置だけを使う。男性体型、筋肉量、個別ポーズは引き継がない。

## 共通プロンプト

```prompt
添付した小鈴用の白い3D体格マネキンを使い、指定する一つの基本動作を8段階で示す連続アクションシートを作成してください。

【最重要】
これは異なる8ポーズの一覧ではなく、一つの動作を時間順に分解した連続フレームです。
01から08へ進むにつれて、同じ人物が自然に動いているようにしてください。
全フレームで体格、頭身、関節構造、カメラ方向、人物サイズを固定してください。

【出力形式】
一枚の16:9横長シート。
上段4パネル、下段4パネル。
上段左から01、02、03、04。
下段左から05、06、07、08。
各パネルに同じ白い3Dマネキンを一体だけ配置する。
各パネル左上に小さな白い番号だけを表示する。
技名、説明文、方向名は画像内に入れない。
人物を頭頂から足先まで完全に収め、手足を境界で切らない。

【マネキン】
小鈴用の21歳成人女性型マネキン。
身長165cm相当、自然な約7.3頭身。
しなやかで均整の取れた健康的な武術体型。
肩幅は標準からやや華奢。
細すぎず、筋肉質すぎない。
白または明るい灰色のマット樹脂素材。
頭部、胸郭、骨盤に薄い中心線。
肩、肘、手首、股関節、膝、足首の位置を明確にする。
手は五本指を簡潔に分ける。
両足は閉じた格闘靴型の簡略形状。

【フレーム一貫性】
前のフレームから次のフレームへ、頭、胸郭、骨盤、肘、手首、膝、足首の位置を少しずつ変える。
手足を突然左右反転させない。
人物の身長、頭の大きさ、肩幅、骨盤幅、手足の長さを変えない。
カメラの焦点距離、アイレベル、床の高さ、人物までの距離を変えない。
動作中の床面上の移動量を連続させる。

【背景】
均一な薄いグレー背景。
水平な薄いグレー床面。
足元に弱い接地影。
柔らかなスタジオ照明。
クリーンな3D人体デッサン教材風。

【禁止事項】
髪、衣装、前垂れ、レギンス、装飾、武器、敵、完成キャラクターの顔、
男性体型、ボディビルダー体型、過剰な筋肉、子供体型、
8体すべて同じポーズ、動作順の逆転、左右の突然の入れ替わり、
異なるカメラ方向、異なる人物サイズ、異なる床の高さ、広角歪み、
余分な腕や脚、六本指、欠けた指、融合した指、逆関節、反転した足首、
浮いた支持脚、接地影と足位置の不一致、
番号以外の文字、技名、説明文、ロゴ、透かし
```

## CAT-MOVE-01 基本移動一覧（非連続・GIF禁止）

```prompt
上記のマネキン、出力形式、背景条件を使用し、基本移動8種類を一枚のアクション一覧シートとして作成してください。

このセクションだけは連続フレームではなく、異なる8種類の基本動作を並べるポーズ辞書です。
歩行GIF、走行GIF、連続アニメーションの入力には使用しないでください。
各パネルに同じ小鈴用成人女性型マネキンを一体ずつ描いてください。

01: 正面から見た自然立ち・待機。両足へ均等に重心を置く。
02: 側面から見た通常歩行。前足の踵が接地する瞬間。
03: 側面から見た早歩き。歩行より歩幅と腕振りが大きい。
04: 側面から見た中速走行。片足接地、反対脚が前へ進む。
05: 斜め前から見た急停止。前脚で減速し、重心を低くする。
06: 斜め後ろから見た90度方向転換。軸足、骨盤、胸郭が回る。
07: 正面から見た横移動。片脚で床を押し、反対側へ重心を運ぶ。
08: 側面から見た後退。後方の足で安全に着地し、上体を立てる。

各動作の支持脚、進行方向、重心が一目で区別できるようにする。
歩行、走行、横移動、後退を同じポーズにしない。
```

## CAT-STRIKE-01 基本打撃一覧（非連続・GIF禁止）

```prompt
上記のマネキン、出力形式、背景条件を使用し、基本打撃8種類を一枚のアクション一覧シートとして作成してください。

このセクションだけは連続フレームではなく、異なる8種類の基本動作を並べるポーズ辞書です。
各パネルに同じ小鈴用成人女性型マネキンを一体ずつ描いてください。
左脚前、右脚後ろの右利き半身構えを基準にします。

01: 正面の基本ガード。左手をやや前、右手で顔と中心線を守る。
02: 右前方45度から見た左前手直突き。右手は守りに残す。
03: 左側面から見た右後手直突き。後足、骨盤、肩、拳をつなぐ。
04: 右側面から見た左掌底または穿掌。開いた掌を相手の中心へ伸ばす。
05: 背面寄りから見た右肘打ち。肩甲骨と骨盤の回転を見せる。
06: 左後方45度から見た左膝蹴り。支持脚へ重心を残す。
07: 左側面から見た左前蹴り。踵または靴底を前方へ向ける。
08: 左前方45度から見た左横蹴り。踵または足刀を接触部位にする。

拳、掌、肘、膝、前蹴り、横蹴りの接触部位を明確に区別する。
攻撃していない手は顔または身体の中心線を守る。
腕や脚だけで攻撃せず、接地足から骨盤、胸郭、肩または股関節へ力をつなげる。
```

## SEQ-WALK-01 / L01 歩く（歩行GIF専用）

```prompt
上記の共通条件で、右方向へ自然に歩く動作を8段階で作成してください。
カメラは人物の完全な左側面に固定し、人物は画面内で右を向きます。
誇張した競歩や走行にせず、落ち着いた通常速度の歩行にします。
これは一つの歩行周期を8段階へ分解した連続フレームです。異なる移動動作を混ぜないでください。
その場歩きとして、全フレームで骨盤中心の水平位置を固定してください。
全フレームで足元の基準線を同じ高さにしてください。

01: 左踵が前方へ接地。右足は後方。右腕が前、左腕が後ろ。
02: 左脚へ荷重し、身体が少し沈む。右足の踵が上がる。
03: 右足が左支持脚の横を通る。骨盤が支持脚の上へ乗る。
04: 左脚で床を押し、身体が少し上がる。右膝が前へ進む。
05: 01の左右反転。右踵が前方へ接地。
06: 右脚へ荷重し、身体が少し沈む。左足の踵が上がる。
07: 左足が右支持脚の横を通る。
08: 右脚で床を押し、左膝が前へ進み、次の01へ自然につながる。

腕は脚と自然に逆位相で振る。
頭部の上下動は小さく、骨盤は歩行に合わせてわずかに回転する。
01から08を順方向に繰り返すと自然な歩行ループになるようにする。
走行、急停止、方向転換、横移動、後退を一切混ぜない。
```

## SEQ-WALK-01-DIAG 左右識別マーカー

歩行シートの初回生成では、白マネキンへ左右診断用リングを追加する。

```prompt
SEQ-WALK-01の全条件に、以下の左右識別条件を追加してください。

白いマネキン本体と薄いグレー背景は維持してください。
左手首と左足首に、細い青色の識別リングを付けてください。
右手首と右足首に、細い赤色の識別リングを付けてください。

青リングは必ず左手足、赤リングは必ず右手足に固定してください。
フレーム途中で色を交換しないでください。
リングを消さない、増やさない、別の関節へ移動しないでください。

01では青い左足首が前方、赤い右足首が後方。
03では青い左脚が支持脚、赤い右足が通過。
05では赤い右足首が前方、青い左足首が後方。
07では赤い右脚が支持脚、青い左足が通過。

前方の脚と反対側の腕を前へ振ってください。
01と05、03と07、04と08で、手足のリング色が正しく交換されて見えるようにしてください。

診断用の色はリングだけに使用してください。
マネキン本体を青や赤に塗らないでください。
```

診断用リング版がGIF審査に合格した後、白一色版または小鈴版へ変換する。

## L02 走る

```prompt
上記の共通条件で、右方向へ自然に走る動作を8段階で作成してください。
カメラは人物の完全な左側面に固定し、人物は画面内で右を向きます。
全力疾走ではなく、安定した中速のランニングにします。

01: 左足が身体のほぼ真下へ接地。右脚は後方。右腕が前。
02: 左膝と足首が曲がり、着地の衝撃を吸収して重心が沈む。
03: 左脚で床を強く押し、右膝と左腕が前へ進む。
04: 両足が床から離れる短い滞空。右脚が前へ移る。
05: 01の左右反転。右足が身体のほぼ真下へ接地。
06: 右脚で衝撃を吸収する。
07: 右脚で床を押し、左膝と右腕が前へ進む。
08: 両足が床から離れる短い滞空。左脚が前へ移り、次の01へつながる。

歩行より上体をわずかに前傾させる。
腕は肘を曲げ、脚と逆位相で前後に振る。
着地、吸収、蹴り出し、滞空を明確に区別する。
01から08を順方向に繰り返すと自然な走行ループになるようにする。
```

## S01 前手直突き

```prompt
上記の共通条件で、左前手による直突きを8段階で作成してください。
カメラは人物の右前方45度に固定します。
左脚を前、右脚を後ろにした軽い半身構えから始めます。

01: 基本構え。両手で顔と中心線を守る。
02: 右後足で床を押し、重心が前へ動き始める。
03: 骨盤と胸郭がわずかに前へ入り、左肩が進む。
04: 左肘が拳の後ろを通り、左腕が半分伸びる。
05: 左拳が相手の中心へ進む。右手は顔の近くで守りを維持する。
06: 到達直前。左肩、肘、拳が自然な力線へそろう。
07: 到達。拳は胸から顔の高さ。手首は前腕の延長上。
08: 到達後の短い余韻。肘は過伸展させず、全身の力線を維持する。

腕だけで突かず、後足、骨盤、胸郭、肩、肘、拳へ力をつなげる。
拳だけを巨大化させない。
戻り動作は生成せず、GIF確認では08から01へ逆順再生する。
```

## S02 前蹴り

```prompt
上記の共通条件で、左脚の前蹴りを8段階で作成してください。
カメラは人物の完全な右側面に固定します。
右脚を支持脚、左脚を蹴り脚にします。

01: 基本構え。両手で顔と中心線を守る。
02: 重心を右支持脚へ移す。右足裏は完全に接地。
03: 左踵が床を離れ、左膝が前へ上がり始める。
04: 左膝を高く引き上げ、左下腿をたたむ。
05: 左膝を支点に下腿が前へ伸び始める。
06: 到達直前。左踵または靴底が標的方向へ向く。
07: 到達。左脚を前へ伸ばし、右支持脚へ重心を残す。
08: 到達後の短い余韻。上体と両手で均衡を保つ。

前蹴りを横蹴りや回し蹴りへ変化させない。
左股関節、膝、足首、踵を同じ進行面へ置く。
戻り動作は生成せず、GIF確認では08から01へ逆順再生する。
```

## S03 横蹴り

```prompt
上記の共通条件で、左脚の横蹴りを8段階で作成してください。
カメラは人物の右前方45度に固定します。
右脚を支持脚、左脚を蹴り脚にします。

01: 半身の基本構え。両手を守りに置く。
02: 重心を右支持脚へ移す。
03: 左膝を胸側へ引き上げ、左下腿をたたむ。
04: 右軸足を回し、骨盤を蹴る方向へ横向きに開く。
05: 左踵を先頭に、蹴り脚を横方向へ伸ばし始める。
06: 到達直前。左膝、踵、標的方向をそろえる。
07: 到達。左踵または足刀で横方向を蹴る。
08: 到達後の短い余韻。右支持脚上に重心を残し、上体と両手で均衡を取る。

舞踊的な大開脚にしない。
支持脚を浮かせず、膝と靴のつま先の方向を一致させる。
戻り動作は生成せず、GIF確認では08から01へ逆順再生する。
```

## 生成後チェック

- [ ] 8体が同じ体格、頭身、関節構造か。
- [ ] カメラ方向、アイレベル、床の高さ、人物サイズが固定されているか。
- [ ] 01から08へ動作が自然につながるか。
- [ ] 支持脚と接地影が一致しているか。
- [ ] 手足が突然左右反転していないか。
- [ ] 歩行と走行は08から01へ自然に戻るか。
- [ ] 突きと蹴りは01から08へ力が増しているか。
- [ ] 人物の手足がパネル境界で切れていないか。
