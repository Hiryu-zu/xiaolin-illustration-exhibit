# 小鈴 一枚絵生成プロンプト

## 目的

小鈴の採用済み立ち絵、マネキン、モーションを、完成一枚絵へ変換するための再利用プロンプト。

三方向GIFやマネキン生成ではなく、note本文、見出し、完成戦闘絵、日常絵に使う。

## 共通キャラクター固定

```prompt
小鈴（シャオリン）、21歳の女性。
身長165cm、7.3頭身前後。
しなやかで均整の取れた健康的な武術体型。
細すぎず、筋肉質すぎず、軽やかで無駄のない身体。

黒髪ショートボブ。
前髪があり、顔の両側に自然なサイドロックがある。
髪は艶のある黒。
瞳は琥珀色から黄褐色。
顔立ちは可愛さと凛々しさの中間。
目つきはやや鋭いが、冷酷すぎない。

白黒ベースの中華武術服。
白い立ち襟トップス、黒い縁取り、黒いチャイナボタン、黒いサイドパネル。
黒レギンス。
白黒配色の中華武術シューズ。
金色や深紅の装飾は少量のみ。

アニメ調、ライトノベル表紙風、精密なデジタルイラスト。
清潔感があり、凛々しく、武術家としての静かな強さが伝わる雰囲気。
```

## 日常一枚絵

```prompt
上記の小鈴を、日常寄りの一枚絵として描く。

一枚絵として完成度の高い構図にする。
小鈴は一人だけを描く。
道場、武館の縁側、稽古後の室内、夕暮れの中庭など、武術の文脈が自然に伝わる背景にする。

表情は少し柔らかいが、凛々しさを残す。
背筋、手の置き方、足の向きに武術家らしさを残す。
衣装は白黒の中華武術服のままにし、私服、制服、メイド服、ドレスにはしない。

おすすめ比率は4:5、16:9、またはnote見出し用の1.91:1。
```

## 戦闘一枚絵

```prompt
上記の小鈴を、戦闘アクションの一枚絵として描く。

一枚絵として迫力があり、同時に読みやすい構図にする。
小鈴は一人だけを描く。
一つの技、一つの瞬間に絞る。
グリッド、分割パネル、連続写真、説明図にはしない。

支持脚、骨盤、胸郭、肩、腕または蹴り脚までの力の流れが分かるようにする。
攻撃部位だけを大きくするのではなく、身体全体から力が出ているように描く。
顔と視線は攻撃方向へ向ける。

髪、袖、前垂れ、砂埃、風圧、流線は技の方向を補うために使う。
背景は補助的にし、主役は小鈴の姿勢、表情、技の軌道にする。

おすすめ比率は16:9または3:2。
```

## 一人称視点・掌底制御

```prompt
構図は攻撃を受ける側の一人称視点にする。
画面内に敵キャラクターを描かない。
相手の頭、肩、背中、腕、シルエットを入れない。
肩越し構図、第三者視点、観察者視点にしない。

小鈴は画面中央の視聴者をまっすぐ見据え、低い中腰から前屈みに踏み込み、掌底を斜め上に突き上げている。
ただ腕だけを前に伸ばすのではなく、後ろ足で床を押し、腰が前へ入り、胸郭と肩が前に通り、その流れで掌底が出るようにする。
掌底は相手の顎、顔面下部、または胸元へ向かって、下から上へ迫る。

掌底は手のひら全体を見せるだけのポーズではなく、明確な掌底打ちとして描く。
打撃面は手のひらの付け根部分。
親指は必ず見せる。
指は自然に揃え、手首は適度に締める。

手のサイズは成人女性として自然な大きさに抑える。
手だけが不自然に巨大化しないようにする。
手は21歳の女性らしく、滑らかで皺の少ない表現にする。
老けた手、皺の多い手、節張った手、男性的な手にはしない。
手だけ妙にリアルにせず、顔や身体のアニメ調に馴染ませる。

手のひらが平坦に見えないように、指の付け根、親指の付け根、掌底部分、手の中心のくぼみに柔らかい陰影を入れる。
指同士の境目を薄く分かるようにする。
手を細長くしすぎず、掌には自然な丸みと厚みを持たせる。
```

## 掌底の手だけを修正する場合

```prompt
掌底の手だけを修正してください。
現在の構図、顔、髪型、衣装、ポーズ、背景、表情、全体の雰囲気は維持してください。

手のひらが平坦に見えないように、指の付け根、親指の付け根、掌底部分、手の中心のくぼみに柔らかい陰影を入れてください。
掌底として自然に見えるように、打撃面である手のひらの付け根部分に軽い厚みと立体感を持たせてください。
親指の付け根を自然に見せ、親指が手のひらから消えたり、貼り付いたように見えたりしないようにしてください。
手のサイズは成人女性として自然な大きさに抑えてください。
手だけが写実的になりすぎないようにし、顔や身体と同じアニメ調の絵柄に馴染ませてください。
```
