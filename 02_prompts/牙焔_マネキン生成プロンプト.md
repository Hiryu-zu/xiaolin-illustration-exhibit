# 牙焔 マネキン生成プロンプト

牙焔の戦闘構図を検討するための、白い3Dデッサン人形生成用プロンプト。

使用する参考画像:

- 牙焔の正本立ち絵
- 必要に応じて牙焔の三面図または既存戦闘画像

参考画像は外見を再現するためではなく、体格、頭身、肩幅、手足の長さを読み取るために使用する。

## 01 体格マネキン四面図

```text
添付した牙焔の立ち絵を参考に、戦闘構図検討用の白い3Dデッサン人形四面図を作成してください。

【参考画像の役割】
添付画像は、牙焔の身長感、頭身、肩幅、胸郭、胴回り、骨盤、腕と脚の長さを読み取るための体格資料です。
髪型、帽子、角、衣装、コート、装飾、炎、色はマネキンへ引き継がないでください。

【出力形式】
・一枚の横長モデルシート
・横4列、縦1行の均等な4パネル
・左から、正面、右側面、背面、左側面
・各パネルに同一人物のマネキンを一体
・全パネルで同じ身長、頭身、肩幅、胸郭、骨盤、手足の長さ
・カメラ距離、人物サイズ、床の高さを統一する
・人物同士を重ねない
・頭頂から足先まで完全に収める
・薄い縦線でパネルを区切る
・文字、番号、方向名は画像内へ入れない

【体格】
・20代前半の成人男性型
・自然なアニメキャラクター向けの約7頭身
・高身長に見える縦長の身体比率
・中肉中背
・標準からやや引き締まった体格
・肩幅は標準からやや広め
・胸郭は自然な厚み
・胴回りと腰は太くしない
・腕は引き締まっているが太すぎない
・太腿とふくらはぎは格闘動作を支えられる自然な太さ
・手足を極端に大きくしない
・重量級格闘家やボディビルダーにしない
・細身の美少年体型や中性的な体格にしない

【マネキンの造形】
・白または明るい灰色のマット樹脂製3Dデッサン人形
・写実的な皮膚や毛穴を描かない
・頭部は簡略化された成人男性型
・頭部、胸郭、骨盤に薄い中心線
・肩、肘、手首、股関節、膝、足首の関節位置を明確にする
・筋肉の境界は控えめにし、関節と身体比率を優先する
・手は五本指を簡潔に分ける
・両足は閉じた靴型の簡略形状

【立ち姿】
・自然な直立姿勢
・頭を水平に保つ
・顎を軽く引く
・肩の力を抜く
・両腕を身体の横へ自然に下ろす
・手の甲を外側へ向ける
・両脚は腰幅から肩幅程度
・両膝と足先を正面へ向ける
・左右の足へ均等に重心を置く
・骨盤や肩を傾けない

【背景と照明】
・均一な薄いグレー背景
・水平な薄いグレーの床面
・足元に弱く自然な接地影
・柔らかなスタジオ照明
・輪郭と関節が背景へ溶けない陰影
・クリーンな3D人体デッサン教材

【禁止事項】
髪、帽子、角、衣装、コート、手袋、装飾品、炎、武器、敵、
表情差分、ポーズ差分、複数の体格、左右非対称な身体、
巨大な胸板、太すぎる首、太い胴体、過剰な筋肉、
細すぎる腰、長すぎる脚、短すぎる腕、
余分な腕や脚、六本指、融合した指、欠けた指、
曲がった直立姿勢、異なるカメラ距離、広角歪み、
文字、番号、ロゴ、透かし
```

## 02 基本格闘構え 八方向

このプロンプトでは、完成した体格マネキン四面図を参考画像として添付する。

```text
添付した牙焔用の白い3D体格マネキンを使い、同じ基本格闘構えを八方向から確認できるポーズ資料を作成してください。

【参考画像の役割】
添付したマネキン四面図は、身長、頭身、肩幅、胸郭、骨盤、腕と脚の長さの正本です。
全8パネルで、この体格を変更しないでください。

【最重要】
8種類の異なる構えを作るのではありません。
一つの同じ格闘構えを固定し、カメラだけを人物の周囲へ45度ずつ移動させてください。
支持脚、足幅、手の高さ、肘の角度、重心、胸郭と骨盤の向きを全パネルで維持してください。

【出力形式】
・一枚の16:9横長モデルシート
・上段4パネル、下段4パネル
・上段左から01、02、03、04
・下段左から05、06、07、08
・各パネルにマネキン一体
・各パネル内に全身、両手、両足を収める
・手足をパネル境界で切らない
・カメラ距離と人物の大きさを統一する
・細い境界線でパネルを区切る
・各パネル左上に小さく01から08の番号だけを表示する

【カメラ方向】
01：正面
02：右前方45度
03：右側面
04：右後方45度
05：背面
06：左後方45度
07：左側面
08：左前方45度

【基本格闘構え】
・右利きを想定した半身の構え
・左脚を前、右脚を後ろへ置く
・両足の間隔は肩幅より少し広い
・前後の足を一直線へ重ねない
・左足は前方へ向ける
・右足は斜め外側へ向ける
・両膝を軽く曲げる
・重心を両脚の中央より少し低い位置へ置く
・両足裏を床へ接地する
・前脚へ体重を乗せすぎない
・後脚へ逃げすぎない

・骨盤は真正面ではなく斜めに構える
・胸郭は骨盤よりわずかに正面へ戻す
・背骨を自然に立てる
・上体を前後左右へ倒しすぎない
・肩を上げない
・顎を軽く引く

・左拳を前方、顎から胸の高さへ置く
・右拳を頬から肋骨付近へ置く
・両拳を顔へ密着させない
・両肘を軽く曲げる
・脇を完全に開かない
・拳の手首を前腕の延長上へ置く
・親指は四本の指の外側へ添える
・拳から指を突き出さない

・顔と視線は常に構えの正面方向を見る
・カメラへ視線を向けない
・背面カメラでも首を不自然に振り返らせない

【動作の印象】
・攻撃開始前の静かな構え
・低い重心
・前進にも防御にも移れる中間姿勢
・威嚇のポーズではなく、実用的な格闘姿勢
・静7、動3

【マネキン表現】
・白または明るい灰色のマット樹脂製3Dデッサン人形
・成人男性型、約7頭身
・牙焔の四面図と同じ中肉中背の引き締まった体格
・頭部、胸郭、骨盤に薄い中心線
・肩、肘、手首、股関節、膝、足首を明確にする
・両手は正しい五本指構造
・両足は閉じた靴型

【背景】
・均一な薄いグレー背景
・薄い床面
・弱い接地影
・柔らかなスタジオ照明
・クリーンな3D人体デッサン教材風

【禁止事項】
異なる8種類のポーズ、攻撃中の姿勢、蹴り、跳躍、炎、武器、
髪、帽子、角、衣装、コート、手袋、装飾、
左右の脚を途中で入れ替える、拳の高さを変える、重心を変える、
カメラへ顔を向ける、背面で振り返る、
前後の足を交差させる、浮いた足、伸び切った膝、
折れた手首、逆関節、余分な腕や脚、六本指、融合した指、
パネルごとに異なる体格、異なる頭身、異なるカメラ距離、
番号以外の文字、技名、ロゴ、透かし
```

## 生成後チェック

### 四面図

- [ ] 四体が同じ頭身、肩幅、胴回りか。
- [ ] 左右側面の胸郭と骨盤の厚みが一致しているか。
- [ ] 重量級や細身体型へ寄っていないか。
- [ ] 膝、肘、股関節の高さが四面で一致しているか。

### 基本構え

- [ ] 全パネルで左脚が前、右脚が後ろか。
- [ ] ポーズではなくカメラだけが回転しているか。
- [ ] 足幅、拳の高さ、肘の角度が維持されているか。
- [ ] 両足が接地し、低い重心が成立しているか。
- [ ] 顔がカメラではなく構えの正面を見ているか。
- [ ] 拳、手首、指の構造が正しいか。
