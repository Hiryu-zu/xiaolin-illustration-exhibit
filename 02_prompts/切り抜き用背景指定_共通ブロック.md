# 切り抜き用背景指定 共通ブロック

キャラクター画像を後で透過素材化・アニメーション素材化するための背景指定。

マネキン検証用と、完成キャラクター切り抜き用では背景指定を分ける。

## 01 マネキン検証用

重心、足裏接地、床面、アイレベルを確認したい段階。
切り抜きやすさよりも、身体の成立を優先する。

```text
【背景】
・均一な薄いグレー背景
・水平な薄いグレーの床面
・足元に弱く自然な接地影
・柔らかなスタジオ照明
・輪郭と関節が背景へ溶けない陰影
・床面と接地影で足裏の位置と重心を確認できるようにする
・背景に模様、線、文字、家具、風景、敵、物体を入れない
```

## 02 切り抜き素材用

キャラクター化後、動画・GIF・別背景合成に使いたい段階。
切り抜きやすさを最優先する。

```text
【切り抜き用背景】
・完全な無地の白背景
・背景は純白に近い単色
・床、地平線、接地影、落ち影、グラデーション、模様を描かない
・人物の周囲に白い縁取りや発光を付けない
・髪、袖、前垂れ、靴の輪郭が背景と混ざらないよう、線画を明瞭にする
・全身、髪、両手、両足、衣装の裾を画面内へ収める
・手足や髪を画面端で切らない
・人物と背景を融合させない
・文字、番号、ロゴ、透かしを入れない
```

## 03 透明背景を狙う場合

画像生成側が透明PNGに対応する場合だけ使う。
対応していない場合は、02の白背景を使う。

```text
【透明背景】
・背景を完全に透明にする
・人物だけを透過PNGとして描く
・床、地平線、接地影、落ち影、白背景、グレー背景を描かない
・人物の輪郭を明瞭にし、半透明の背景残りを作らない
・全身、髪、両手、両足、衣装の裾を画面内へ収める
```

## 運用メモ

- マネキン段階では `01 マネキン検証用` を使う。
- キャラクター化して素材化する段階では `02 切り抜き素材用` を使う。
- 透明PNGが確実に出せる環境だけ `03 透明背景` を試す。
- 小鈴は白い衣装が背景と近いため、白背景でも黒い縁取りと線画を強めに指定する。
