# 牙焔 マネキン経由戦闘構図ワークフロー

## 目的

牙焔の戦闘構図を、固有技や炎エフェクトからではなく、体格、重心、関節、基本動作から再設計する。

制作順は次の通り。

1. 立ち絵から体格を抽出し、マネキン四面図を作る。
2. 基本構えを八方向から確認する。
3. 打撃、蹴り、受け、移動、回転、空中動作をマネキンで網羅する。
4. 良い構図を牙焔へキャラクター化する。
5. 最後に固有技と炎エフェクトを加える。

固有技を先に作ると、技名とエフェクトに身体が引っ張られ、足首、指、支持脚、武器の接続が崩れやすい。先に「牙焔が自然に取れる身体動作の辞書」を作り、その上へ技を構築する。

## 資料の役割

- **牙焔の立ち絵**: 顔、年齢、体格、頭身、衣装、帽子、角、配色の正本。
- **マネキン四面図**: 衣装を外した身体比率と関節位置の正本。
- **ポーズマネキン**: 姿勢、支持脚、重心、カメラ方向、手足の前後関係の正本。
- **戦闘構図ガイド**: 遠近、武器、指、エフェクト、生成後検品の補助資料。
- **文章プロンプト**: 動作の目的、瞬間、視線、運動方向、禁止事項を定義する。

## 第1段階 体格マネキン四面図

## 1. 立ち絵から抽出する要素

衣装の見た目ではなく、その下にある身体を推定する。

- 20代前半の成人男性。
- 高身長だが重量級ではない。
- 自然なアニメ調の約7頭身。
- 中肉中背から、やや引き締まった体格。
- 肩幅は標準からやや広め。
- 胸板、腕、太腿を過剰に太くしない。
- 胴回りは細めに保ち、縦長のシルエットを作る。
- ボディビルダー、重量級格闘家、細身の美少年にしない。
- 強さは筋肉量ではなく、骨盤の安定、姿勢、鋭い視線で表す。

## 2. 四面図の構成

一枚の横長資料へ、同じマネキンを四体並べる。

1. 正面
2. 右側面
3. 背面
4. 左側面

すべて自然な直立姿勢とし、カメラ距離、人物サイズ、足位置を統一する。

### 四面図プロンプト

```text
添付した牙焔の立ち絵から、衣装の下にある身体比率を抽出し、
同一人物の白い3Dデッサン人形四面図を作成する。

一枚の横長画像を4分割し、左から正面、右側面、背面、左側面。
全パネルで同じ頭身、身長、肩幅、胸郭、骨盤、手足の長さを維持する。
人物の大きさ、足元の高さ、カメラ距離をそろえる。

成人男性、20代前半、自然なアニメ調約7頭身。
高身長だが重量級ではない。
中肉中背で、標準からやや引き締まった体格。
肩幅は標準からやや広め、胴回りは太くしない。
腕、胸、太腿をボディビルダーのように肥大させない。

白いマット樹脂製の3Dデッサン人形。
頭部、胸郭、骨盤に薄い中心線。
肩、肘、手首、股関節、膝、足首を明確にする。
髪、帽子、角、衣装、コート、装飾、炎は付けない。
両手は自然に下ろし、両足は肩幅程度。
薄いグレー背景、弱い床面、薄い接地影。
```

### 四面図チェック

- [ ] 四体が同一人物の体格か。
- [ ] 正面と背面で肩幅が一致しているか。
- [ ] 左右側面で胸郭、腹部、骨盤の厚みが一致しているか。
- [ ] 頭身、腕の長さ、股下、膝位置が一致しているか。
- [ ] 筋肉質、重量級、細身へ過剰に変化していないか。
- [ ] 左右の肩、骨盤、膝、足首の高さがそろっているか。

四面図を承認するまで、戦闘ポーズへ進まない。

## 第2段階 基本構えの八方向

## 1. 八方向の定義

カメラを人物の周囲へ45度ずつ移動させる。

|番号|方向|主な確認点|
|---|---|---|
|01|正面|顔、拳、両脚、左右差|
|02|右前方45度|前後の腕、胸郭と骨盤のひねり|
|03|右側面|重心、歩幅、背骨の傾き|
|04|右後方45度|肩甲骨、後脚、拳の奥行き|
|05|背面|背骨、骨盤、支持脚|
|06|左後方45度|反対側の肩と後脚|
|07|左側面|重心と進行方向|
|08|左前方45度|主構図候補、顔と手足の前後差|

鏡像反転ではなく、各方向で関節と前後関係を再構築する。

## 2. 最初の基本構え

技を出していない、牙焔の基準となる格闘構えを作る。

- 足幅は肩幅より広い。
- 利き手側を少し後ろへ置く半身。
- 両膝を軽く曲げ、重心を低くする。
- 前拳は顎から胸の高さ。
- 後拳は頬から肋骨付近。
- 肩を上げず、肘を身体から離しすぎない。
- 顎を軽く引き、視線は前方。
- 両足の踵を完全に一直線へ並べない。

### 八方向プロンプト

```text
承認済みの牙焔体格マネキンを使い、同じ基本格闘構えを八方向から示す。
一枚の比較用モデルシートに8パネルで配置する。
各パネルは同一ポーズを、人物の周囲45度刻みのカメラから見る。

ポーズを回転させず、カメラだけを移動する。
全パネルで支持脚、手の高さ、足幅、重心、頭身を維持する。
髪、衣装、帽子、角、炎、武器は付けない。
```

## 第3段階 基本動作の網羅

技名を付けず、運動の種類として作る。

## 動作カテゴリ

### A. 拳撃

1. 前手の直突き
2. 後手の直突き
3. 左右の鉤突き
4. 下からの突き上げ
5. 肘打ち
6. 掌底

確認点:

- `足 -> 骨盤 -> 背骨 -> 肩 -> 肘 -> 拳または掌` が連続している。
- 手首を折らない。
- 突く腕と引く腕を分ける。
- 拳だけを前景へ巨大化しない。

### B. 蹴り

1. 前蹴り
2. 横蹴り
3. 下段回し蹴り
4. 中段回し蹴り
5. 後ろ蹴り
6. 膝蹴り
7. 踏みつけ

確認点:

- 支持脚を明確にする。
- 膝、脛、踵、つま先を同じ運動面へ置く。
- 外へ振る足の裏を内側へ反転させない。
- 空中技でなければ軸足を浮かせない。

### C. 防御と受け

1. 上段受け
2. 外受け
3. 内受け
4. 両腕の交差防御
5. 掌での受け流し
6. 身体を沈める回避

確認点:

- 左右の腕を融合させない。
- 掌、手首、肘、肩の順に接続を確認する。
- 防御面の後ろに顔を残す。

### D. 移動

1. 前進
2. 後退
3. 左右への横移動
4. 斜め前への踏み込み
5. 斜め後ろへの退避
6. 急停止
7. 低い姿勢からの加速

確認点:

- 前進と後退を骨盤位置で区別する。
- 足を不自然に交差させない。
- 接地足、離地する足、次に着く足を明記する。

### E. 回転

1. 方向転換
2. 回転拳
3. 回転肘
4. 回転蹴り
5. 背面から正面への振り返り

確認点:

- 軸足を固定する。
- 軸足、骨盤、胸郭、肩、手足の回転方向をそろえる。
- 顔と視線を身体より少し先行させる。

### F. 高低差と空中

1. 深く沈む構え
2. しゃがみ回避
3. 跳躍開始
4. 空中の上昇
5. 空中の下降
6. 着地
7. 落下攻撃の構え

確認点:

- 上昇、頂点、下降、着地を一つの姿勢へ混在させない。
- 両脚を同じ平面へ重ねない。
- 曲げる脚と伸ばす脚の役割を分ける。

### G. 組みと武器

固有技より前に必要な場合だけ追加する。

1. 片手で掴む
2. 両手で押さえる
3. 短い鎖を握る
4. 鎖を前方へ投げる
5. 鎖を引き戻す

敵は描かず、掴む位置や鎖の経路だけを示す。

## 動作と方向の増やし方

すべての動作を最初から八方向で作ると量が多すぎる。以下の順で増やす。

### 第1確認

各動作を主構図一枚で作る。

- 直線動作: 前方45度。
- 横方向動作: 側面または斜め前。
- 回転動作: 後方45度または斜め上。
- 接地確認: 正面または側面。

### 第2確認

採用候補だけ四方向へ展開する。

1. 正面
2. 側面
3. 背面
4. 斜め前

### 第3確認

左右差や死角が重要な動作だけ八方向へ展開する。

優先して八方向化する動作:

- 基本構え
- 回転蹴り
- 方向転換
- 空中姿勢
- 鎖を扱う姿勢

## 第4段階 キャラクター化

マネキンで成立した構図だけを牙焔へ変換する。

### 参考画像の役割

```text
参考画像1: 牙焔の正本立ち絵。顔、年齢、体格、髪、帽子、角、衣装、配色の正本。
参考画像2: 承認済みマネキン。姿勢、支持脚、重心、関節、手足の前後関係、カメラ方向の正本。
```

### キャラクター化で固定するもの

- 20代前半の若い男性。
- 高身長だが太い重量級ではない。
- 黒髪が片目へかかる。
- 目深の帽子と短い鬼角。
- 黒い学ラン風ロングコートと深紅のライン。
- 手袋とブーツ。
- 長いコート裾。
- 角を巨大化しない。
- 翼、尻尾、獣耳を追加しない。

### コートの扱い

マネキンの関節を衣装で壊さない。

- 肩、肘、手首の位置をコートや手袋で曖昧にしない。
- コート裾は身体より少し遅れて運動方向の逆側へ流す。
- 前後の脚を裾で完全に隠さない。
- 回転時は裾を一つの大きな弧へまとめる。
- 裾を脚や炎と融合させない。

### 出力形式

完成画像は一枚につき一人、一動作。

```text
複数の動作を一枚へ並べない。
モデルシート、グリッド、分割パネル、コラージュにしない。
16:9横長、均一な白背景、敵なし。
```

## 第5段階 固有技への展開

固有技は、承認済みの基本動作を土台にする。

```text
固有技 = 基本動作 + 重心変化 + 遠近 + エフェクト + 技固有の手または武器
```

例:

- 炎脚・回転蹴り = 中段回し蹴り + 回転の遠近 + 脚に沿う炎。
- 瞬火・高速接近 = 低い加速姿勢 + 斜め前への踏み込み + 後方へ抜ける炎。
- 炎獣形意拳・顎門 = 上下の開いた掌 + 正面の挟撃構図 + 顎状の炎。
- 獄炎鎖 = 鎖を握る姿勢 + 一本の鎖経路 + 炎への変換。

基本動作の身体を変えず、エフェクトだけを追加する。


## 生成後チェック

### 体格

- [ ] 四面図と同じ頭身、肩幅、胴回りか。
- [ ] 重量級やボディビルダーへ変化していないか。
- [ ] 細身、美少年体型へ変化していないか。

### 身体

- [ ] 支持脚と重心が成立しているか。
- [ ] 膝と足先が同じ方向か。
- [ ] 肩、肘、手首が自然につながっているか。
- [ ] 骨盤、膝、足首が自然につながっているか。
- [ ] 拳または掌は五本指構造として成立しているか。
- [ ] 顔と視線が動作方向を向いているか。

### 動作

- [ ] 前進と後退が明確に区別できるか。
- [ ] 回転方向が全身で一致しているか。
- [ ] 接地、上昇、下降、着地が混在していないか。
- [ ] 別の蹴りや別の防御動作へ変化していないか。

### キャラクター化

- [ ] 帽子、短い角、髪、目、コート、手袋、ブーツが維持されているか。
- [ ] コート裾が脚や関節を隠していないか。
- [ ] 衣装がマネキンの重心や関節を変えていないか。
- [ ] 炎が顔、指、足首、接続部を隠していないか。

## 推奨制作順

1. 体格マネキン四面図。
2. 基本構えの八方向。
3. 前手直突き、後手直突き、掌底。
4. 前蹴り、横蹴り、下段と中段の回し蹴り。
5. 上段受け、交差防御、受け流し。
6. 前進、後退、横移動、低姿勢加速。
7. 方向転換、回転拳、回転蹴り。
8. 跳躍、上昇、下降、着地。
9. 必要な動作だけ四方向または八方向へ展開。
10. 採用したマネキンを牙焔へキャラクター化。
11. 最後に固有技と炎、鎖を追加。
