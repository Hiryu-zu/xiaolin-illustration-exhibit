# 牙焔 実戦基礎動作 八方向プロンプト

キャラクター化する前に、[牙焔 マネキン実戦基礎動作 八方向プロンプト](./牙焔_マネキン実戦基礎動作_八方向プロンプト.md)で身体構造を確認する。
マネキン資料が完成した後は、[牙焔 マネキン実戦動作 キャラクター変換プロンプト](./牙焔_マネキン実戦動作_キャラクター変換プロンプト.md)を使用する。

## 現行の基本構え画像について

`GPT出力イラスト_牙焔`へ出力した8枚は、牙焔の顔、帽子、角、体格、衣装、ブーツの再現資料として使用できる。

ただし、カメラ方向は十分に八方向化されていない。

- 01から04は、ほぼ同じ斜め前の構図。
- 05は背面構図として有効。
- 06から08も、ほぼ同じ斜め前の構図。
- 足幅、拳の高さ、体格、衣装はおおむね安定。
- コート裾の間から両脚が確認でき、基礎資料として使いやすい。

次段階では、一つの静止構えを回転させるのではなく、八方向それぞれへ異なる実戦動作を割り当てる。

## 目的

牙焔に適した基本打撃と蹴りを、全周のカメラ方向から収集する。

これは固有技集ではない。炎、鎖、技名、魔法表現を加える前の、身体動作と構図の基礎資料である。

八方向で次の要素を網羅する。

- 正面から見た直線打撃。
- 斜め前から見た全身連動。
- 側面から見た踏み込みと間合い。
- 斜め後ろから見た肩、背骨、骨盤の回転。
- 背面から見た肘打ち。
- 反対側後方から見た膝蹴り。
- 反対側面から見た前蹴り。
- 斜め前から見た回転蹴り。

## 共通プロンプト

```text
添付した参考画像を使用し、牙焔の実戦的な基本打撃と蹴りを作成してください。

【最重要：出力方法】
01から08を、必ず8枚の独立した画像として一枚ずつ生成する。
一枚に複数動作を並べない。
モデルシート、グリッド、分割パネル、コラージュにしない。
一枚につき牙焔一人、一つの動作だけを描く。

【制作目的】
固有技や炎を加える前に、牙焔の実戦的な身体動作を確認する。
威嚇用の決めポーズではなく、攻撃が相手へ届く途中または直前の瞬間を描く。
格闘技として支持脚、重心、間合い、全身連動が成立していることを優先する。

【牙焔の固定外見】
・20代前半の成人男性
・高身長だが重量級ではない
・自然なアニメ調約7頭身
・中肉中背、標準からやや引き締まった体格
・肩幅は標準からやや広め
・胴回り、胸板、腕、太腿を過剰に太くしない
・黒い短髪、片目へかかる前髪
・鋭い赤い目
・不機嫌で集中した表情
・目深に被った黒い帽子
・帽子から突き出る短い鬼角二本
・黒基調の学ラン風ロングコート
・深紅のライン
・黒い手袋
・黒い編み上げブーツ

【身体動作】
・攻撃の力を、接地足から骨盤、背骨、肩または股関節へ連続させる
・支持脚を明確にする
・膝とブーツのつま先を同じ方向へ向ける
・肩、肘、手首を自然につなげる
・股関節、膝、足首を自然につなげる
・上体だけで攻撃せず、骨盤と脚を使う
・攻撃していない手は顔または身体の中心線を守る
・顔と視線はカメラではなく攻撃の到達方向を見る

【拳と手袋】
・拳の親指を四本の指の外側へ添える
・拳から指を突き出さない
・手首を前腕の延長上へ置く
・左右の手袋を維持する
・手袋から素手の指を二重に描かない

【蹴りとブーツ】
・支持脚と蹴り脚の両方に同じブーツを維持する
・裸足や別の靴へ変更しない
・蹴り脚の膝、脛、踵、つま先を同じ運動面へ置く
・ブーツの輪郭で接触部位と足首の向きを示す
・足裏を内側へ反転させない

【コート】
・コートは身体より少し遅れて、運動方向の逆側へ流す
・肩、肘、骨盤、膝、足首をコートで完全に隠さない
・前後の脚が判別できるように裾を分ける
・裾を腕、脚、炎と融合させない
・回転動作では一つの大きな弧へまとめる

【画像仕様】
・16:9横長
・均一な白背景
・敵、標的、他の人物なし
・全身、帽子、角、両手、両足、コート裾を画面内に収める
・攻撃方向へ余白を取る
・床、接地影、風景、建物を描かない
・文字、番号、技名、ロゴ、透かしを入れない

【この段階では使わないもの】
炎、火花、熱歪み、オーラ、鎖、武器、爆発、固有技の演出。
まず身体動作だけを明確にする。

【禁止事項】
巨大な角、帽子なし、翼、尻尾、獣耳、長髪、
重量級、ボディビルダー、中年男性、細身の美少年、
剣、銃、鎧、炎、敵、背景風景、
余分な腕や脚、六本指、融合した指、二重の手、
折れた手首、逆関節、反転した足首、浮いた支持脚、
膝と足先の不一致、足とブーツの分離、
カメラ目線、身体から切断して見える前景の拳や足
```

## 生成後チェック

- [ ] 8枚が異なるカメラ方向になっているか。
- [ ] 正面、両側面、背面、四つの斜め方向が揃っているか。
- [ ] 各動作の支持脚と重心が成立しているか。
- [ ] 攻撃していない手が防御へ残っているか。
- [ ] 拳、掌、肘、膝、ブーツの接触部位が明確か。
- [ ] 膝とブーツのつま先が同じ方向か。
- [ ] コートで骨盤、支持脚、膝、足首が隠れていないか。
- [ ] 顔と視線が攻撃方向を向いているか。
- [ ] 炎や固有技の演出が混入していないか。
