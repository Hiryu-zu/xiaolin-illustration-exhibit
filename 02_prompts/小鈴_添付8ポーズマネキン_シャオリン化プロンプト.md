# 小鈴 添付8ポーズマネキン シャオリン化プロンプト

## 目的

添付された白い3Dマネキン8ポーズ画像を使い、各ポーズを小鈴（シャオリン）の完成イラストへ変換するためのプロンプト。

このプロンプトでは、マネキン画像の姿勢とカメラ方向だけを使い、体格・顔・髪・衣装・画風は小鈴へ置き換える。

## 添付する参考画像

1. 小鈴の正本立ち絵候補。
2. 小鈴の顔・表情・髪型が分かる参考画像。
3. 白い3Dマネキン8ポーズ画像。

必要なら、小鈴の基本構え画像や一枚絵成功画像を追加する。

## 共通プロンプト

```prompt
添付した白い3Dマネキン8ポーズ画像を、ポーズとカメラ方向の参考として使用してください。
指定した番号のマネキンポーズを、小鈴（シャオリン）の完成イラストとして再現してください。

【参考画像の役割】
小鈴の正本画像:
顔、見た目年齢、髪型、瞳、衣装、レギンス、靴、配色の正本。
キャラクターの外見は必ず小鈴の正本画像を優先する。

白い3Dマネキン8ポーズ画像:
指定番号の姿勢、カメラ方向、支持脚、重心、骨盤、胸郭、肩、肘、拳または掌、膝、足先の向きの参考。
男性的な筋肉量、坊主頭、白い樹脂素材、関節線、下着状の形状、番号、グリッド、パネル線は引き継がない。

【小鈴の固定外見】
小鈴（シャオリン）、21歳の女性。
身長165cm、自然なアニメ調の約7.3頭身。
しなやかで均整の取れた健康的な武術体型。
細すぎず、筋肉質すぎず、過剰にグラマーにしない。

黒髪ショートボブ。
前髪があり、顔の両側に自然なサイドロックがある。
艶のある黒髪。
琥珀色から黄褐色の瞳。
可愛さと凛々しさの中間の顔立ち。
戦闘時はやや鋭く集中した目つき。
冷酷な悪役顔や幼い丸目にしない。

白黒ベースの中華武術服。
白い立ち襟トップス、黒い縁取り、黒いチャイナボタン、黒いサイドパネル。
七分袖から長袖寄りの袖、袖口は黒い縁取り。
腰から前へ垂れる白い前垂れ、裾は黒い縁取り。
左右に短い黒い腰布またはサイドパネル。
黒レギンス。
白黒配色の中華武術シューズ。
金色装飾は少量のみ。

【ポーズ変換】
指定番号のマネキンと同じカメラ方向を維持する。
支持脚、足幅、重心、骨盤、胸郭、肩、肘、拳または掌、膝、足先の向きを参考にする。
攻撃していない手は顔または身体の中心線を守る。
顔と視線はカメラではなく、技の到達方向へ向ける。
上体だけ、腕だけ、脚だけのポーズにしない。
接地足から骨盤、胸郭、肩、腕または蹴り脚へ力がつながるようにする。

【小鈴向けの調整】
男性マネキンの強い筋肉量は引き継がない。
小鈴らしい軽やかな中華武術の足運びにする。
拳ポーズは必要に応じて掌、手刀、受け流しへ変換してよいが、指定番号の姿勢とカメラ方向は維持する。
前垂れや腰布で股関節、膝、支持脚、足先を完全に隠さない。
手足を画面端で切らない。

【画風】
高品質な日本のアニメ調デジタルイラスト。
ノベルゲーム用の戦闘イラスト。
繊細で明瞭な線画。
柔らかなセル塗り。
正本と同じ顔、体格、衣装、配色。

【出力】
一枚に小鈴一人だけを描く。
一つの番号、一つのポーズだけを描く。
グリッド、分割パネル、一覧画像、モデルシートにしない。
番号、技名、説明文、ロゴ、透かしを入れない。
敵、武器、派手な気功エフェクトを入れない。
```

## 01 正面ガード構え

```prompt
マネキン画像の01を参考にしてください。
正面カメラのガード構え。
片拳を前へ置き、反対の拳で顔と身体の中心線を守る。
両足を前後に開き、膝を軽く曲げ、重心を低く安定させる。
小鈴らしい軽やかな中華武術の構えに調整する。
```

## 02 斜め正面の直突き

```prompt
マネキン画像の02を参考にしてください。
斜め正面カメラの直突き。
片拳を肩から胸の高さへまっすぐ突き出す。
反対の拳は顔または胸の近くで守りを残す。
後ろ足から骨盤、胸郭、肩、肘、拳へ力がつながるようにする。
拳だけを巨大化させず、肩と肘の接続を残す。
```

## 03 側面の掌打・穿掌

```prompt
マネキン画像の03を参考にしてください。
側面カメラで、片腕を前方へ伸ばす掌打または穿掌の到達姿勢にする。
開いた掌を相手の中心へ差し込む。
指先、手首、肘、肩を自然な線へつなげる。
反対の手は胸の近くで守りを残す。
停止ジェスチャーではなく、武術の掌技として見せる。
```

## 04 背面寄りの突き

```prompt
マネキン画像の04を参考にしてください。
背面寄りのカメラで、片拳を前方へ突き出す。
肩甲骨、背中、骨盤の回転が分かるようにする。
反対の手は顔または胸の近くに残す。
首だけをカメラへ振り返らせない。
顔と視線は技の到達方向へ向ける。
```

## 05 背面ガード構え

```prompt
マネキン画像の05を参考にしてください。
真後ろカメラのガード構え。
背中、肩甲骨、骨盤、足幅が分かるようにする。
両手は顔と身体の中心線を守る位置に置く。
小鈴の髪や衣装で背中と肩の動きが完全に隠れないようにする。
```

## 06 膝上げ

```prompt
マネキン画像の06を参考にしてください。
膝上げの姿勢。
片脚を支持脚として床へ接地し、反対の膝を高く引き上げる。
両手は顔と身体の中心線を守る。
前蹴りに入る直前の準備姿勢として見せる。
支持脚の膝を軽く緩め、重心を支持脚上へ残す。
```

## 07 側面の前蹴り

```prompt
マネキン画像の07を参考にしてください。
側面カメラの前蹴り到達姿勢。
支持脚を床へ接地し、蹴り脚を前方へ伸ばす。
踵または靴底で前方を押す。
股関節、膝、足首、靴の向きを同じ進行面へそろえる。
両手は守りを残す。
横蹴りや回し蹴りにしない。
```

## 08 斜め正面の横蹴り

```prompt
マネキン画像の08を参考にしてください。
斜め正面カメラの横蹴り到達姿勢。
軸足を床へ接地し、蹴り脚を横方向へ伸ばす。
踵または足刀を接触部位にする。
骨盤を横へ開き、膝と靴のつま先を同じ方向へそろえる。
上体と両手でバランスを取り、舞踊的な大開脚にしない。
```

## ネガティブプロンプト

```prompt
男性体型にしない。
ボディビルダー体型にしない。
白いマネキン素材、関節線、坊主頭、下着状の形状を引き継がない。
番号、グリッド、パネル線、技名、説明文を入れない。
メイド服、制服、チャイナドレス、現代スポーツウェアにしない。
裸足、サンダル、ハイヒール、ブーツにしない。
敵、武器、派手なエフェクトを入れない。
ロングヘア、ツインテール、ポニーテールにしない。
手足を不自然に巨大化しない。
六本指、欠けた指、融合した指、折れた手首、逆関節、反転した足首、浮いた支持脚にしない。
```
