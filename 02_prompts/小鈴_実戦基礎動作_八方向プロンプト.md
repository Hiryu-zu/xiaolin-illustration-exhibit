# 小鈴 実戦基礎動作 八方向プロンプト

小鈴をキャラクター化する前に、白い3Dデッサン人形で実戦的な基本動作を確認するためのプロンプト。

## 使用する参考画像

1. 承認済みの小鈴用体格マネキン四面図。
2. 必要に応じて、小鈴の基本武術構えマネキン。

参考画像は体格、頭身、関節位置の正本として使用する。小鈴の髪や衣装はまだ使用しない。

## 一括生成プロンプト

```text
添付した小鈴用の白い3D体格マネキンを使い、実戦的な中華武術の基本動作8種類を、一枚のポーズ資料として作成してください。

【目的】
小鈴の完成イラストや固有技を制作する前に、支持脚、重心、関節、身体の回転、カメラ方向を確認するための資料です。
決めポーズではなく、攻撃や受け流しが成立する直前または途中の実用的な動作を描いてください。

【出力形式】
・一枚の16:9横長モデルシート
・上段4パネル、下段4パネル
・上段左から01、02、03、04
・下段左から05、06、07、08
・各パネルに白い3Dデッサン人形を一体だけ配置
・一体につき一つの動作だけ
・人物同士を重ねない
・各パネル内に頭頂、両手、両足を完全に収める
・手足をパネル境界で切らない
・全パネルで同じ体格、頭身、関節構造
・細い境界線でパネルを区切る
・各パネル左上に小さく01から08の番号だけを表示
・技名、説明文、方向名は画像内へ入れない

【マネキン】
・承認済みの小鈴用体格マネキンと同じ成人女性型
・21歳相当
・自然なアニメ調約7.3頭身
・しなやかで均整の取れた健康的な武術体型
・肩幅は標準からやや華奢
・細すぎず、筋肉質すぎない
・白または明るい灰色のマット樹脂素材
・頭部、胸郭、骨盤に薄い中心線
・肩、肘、手首、股関節、膝、足首を明確にする
・手は五本指を簡潔かつ明瞭に分ける
・両足は閉じた格闘靴型の簡略形状

【全動作共通】
・右利きを基本とする
・左脚を前、右脚を後ろにした構えを出発点とする
・攻撃または受け流しの力を、接地足から骨盤、背骨、肩または股関節へつなげる
・支持脚を明確にする
・重心線が支持脚または両足の間へ落ちるようにする
・攻撃していない手は顔または身体の中心線を守る
・顔と視線はカメラではなく技の到達方向を見る
・上体だけで動かず、脚と骨盤を使う
・人体として実行可能な関節角度にする

【構図補助】
・薄い床面と接地影を入れ、足裏の位置と重心を確認できるようにする
・遠近がある動作では、画面内にアイレベルを想定し、攻撃方向または身体の奥行き方向に消失点を置く
・前景の手足だけを巨大化させず、肩、肘、股関節、膝との接続を残す
・床面上の足位置、骨盤、胸郭、頭部の奥行き関係が読めるようにする
・攻撃動作は、足、膝、骨盤、背骨、肩、肘、手先または足先まで一本の力線としてつなげる

【開いた掌】
・四本指と親指一本の合計五本
・掌の向きと親指の位置を一致させる
・指を極端に開かない
・手首を折りすぎない

【拳】
・四本の指を内側へ握る
・親指を四本の指の外側へ添える
・拳から指を突き出さない
・手首を前腕の延長上へ置く

【蹴り】
・支持脚を床へ完全に接地させる
・支持脚の膝を軽く緩める
・蹴り脚の股関節、膝、足首を自然につなげる
・膝と足先を技として正しい方向へそろえる
・足裏を内側へ反転させない
・前蹴りと横蹴りを混同しない

【背景】
・各パネルは均一な薄いグレー背景
・水平な薄い床面
・弱い接地影
・柔らかなスタジオ照明
・クリーンな3D人体デッサン教材風

【01 正面：弓歩直突き】
カメラは人物の正面。
左脚を前、右脚を後ろに置く。
前脚を曲げ、後脚で床を押す。
右拳または左拳を相手の中心へ肩から胸の高さで突き出す。
反対の手は胸の近くへ引き、守りを残す。
正面からも両脚の前後差と骨盤の回転が分かるようにする。

【02 右前方45度：穿掌】
カメラは人物の右前方45度。
前方へ開いた掌を差し込むように突き出す。
掌根ではなく、指先から掌全体へ伸びる穿つ動作。
指先、手首、肘、肩を一つの線へつなげる。
反対の手は胸の近くで中心線を守る。
手首を反らしすぎず、掌の向きと親指の位置を明確にする。

【03 右側面：前蹴り】
カメラは人物の右側面。
右脚を支持脚にする。
左膝を先に引き上げ、左足の踵または足裏で前方を押す。
左股関節、膝、足首、踵を同じ進行面へ置く。
上体は蹴り方向と反対へわずかに傾けるが、反りすぎない。
右足裏を床へ完全に接地させ、重心線を右支持脚へ落とす。
両手で中心線を守る。
横蹴りや回し蹴りにしない。

【04 右後方45度：横蹴り】
カメラは人物の右後方45度。
軸足を接地し、蹴り脚を横方向へ伸ばす。
蹴り足の踵または足刀を接触部位にする。
骨盤を横へ開き、膝と足先を同じ方向へそろえる。
上体を蹴り脚と反対側へ自然に倒し、重心を軸足上へ残す。
両手は、顔を守る手とバランスを取る手に分ける。
舞踊的な大開脚にしない。

【05 背面：円掌受け】
カメラは人物の真後ろ。
片方の掌で外から内へ、または内から外へ円を描くように受け流す。
肩甲骨、背骨、骨盤の回転が分かるようにする。
反対の手は胸または顔の近くへ残す。
首だけをカメラへ振り返らせない。
視線は受け流す方向へ向ける。

【06 左後方45度：双掌押し】
カメラは人物の左後方45度。
両掌を前方へ押し出す。
左右の掌を同じ高さに完全固定せず、自然な前後差を付ける。
両肘を伸ばし切らない。
後足から骨盤、背骨、両肩、両掌へ力をつなげる。
顔は掌の間から技の方向を見る。

【07 左側面：退歩引化】
カメラは人物の左側面。
相手の力を引き込みながら一歩後退する動作。
後脚へ重心を移し、前脚を軽く引く。
両手は相手の力を下または斜め後ろへ導くように配置する。
前進攻撃に見せない。
背骨を保ち、腰だけを引かない。

【08 左前方45度：旋身換歩】
カメラは人物の左前方45度。
軸足を接地し、身体を回転させながら足位置を入れ替える途中。
軸足、骨盤、胸郭、肩、顔と視線を同じ回転方向へつなげる。
顔と視線は身体の回転より少し先行する。
片手は守り、片手は次の攻撃または受けへ移る位置。
回転しながらも支持脚と重心を明確にする。
跳躍や舞踊ポーズにしない。

【禁止事項】
髪、衣装、前垂れ、レギンス、靴、装飾、武器、敵、
キャラクターの顔や配色、完成イラスト風の衣装、
8体すべて同じポーズ、同じカメラ方向、方向の重複、
決めポーズ、威嚇姿勢、非実用的な大開脚、
余分な腕や脚、六本指、欠けた指、融合した指、
二重の手、折れた手首、逆関節、反転した足首、
浮いた支持脚、膝と足先の不一致、身体から切断して見える手足、
番号以外の文字、技名、ロゴ、透かし
```

## 生成後チェック

- [ ] 8体が同じ体格と頭身か。
- [ ] 正面、背面、左右側面、四つの斜め方向があるか。
- [ ] 各動作の支持脚と重心が成立しているか。
- [ ] 攻撃していない手が防御へ残っているか。
- [ ] 掌、拳、踵、足刀などの接触部位が明確か。
- [ ] 肩、肘、手首が自然につながっているか。
- [ ] 股関節、膝、足首が自然につながっているか。
- [ ] 手は五本指で、親指の位置が正しいか。
- [ ] 前蹴りと横蹴りが明確に区別されているか。
- [ ] 完成キャラクターの髪や衣装が混入していないか。
