# 牙焔 戦闘構図ガイド

牙焔の戦闘イラストを生成するときに、再現性の高かった構図と確認事項をまとめた資料。
ノベルゲーム用の横長画像と、後から背景へ合成する白背景素材を前提とする。

## 基本仕様

- 画面比率は原則 `16:9` の横長。
- 背景は均一な白。床、地平線、影、グラデーションを描かない。
- 敵や人型の標的は描かない。
- 黒、深紅、暗い赤を中心とし、黄色や橙色の炎は抑える。
- 帽子、角、片目を隠す黒髪、赤い目、ロングコート、手袋、ブーツを維持する。
- 技のエフェクトは使用してよいが、顔、手、足、関節を隠さない。

## 良かった構図

### 接地技と防御

- [爆炎踏破](../03_outputs/牙焔_戦闘04_爆炎踏破_白背景.png)
  - 高めの三四分視点で、踏み足、支持脚、骨盤の荷重関係が読みやすい。
  - 円形エフェクトを足元へ限定すると、全身の輪郭を保ちやすい。
- [炎鎧・受けの型](../03_outputs/牙焔_戦闘05_炎鎧受けの型_白背景.png)
  - 正面寄りの構図は、防御姿勢と表情を同時に見せやすい。
  - 交差腕は左右を独立して描き、手首、肘、肩の順で接続を確認する。

### 移動と空中動作

- [瞬火・高速接近](../03_outputs/牙焔_戦闘06_瞬火高速接近_白背景.png)
  - 低い三四分視点と対角線移動で速度を表現できる。
  - 前脚の膝と足先を進行方向へ揃え、接地手の掌を前下方へ向ける。
- [墜炎・空中急襲](../03_outputs/牙焔_戦闘09_墜炎空中急襲_白背景.png)
  - 拳、肩、背骨を一本の荷重線として描くと落下方向が明確になる。
  - 両脚は前後に分け、交差や内向きの足先を避ける。

### 正面型と大技

- [炎獣形意拳・顎門](../03_outputs/牙焔_戦闘07B_炎獣形意拳・顎門_正面_白背景.png)
  - 人物を正面中央へ置き、上下の腕と炎顎で縦方向の挟撃を表現する。
  - 上の掌は下向き、下の掌は上向き。両手が同じ空間を挟むようにする。
- [黒陽爆界](../03_outputs/牙焔_戦闘10_黒陽爆界_白背景.png)
  - 静止した人物と巨大な円形エフェクトの対比が奥義表現に向く。
  - 炎輪は人物の背後へ分離し、角、帽子、身体と接触させない。

## 武器の使い方

### 長い武器の原則

鎖、鞭、長布などは、複数の経路や交差を一度に指定すると分断、分岐、融合が起きやすい。
武器の経路は文章ではなく、一本の順序として定義する。

```text
握り手 -> 小さい連続リンク -> 中距離のリンク -> 大きい前景リンク -> 画面外
```

### 鎖の成功パターン

- [短鎖から炎へ変換](../03_outputs/牙焔_戦闘08A_獄炎鎖_短鎖炎変換_白背景.png)
  - 実物は握り手から5リンク程度に限定する。
  - 最後の金属リンクへ炎を接触させ、以降は純粋な炎へ変換する。
  - 腰の装飾鎖は隠すか、能動鎖から完全に分離する。
- [一本のC字物理鎖](../03_outputs/牙焔_戦闘08B_獄炎鎖_C字物理鎖_白背景.png)
  - 拳から自由端まで、交差しない一本の開いたC字に限定する。
  - 円、螺旋、二周目、身体の背後への回り込みを避ける。
- [遠近で画面外へ逃がす](../03_outputs/牙焔_戦闘08C_獄炎鎖_遠近突進_白背景.png)
  - 牙焔を奥側へ小さく置き、鎖をカメラ方向へ伸ばす。
  - 奥から手前へリンクを段階的に大きくする。
  - 手前の3から5リンクを大きく描き、自由端を画面下へ出す。
  - 全長や終端を描かないことで、接続問題を減らしながら迫力を出せる。

### 鎖で避けること

- 握っている鎖と旋回している鎖を別々に描く。
- 一本の鎖を身体の前後で何度も交差させる。
- 能動鎖と腰の装飾鎖を近接させる。
- 炎でリンクを完全に隠し、接続位置を曖昧にする。
- 画面内に複数の自由端を作る。

## 遠近の使い方

### 奥から手前

- 人物を上側または奥側へ小さく配置する。
- 武器、拳、足など主役となる部位を画面下側の前景へ伸ばす。
- 大きさは急に変えず、奥、小、中、大の順で段階的に変化させる。
- 前景物は人物の顔を完全に隠さない位置へ逃がす。
- 進行方向へ炎、火花、コート裾、髪、チェーンを揃える。

### 横方向の運動

- 身体を画面中央から少し外し、技が向かう側へ余白を取る。
- コート裾とエフェクトは、身体の運動方向と逆側へ流す。
- 回転技では、腰、肩、脚、エフェクトの回転方向を統一する。

### ローアングル

- 突進、落下、奥義に有効。
- 手前の部位を大きくしても、肩や股関節との接続線を残す。
- 帽子と角をフレーム内へ収め、顔を拳や炎で完全に隠さない。

## 身体の向き

### 立ち姿

- 足幅は肩幅以上。膝は足先と同じ方向へ向ける。
- 骨盤の向きと胸郭のひねりを分け、背骨を無理に折らない。
- 支持脚の足裏または母趾球を明確に接地させる。
- 技の力が `足 -> 骨盤 -> 背骨 -> 肩 -> 腕` と連続するようにする。

### 突進

- 前脚は膝と足先を進行方向へ揃える。
- 後脚は股関節から足先まで一方向に伸ばす。
- 上体だけを前へ倒さず、骨盤ごと移動方向へ乗せる。

### 回転蹴り

- 蹴り足の膝、脛、踵、つま先が同じ回転面に乗るようにする。
- 外側へ振る蹴りでは、足裏や踵が不自然に内側へ反転しないこと。
- 軸足、骨盤、胸、蹴り足の順に回転方向を確認する。
- 生成後は左右反転したような足首や、膝と足先の不一致を必ず確認する。

### 空中姿勢

- 両脚を同じ平面へ重ねず、前後差を作る。
- 曲げる脚と伸ばす脚の役割を分ける。
- コート裾と髪は落下方向と逆へ流す。

## 手と指

### 拳

- 親指は四本の指の外側に添え、拳の内側へ埋め込まない。
- 手首を前腕の延長上へ置き、上下左右へ折りすぎない。
- 手前へ突き出す拳は顔を完全に隠さない大きさにする。

### 開いた掌

- `四本指 + 親指一本 = 合計五本`を明記する。
- 掌の向きと親指の位置をセットで指定する。
- 指を鉤状にする場合も、人間の手のままにし、獣の前脚へ変形させない。
- 手の周辺はエフェクトを薄くして、指の輪郭を確認できる余白を作る。

### 顎門の手

```text
上の手: 頭上、掌は下向き、指先は挟撃中心へ
下の手: 腰下、掌は上向き、指先は同じ挟撃中心へ
```

- 両掌が同じ方向を向かないようにする。
- 手の後ろに残像、炎の指、別の素手を置かない。
- 生成後は上下それぞれの指を目視で数える。

### 鎖を握る手

- 終端リンクを掌の中へ通し、拳の外側から次のリンクを出す。
- 鎖が手首や袖から生えて見えないようにする。
- 握り手から最初の3リンクまでは、炎を弱くして接続を見せる。

## エフェクト

- エフェクトは動作の原因と結果を補強するために使う。
- 炎の流れを身体、武器、コートと同じ運動方向へ揃える。
- 接続確認が必要な手、足、武器の根元ではエフェクトを薄くする。
- 大技の円形エフェクトは人物の背面へ配置し、前後関係を固定する。
- 炎獣表現は頭部全体より、顎、牙、爪など一部分へ限定すると読みやすい。

## 失敗しやすい条件

- 長い物体を複数回ループさせる。
- 腕、鎖、炎など似た線状要素を同じ場所へ重ねる。
- 「高速連打」を複数の実体腕で表現する。
- 手をエフェクトで隠したまま、複雑な指形を要求する。
- 回転蹴りで足先の向きを指定しない。
- 接地、空中、突進など複数の状態を一つの姿勢へ混在させる。

## プロンプト記述テンプレート

```text
画面:
16:9横長、均一な白背景、敵なし。

身体:
人物の位置、カメラ角度、胸と骨盤の向き、支持脚、進行方向を指定。

手足:
左右を明記し、肩・肘・手首、股関節・膝・足首の順で向きを指定。
開いた手は四本指と親指一本。膝と足先は同じ方向。

武器:
握り手から終端まで、一本の連続経路を矢印順で指定。
交差、分岐、背後への回り込み、腰への接続は禁止。

遠近:
奥の人物 -> 小さい武器要素 -> 中距離 -> 大きい前景 -> 画面外。

エフェクト:
武器や身体の運動方向へ沿わせる。
顔、手、足、接続部を隠さない。
```

## 生成後チェックリスト

- [ ] 人物は一人だけか。
- [ ] 帽子、角、髪、目、コート、手袋、ブーツが維持されているか。
- [ ] 左右の腕と脚が正しい肩、股関節へ接続されているか。
- [ ] 手は左右それぞれ五本指か。
- [ ] 親指が掌の向きと一致しているか。
- [ ] 膝と足先が同じ方向か。
- [ ] 接地足、支持脚、重心が成立しているか。
- [ ] 武器は握り手から一本で連続しているか。
- [ ] 武器の自由端が複数存在しないか。
- [ ] 武器が腰、袖、身体から生えていないか。
- [ ] 前景物が顔を完全に隠していないか。
- [ ] エフェクトが関節や武器の接続部を隠していないか。
- [ ] 白背景に床、影、灰色のムラが混入していないか。
