# 牙焔 マネキン実戦動作 キャラクター変換プロンプト

白い3D体格マネキンで作成した実戦基礎動作8種類を、同じ姿勢とカメラ方向のまま牙焔へ変換するためのプロンプト。

## 添付する参考画像

1. 牙焔の正本立ち絵。
2. 牙焔の基本格闘構え画像。
3. 白い3Dマネキンの実戦基礎動作8種類一覧。

必要に応じて、牙焔の体格マネキン四面図も追加する。

## 共通変換プロンプト

```text
添付した参考画像を使用し、白い3Dマネキンの実戦基礎動作01から08を、牙焔の戦闘イラストとして再現してください。

【参考画像の役割】
参考画像1：牙焔の正本立ち絵
・顔、見た目年齢、髪型、目、帽子、角、衣装、手袋、ブーツ、配色の正本
・キャラクターの外見は必ずこちらを優先する

参考画像2：牙焔の基本格闘構え画像
・戦闘時の顔、体格、衣装の見え方、コートの構造、低い重心の補助資料

参考画像3：白い3Dマネキンの実戦基礎動作8種類
・指定番号の姿勢、カメラ方向、支持脚、重心、骨盤、胸郭、手足の前後関係、接触部位の正本
・マネキンの顔、白い材質、関節線、簡略化された手足は引き継がない
・指定番号以外の動作を混合しない

【最重要：出力方法】
01から08を、必ず8枚の独立した画像として一枚ずつ順番に生成する。

1枚目：01 左前手直突き
2枚目：02 右後手直突き
3枚目：03 左踏み込み掌底
4枚目：04 右フック
5枚目：05 左後ろ肘
6枚目：06 右膝蹴り
7枚目：07 左前蹴り
8枚目：08 右中段回し蹴り

一枚に複数動作を並べない。
モデルシート、グリッド、分割パネル、一覧表、コラージュにしない。
一枚につき牙焔一人、一つの動作だけを描く。
各画像を単独で使用できる完成イラストにする。

【ポーズの再現】
・指定番号のマネキンと同じカメラ方向を維持する
・指定番号の支持脚、足幅、重心を維持する
・骨盤と胸郭の向きを維持する
・肩、肘、手首、拳または掌の位置を維持する
・股関節、膝、足首、接触部位を維持する
・攻撃していない手の防御位置を維持する
・顔と視線を攻撃の到達方向へ向ける
・衣装の都合で関節や重心を変更しない
・マネキンに解剖学的な小さな問題がある場合のみ、技の意味を変えず自然に修正する

【牙焔の固定外見】
・20代前半に見える成人男性
・若いが幼くない、荒々しさのある無愛想な青年
・高身長だが重量級ではない
・自然なアニメ調約7頭身
・中肉中背、標準からやや引き締まった体格
・肩幅は標準からやや広め
・胸板、腕、胴、太腿を過剰に太くしない
・細身の美少年、重量級格闘家、ボディビルダー、中年男性へ変更しない

・黒い短髪
・前髪が片目へ自然にかかる
・鋭い赤い目
・不機嫌で集中した戦闘表情
・目深に被った黒い帽子
・帽子から短い鬼角が二本だけ突き出る
・角を巨大化させない
・帽子を消さない

【衣装】
・黒基調の学ラン風特攻服
・深紅のラインとパイピング
・長いロングコート
・黒いフィンガーレス手袋
・黒い編み上げブーツ
・ベルトと腰の装飾鎖
・正本にない鎧、武器、翼、尻尾、獣耳を追加しない

【拳と手袋】
・拳は四本の指を内側へ握る
・親指を四本の指の外側へ添える
・拳から指を突き出さない
・手首を前腕の延長上へ置く
・左右とも手袋を維持する
・手袋と素手を二重に描かない
・攻撃していない拳は顔または身体の中心線を守る

【掌底】
・四本指と親指一本の合計五本
・掌の向きと親指の位置を一致させる
・掌根を接触部位として前方へ向ける
・指を極端に開かない
・手首を反らしすぎない
・手袋を維持したまま掌の形を判読できるようにする

【蹴りとブーツ】
・支持脚と蹴り脚の両方に同じ黒い編み上げブーツを維持する
・裸足、足指、異なる靴を描かない
・蹴り脚の股関節、膝、足首を自然につなげる
・ブーツの輪郭で踵、靴底、甲、つま先の方向を示す
・足裏を内側へ反転させない
・支持脚を完全に接地させる
・前蹴り、膝蹴り、回し蹴りを混同しない

【コート】
・コートは身体の運動より少し遅れて流れる
・攻撃方向と逆側へ自然になびかせる
・肩、肘、手首、骨盤、膝、足首をコートで完全に隠さない
・前後の脚が判別できるよう、裾の間に空間を残す
・裾を腕、脚、ブーツ、腰の鎖へ融合させない
・回転動作では裾を一つの大きな弧へまとめる
・直線打撃では裾を過剰に広げない

【腰の装飾鎖】
・腰の装飾鎖は短く、衣装へ固定された受動的な装飾として描く
・手に持たせない
・攻撃用の長い鎖へ変化させない
・脚、コート、腕へ融合させない

【顔と視線】
・顔はカメラではなく攻撃の到達方向を見る
・鼻先、顎、両目を攻撃方向へそろえる
・後方構図でも首だけをカメラへ振り返らせない
・前景の拳、肘、膝、足で顔を完全に隠さない

【画像仕様】
・16:9横長
・ノベルゲーム用の戦闘イラスト
・均一な白背景
・床、地平線、接地影、風景を描かない
・敵、標的、他の人物を描かない
・全身、帽子、角、両手、両足、コート裾を画面内へ収める
・攻撃方向へ十分な余白を設ける
・手足を画面端で切らない
・文字、番号、技名、ロゴ、透かしを入れない

【画風】
・高品質な日本のノベルゲーム用アニメイラスト
・繊細で明瞭な線画
・柔らかなセル塗り
・正本と同じ顔、体格、衣装、配色
・実用的な格闘動作と身体のシルエットを優先する

【この段階では使用しないもの】
炎、火花、熱歪み、オーラ、爆発、鎖攻撃、武器、固有技の演出。
身体動作を確認するため、エフェクトを一切描かない。

【禁止事項】
指定番号以外の動作との混合、複数人物、敵、武器、炎、
巨大な角、帽子なし、翼、尻尾、獣耳、長髪、
重量級、ボディビルダー、中年男性、細身の美少年、
中世鎧、軍装、バイクスーツ、
余分な腕や脚、六本指、欠けた指、融合した指、二重の手、
手袋から生える素手、拳から突き出る指、
折れた手首、逆関節、反転した足首、浮いた支持脚、
膝とブーツのつま先の不一致、足とブーツの分離、
カメラ目線、身体から切断して見える拳や足、
コートによる関節と支持脚の完全な遮蔽
```

## 01 左前手直突き

```text
まず01だけを、新しい独立した一枚として生成してください。

マネキン資料の01と同じ正面カメラ。
左脚を前、右脚を後ろに置く。
左拳を相手の中心へ肩から胸の高さで突き出す。
左肘を完全に伸ばし切らない。
右拳は顎の横へ残し、顔を守る。
後ろ足から骨盤、背骨、左肩、左拳へ力をつなげる。
正面からも両脚の前後差を残す。
拳で顔を完全に隠さない。
静4、動6。
```

## 生成後チェック

- [ ] マネキンと同じ動作、支持脚、重心、カメラ方向か。
- [ ] 8枚が独立画像として生成されているか。
- [ ] 顔、体格、帽子、短い角、衣装が統一されているか。
- [ ] 攻撃していない手が防御位置へ残っているか。
- [ ] 拳、掌、肘、膝、ブーツの接触部位が明確か。
- [ ] 手袋と拳または掌の構造が成立しているか。
- [ ] 支持脚と蹴り脚の両方にブーツがあるか。
- [ ] コートで関節、骨盤、支持脚が隠れていないか。
- [ ] 顔と視線が攻撃方向を向いているか。
- [ ] 炎、武器、敵、背景が混入していないか。

## 次段階

身体動作が承認された画像だけを、固有技へ発展させる。

```text
固有技 = 承認済み身体動作 + 遠近調整 + 炎または鎖 + 技固有の演出
```

固有技化するときも、支持脚、重心、関節、接触部位は変更しない。
