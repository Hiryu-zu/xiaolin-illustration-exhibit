# マネキン構図強化 共通ブロック

アクションポーズ資料の観察をもとに、白い3Dデッサン人形プロンプトへ追加する共通ブロック。
小鈴、リーファン、牙焔など、キャラを問わず使える。

## 01 構図補助

```text
【構図補助】
・薄い床面と接地影を入れ、足裏の位置と重心を確認できるようにする。
・全パネルでアイレベル、床の高さ、カメラ距離、人物サイズを統一する。
・遠近がある動作では、画面内にアイレベルを想定し、攻撃方向または身体の奥行き方向に消失点を置く。
・前景の手足だけを巨大化させず、肩、肘、股関節、膝との接続を残す。
・床面上の足位置、骨盤、胸郭、頭部の奥行き関係が読めるようにする。
・全身、両手、両足を画面内へ収め、手足を画面端やパネル境界で切らない。
```

## 02 重心と力線

```text
【重心と力線】
・支持脚を床へ完全に接地させる。
・支持脚の膝を軽く緩める。
・重心線が支持脚または両足の間へ落ちるようにする。
・攻撃動作は、足、膝、骨盤、背骨、肩、肘、手先または足先まで一本の力線としてつなげる。
・上体だけ、腕だけ、脚だけの動作にしない。
・骨盤、胸郭、頭部の向きをそれぞれ指定し、身体のひねりを読めるようにする。
```

## 03 顔と視線

```text
【顔と視線】
・顔と視線はカメラではなく、攻撃または受け流しの方向を見る。
・鼻先、顎、両目を技の到達方向へそろえる。
・背面構図でも首だけを不自然に振り返らせない。
・回転動作では、顔と視線を身体の回転より少し先行させる。
```

## 04 前蹴り

```text
【前蹴り補助】
・支持脚を床へ完全に接地させる。
・蹴り脚の膝を先に引き上げ、踵または足裏で前方を押す。
・蹴り脚の股関節、膝、足首、踵を同じ進行面へ置く。
・上体は蹴り方向と反対へわずかに傾けるが、反りすぎない。
・両手は顔または身体の中心線を守る。
・横蹴りや回し蹴りへ変化させない。
```

## 05 横蹴り・ミドルキック

```text
【横蹴り・ミドルキック補助】
・軸足を床へ完全に接地させ、足裏とつま先の方向を明確にする。
・蹴り脚の股関節を開き、膝、脛、足先を同じ回転面へ置く。
・上体を蹴り脚と反対側へ自然に倒し、重心を軸足上へ残す。
・片手は顔を守り、もう片手は回転またはバランスを取る位置へ置く。
・完全な水平開脚や舞踊的な姿勢にしない。
・足裏を内側へ反転させない。
```

## 06 直突き

```text
【直突き補助】
・後ろ足で床を押す。
・後ろ足、膝、骨盤、背骨、肩、肘、拳を一本の力線へつなげる。
・突く腕の肘を完全に伸ばし切らない。
・引き手は顔または胸の近くへ残す。
・肩をすくめない。
・拳で顔を完全に隠さない。
```

## 07 回転動作

```text
【回転動作補助】
・軸足を明確にし、床へ接地させる。
・軸足、骨盤、胸郭、肩、肘、手先または足先を同じ回転方向へつなげる。
・背面構図では、肩甲骨、背骨、骨盤の回転が読めるようにする。
・顔と視線は回転方向へ少し先行させる。
・首だけをカメラへ振り返らせない。
・腕、胴体、脚が重なりすぎる場合は、間に小さな空間を残す。
```

## 08 八方向構え

```text
【八方向構え補助】
・8種類の異なる構えではなく、一つの同じ構えを固定し、カメラだけを45度ずつ移動させる。
・前後脚を途中で入れ替えない。
・足幅、膝の曲げ、骨盤の向き、胸郭の向き、手の高さを全パネルで維持する。
・背面カメラでも首をカメラへ向けない。
・全パネルで床面、接地影、アイレベル、人物サイズを統一する。
```
