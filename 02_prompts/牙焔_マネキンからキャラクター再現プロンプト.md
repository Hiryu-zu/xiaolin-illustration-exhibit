# 牙焔 マネキンからキャラクター再現プロンプト

生成済みの牙焔用マネキンと牙焔の正本イラストを使用し、体格と姿勢を維持したままキャラクター化するためのプロンプト。

生成時には次の画像を添付する。

2. 牙焔用の体格マネキン四面図

### 基本戦闘姿勢

参考画像ごとの役割を混同させない。

## 01 牙焔 四面立ち絵

```text
添付した参考画像を使用し、牙焔の全身四面立ち絵を作成してください。

【参考画像の役割】
参考画像1：牙焔の正本イラスト
・顔、見た目年齢、髪型、目、帽子、角、衣装、手袋、ブーツ、配色の正本
・キャラクターの外見は必ずこちらを優先する

参考画像2：白い3D体格マネキン四面図
・身長、頭身、肩幅、胸郭、胴回り、骨盤、腕と脚の長さ、四方向の身体形状の正本
・マネキンの顔、材質、白色、関節線は引き継がない

【出力形式】
・一枚の横長キャラクター設定シート
・横4列、縦1行の均等な4パネル
・左から正面、右側面、背面、左側面
・各パネルに牙焔を一人ずつ配置する
・全4体は同一人物、同じ直立姿勢、同じ表情
・身長、頭身、肩幅、胴回り、手足の長さを統一する
・人物サイズ、床の高さ、カメラ距離をそろえる
・頭頂、帽子、角、コート裾、ブーツを完全に画面内へ収める
・人物同士を重ねない
・薄い境界線でパネルを区切る
・文字、方向名、番号、寸法線は入れない

【牙焔の固定外見】
・20代前半に見える成人男性
・若いが幼くない、荒々しさのある無愛想な青年
・高身長だが重量級ではない
・自然なアニメ調約7頭身
・中肉中背、標準からやや引き締まった体格
・肩幅は標準からやや広め
・胸板、腕、太腿を過剰に太くしない
・胴回りを太くしない
・細身の美少年や中性的な体格にしない
・30代以上の中年顔、深い皺、重い顎にしない

・黒い短髪
・前髪が片目へ自然にかかる
・鋭い赤い目
・不機嫌で静かな表情
・目深に被った黒い帽子
・帽子から短い鬼角が二本だけ突き出る
・角を巨大化させない
・帽子を消さない

【衣装】
・黒基調の学ラン風特攻服
・深紅のラインとパイピング
・縦長でスマートなシルエット
・長いロングコートの前後の裾
・黒い手袋
・黒いブーツ
・金属装飾は正本の範囲に限定する
・中世鎧、軍用装備、バイクスーツへ変更しない
・翼、尻尾、獣耳を追加しない

【四方向の整合性】
・正面、側面、背面で帽子と角の位置を一致させる
・左右側面で顔、鼻、顎、後頭部の形状を一致させる
・コートの襟、肩、袖、腰、裾の構造を四方向で統一する
・正面と背面で肩幅、腰幅、脚幅を一致させる
・手袋とブーツの形状を四方向で統一する
・側面と背面でも前髪と後ろ髪の長さを維持する
・背面へ正面用のボタンや胸飾りを複製しない
・左右側面を単純な鏡像にせず、髪と衣装の前後関係を正しく描く

【姿勢】
・マネキン四面図と同じ自然な直立姿勢
・頭を水平に保つ
・顎を軽く引く
・両肩を水平にする
・両腕を身体の横へ自然に下ろす
・両脚を腰幅から肩幅程度に開く
・両膝とブーツのつま先を正面へ向ける
・左右の足へ均等に重心を置く
・衣装でマネキンの体格や関節位置を変更しない

【背景と画風】
・均一な白背景
・床、地平線、接地影、風景を描かない
・高品質な日本のアニメ、ノベルゲーム用キャラクター設定画
・繊細で明瞭な線画
・柔らかなセル塗り
・全パネルで同じ描画密度、色、照明
・炎、火花、熱歪みは使用しない

【禁止事項】
異なる衣装、異なる年齢、異なる体格、異なる表情、
巨大な角、帽子なし、長髪、白髪、獣耳、翼、尻尾、
重量級、ボディビルダー、細身の美少年、中年男性、
中世ファンタジー鎧、現代軍装、バイクスーツ、
武器、炎、エフェクト、敵、背景風景、床、影、
余分な腕や脚、六本指、融合した指、左右非対称な身体、
文字、方向名、番号、ロゴ、透かし
```

## 02 牙焔 基本格闘構え 八方向

八方向は一枚のシートではなく、構図精度を優先して独立画像として生成する。

```text
添付した参考画像を使用し、牙焔が基本格闘構えを取った姿を、八方向から8枚の独立した画像として順番に作成してください。

【参考画像の役割】
参考画像1：牙焔の正本イラスト
・顔、年齢、髪型、目、帽子、角、衣装、手袋、ブーツ、配色の正本

参考画像2：牙焔用の白い3D体格マネキン四面図
・身長、頭身、肩幅、胴回り、骨盤、手足の長さの正本

参考画像3：基本格闘構えの八方向マネキン
・各番号のカメラ方向、支持脚、足幅、重心、拳の高さ、肘の角度、胸郭と骨盤の向きの正本
・マネキンの顔、材質、白色、関節線は引き継がない

【最重要：出力方法】
八方向を、必ず8枚の独立した画像として一枚ずつ生成する。

1枚目：01 正面
2枚目：02 右前方45度
3枚目：03 右側面
4枚目：04 右後方45度
5枚目：05 背面
6枚目：06 左後方45度
7枚目：07 左側面
8枚目：08 左前方45度

一枚に複数方向を並べない。
モデルシート、グリッド、分割パネル、一覧表、コラージュにしない。
一枚につき牙焔一人、一つの基本構えだけを描く。
各画像を単独で使用できる完成イラストにする。

【ポーズの統一】
8種類の異なるポーズを作らない。
一つの同じ基本格闘構えを固定し、カメラだけを人物の周囲へ45度ずつ移動させる。
全8枚で左脚が前、右脚が後ろ。
支持脚、足幅、拳の高さ、肘の角度、重心、胸郭と骨盤の向きを維持する。
左右反転や手足の入れ替えを行わない。

【基本格闘構え】
・右利きの半身姿勢
・左脚を前、右脚を後ろ
・足幅は肩幅より少し広い
・前後の足を一直線へ重ねない
・左足は前方、右足は斜め外側へ向ける
・両膝を軽く曲げる
・腰を落とし、低い重心を作る
・両足裏を完全に接地する

・骨盤は斜め
・胸郭は骨盤よりわずかに正面へ戻す
・背骨を自然に立てる
・上体を倒しすぎない
・肩を上げない
・顎を軽く引く

・左拳を前方、顎から胸の高さ
・右拳を頬から肋骨付近
・両肘を軽く曲げる
・両脇を開きすぎない
・手首を前腕の延長上へ置く
・拳の親指を四本の指の外側へ添える
・顔と視線は構えの正面方向を見る
・カメラ目線にしない

【牙焔の固定外見】
・20代前半の成人男性
・高身長だが重量級ではない
・中肉中背、標準からやや引き締まった体格
・黒い短髪、片目へかかる前髪
・鋭い赤い目
・不機嫌で集中した表情
・目深に被った黒い帽子
・帽子から突き出る短い鬼角二本
・黒基調の学ラン風特攻服
・深紅のライン
・黒い手袋とブーツ
・長いロングコート

【コートの扱い】
・コートを身体の動きへ自然に合わせる
・肩、肘、手首、骨盤、膝、足首の位置を衣装で変えない
・前後の脚をコート裾で完全に隠さない
・基本構えなので裾を派手になびかせない
・裾は重力に従って静かに垂らす
・側面と背面でもコートの前後構造を維持する
・コート裾を脚や腕へ融合させない

【画像仕様】
・16:9横長
・均一な白背景
・敵、標的、他の人物なし
・全身、帽子、角、両手、両足、コート裾を画面内へ収める
・技が進む正面方向へ適度な余白を取る
・床、地平線、影、背景風景を描かない
・炎、火花、熱歪み、武器を描かない
・画像内に番号、方向名、文字を入れない

【画風】
・高品質な日本のノベルゲーム用アニメイラスト
・繊細で明瞭な線画
・柔らかなセル塗り
・全8枚で顔、体格、衣装、配色、描画密度を統一する
・技を出す前の静かな緊張感
・静7、動3

【身体の禁止事項】
足の入れ替え、左右反転、浮いた足、伸び切った膝、
異なる拳の高さ、異なる足幅、異なる重心、
折れた手首、逆関節、膝と足先の不一致、
余分な腕や脚、六本指、欠けた指、融合した指、
拳から突き出る指、身体から切断して見える手足

【キャラクターの禁止事項】
巨大な角、帽子なし、翼、尻尾、獣耳、長髪、
重量級、ボディビルダー、細身の美少年、中年男性、
中世鎧、軍装、バイクスーツ、剣、銃、
炎、オーラ、爆発、敵、背景風景、文字、ロゴ、透かし
```

## 八方向の実行指示

共通プロンプトを送った後、次の指示で一枚ずつ生成する。

### 01 正面

```text
まず01の正面構図だけを、新しい独立した一枚として生成してください。
マネキン資料の01と同じカメラ方向、支持脚、重心、手足の配置を維持してください。
```

### 02以降

```text
次は02の右前方45度だけを、新しい独立した一枚として生成してください。
ポーズを変更せず、カメラ方向だけをマネキン資料の02へ変更してください。
外見、体格、衣装、足幅、拳の高さ、重心は01と統一してください。
```

番号と方向を入れ替え、08まで繰り返す。

## 生成後チェック

### 四面立ち絵

- [ ] 4体の顔、年齢、体格、衣装が同一か。
- [ ] 帽子と短い角の位置が四方向で一致しているか。
- [ ] 正面と背面の肩幅、腰幅、脚幅が一致しているか。
- [ ] 左右側面の顔、胸郭、骨盤、コートの厚みが一致しているか。
- [ ] コート前面の装飾が背面へ複製されていないか。

### 基本戦闘姿勢

- [ ] 全画像で左脚が前、右脚が後ろか。
- [ ] 足幅、重心、拳の高さ、肘の角度が統一されているか。
- [ ] カメラだけが45度ずつ移動しているか。
- [ ] 顔が構えの正面方向を向き、カメラ目線になっていないか。
- [ ] コートで支持脚や関節が隠れていないか。
- [ ] 帽子、短い角、髪、手袋、ブーツが維持されているか。
- [ ] 炎や技の演出が混入していないか。
