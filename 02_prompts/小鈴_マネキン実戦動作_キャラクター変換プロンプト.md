# 小鈴 マネキン実戦動作 キャラクター変換プロンプト

白い3D体格マネキンで作成した実戦基礎動作8種類を、同じ姿勢とカメラ方向のまま小鈴へ変換するためのプロンプト。

## 添付する参考画像

1. 小鈴の正本立ち絵候補。
2. 小鈴の基本武術構え画像。
3. 白い3Dマネキンの実戦基礎動作8種類一覧。

必要に応じて、小鈴の体格マネキン四面図も追加する。

## 添付8ポーズマネキン画像を使う場合

ユーザー添付の白い3Dマネキン8ポーズ画像は、既存の「中華武術8動作セット」とは別枠で扱う。

- 使うもの: 姿勢、カメラ方向、支持脚、重心、骨盤、胸郭、肩、肘、拳、膝、足先。
- 使わないもの: 男性体型、強い筋肉量、坊主頭、白い素材、関節線、番号、グリッド。
- 用途: 単発の一枚絵、直突き、掌打、前蹴り、横蹴り、背面構えの補助。
- 注意: `01構え -> 02途中 -> 03到達` の連続性を確認するメイン資料にはしない。

専用プロンプト:

- `02_prompts/小鈴_添付8ポーズマネキン_シャオリン化プロンプト.md`

## 共通変換プロンプト

```text
添付した参考画像を使用し、白い3Dマネキンの実戦基礎動作01から08を、小鈴の戦闘イラストとして再現してください。

【参考画像の役割】
参考画像1：小鈴の正本立ち絵候補
・顔、見た目年齢、髪型、目、衣装、レギンス、靴、配色の正本
・キャラクターの外見は必ずこちらを優先する

参考画像2：小鈴の基本武術構え画像
・戦闘時の顔、体格、衣装の見え方、軽やかな重心の補助資料

参考画像3：白い3Dマネキンの実戦基礎動作8種類
・指定番号の姿勢、カメラ方向、支持脚、重心、骨盤、胸郭、手足の前後関係、接触部位の正本
・マネキンの顔、白い材質、関節線、簡略化された手足は引き継がない
・指定番号以外の動作を混合しない

【最重要：出力方法】
01から08を、必ず8枚の独立した画像として一枚ずつ順番に生成する。

1枚目：01 弓歩直突き
2枚目：02 穿掌
3枚目：03 前蹴り
4枚目：04 横蹴り
5枚目：05 円掌受け
6枚目：06 双掌押し
7枚目：07 退歩引化
8枚目：08 旋身換歩

一枚に複数動作を並べない。
モデルシート、グリッド、分割パネル、一覧表、コラージュにしない。
一枚につき小鈴一人、一つの動作だけを描く。
各画像を単独で使用できる完成イラストにする。

【ポーズの再現】
・指定番号のマネキンと同じカメラ方向を維持する
・指定番号の支持脚、足幅、重心を維持する
・骨盤と胸郭の向きを維持する
・肩、肘、手首、拳または掌の位置を維持する
・股関節、膝、足首、接触部位を維持する
・攻撃していない手の防御位置を維持する
・顔と視線を攻撃または受け流しの到達方向へ向ける
・衣装の都合で関節や重心を変更しない
・マネキンに解剖学的な小さな問題がある場合のみ、技の意味を変えず自然に修正する

【構図と重心】
・マネキン段階で決めたアイレベル、床面、足元の奥行き関係を維持する
・支持脚を床へ接地させ、重心線が支持脚または両足の間へ落ちるようにする
・攻撃動作は、足、膝、骨盤、背骨、肩、肘、手先または足先まで一本の力線としてつなげる
・前景の拳、掌、足だけを巨大化させず、肩、肘、股関節、膝との接続を残す
・上体だけ、腕だけ、脚だけの動作にしない

【小鈴の固定外見】
・21歳に見える成人女性
・身長165cm、自然なアニメ調約7.3頭身
・しなやかで均整の取れた健康的な武術体型
・細すぎず、筋肉質すぎず、過剰にグラマーにしない
・黒髪ショートボブ、あご下から肩に付かない程度
・前髪あり、両サイドに顔を縁取る毛束
・琥珀色から黄褐色の瞳
・可愛さと凛々しさの中間の顔立ち
・戦闘時はやや鋭く集中した目つき
・冷酷な悪役顔や幼い丸目にしない

【衣装】
・白い立ち襟の中華武術トップス
・前中心に黒いチャイナボタンまたは組紐留め
・黒い縁取り
・肩、脇、体側に黒い切り替えパネル
・七分袖から長袖寄りの袖、袖口は黒い縁取り
・腰から前へ垂れる白い前垂れ、裾は黒い縁取り
・左右に短い黒い腰布またはサイドパネル
・黒いレギンス
・白黒配色の中華武術シューズ
・金色装飾は少量のみ
・メイド服、制服、チャイナドレス、現代スポーツウェアにしない

【手】
・すべての手は五本指
・掌は四本指と親指一本を明確に分ける
・掌の向きと親指の位置を一致させる
・拳は四本の指を内側へ握り、親指を外側へ添える
・手首を折りすぎない
・攻撃していない手は顔または身体の中心線を守る

【蹴りと靴】
・支持脚と蹴り脚の両方に同じ白黒の中華武術シューズを維持する
・裸足、足指、サンダル、ヒール、ブーツを描かない
・蹴り脚の股関節、膝、足首を自然につなげる
・靴の輪郭で踵、靴底、甲、つま先の方向を示す
・足裏を内側へ反転させない
・支持脚を完全に接地させる
・前蹴りと横蹴りを混同しない

【前垂れと腰布】
・前垂れは身体の運動より少し遅れて自然に流れる
・股関節、膝、足首、支持脚を完全に隠さない
・前後の脚が判別できるよう、裾の間に空間を残す
・前垂れを長いスカートやドレスへ変化させない
・回転動作では前垂れと腰布を一つの大きな流れへまとめる
・布を腕、脚、靴へ融合させない

【顔と視線】
・顔はカメラではなく技の到達方向を見る
・鼻先、顎、両目を攻撃または受け流し方向へそろえる
・後方構図でも首だけをカメラへ振り返らせない
・前景の拳、掌、膝、足で顔を完全に隠さない

【画像仕様】
・16:9横長
・ノベルゲーム用の戦闘イラスト
・完全な無地の白背景
・背景は純白に近い単色
・床、地平線、接地影、落ち影、グラデーション、風景を描かない
・敵、標的、他の人物を描かない
・全身、髪、両手、両足、前垂れを画面内へ収める
・攻撃方向へ十分な余白を設ける
・手足を画面端で切らない
・人物の周囲に白い縁取りや発光を付けない
・髪、袖、前垂れ、靴の輪郭が背景と混ざらないよう、線画を明瞭にする
・文字、番号、技名、ロゴ、透かしを入れない

【画風】
・高品質な日本のノベルゲーム用アニメイラスト
・繊細で明瞭な線画
・柔らかなセル塗り
・正本と同じ顔、体格、衣装、配色
・実用的な中華武術動作と身体のシルエットを優先する

【この段階では使用しないもの】
派手な気功、光、花びら、武器、固有技の演出。
身体動作を確認するため、エフェクトを一切描かない。

【禁止事項】
指定番号以外の動作との混合、複数人物、敵、武器、派手なエフェクト、
ロングヘア、ツインテール、ポニーテール、髪色変更、
子供体型、過剰なグラマー体型、ボディビルダー体型、
メイド服、制服、チャイナドレス、ドレス、現代スポーツウェア、
裸足、サンダル、ヒール、ブーツ、素足化、
余分な腕や脚、六本指、欠けた指、融合した指、二重の手、
拳から突き出る指、折れた手首、逆関節、反転した足首、浮いた支持脚、
膝と靴のつま先の不一致、足と靴の分離、
カメラ目線、身体から切断して見える拳や足、
前垂れや腰布による関節と支持脚の完全な遮蔽
```

## 01 弓歩直突き

```text
まず01だけを、新しい独立した一枚として生成してください。

マネキン資料の01と同じ正面カメラ。
左脚を前、右脚を後ろに置く。
前脚を曲げ、後脚で床を押す。
拳を相手の中心へ肩から胸の高さで突き出す。
反対の手は胸の近くへ引き、守りを残す。
正面からも両脚の前後差と骨盤の回転を残す。
拳で顔を完全に隠さない。
静4、動6。
```

## 02 穿掌

```text

前方へ開いた掌を差し込むように突き出す。
指先、手首、肘、肩を一つの線へつなげる。
反対の手は胸の近くで中心線を守る。
掌を停止のジェスチャーにしない。
手首を反らしすぎない。
静3、動7。
```

## 03 前蹴り

```text
次は03だけを、新しい独立した一枚として生成してください。

マネキン資料の03と同じ右側面カメラ。
右脚を支持脚にする。
左膝を先に引き上げ、左靴の踵または靴底で前方を押す。
左股関節、左膝、左足首、靴の踵を同じ進行面へ置く。
右足裏を完全に接地し、右膝を軽く緩める。
上体を立て、両手で中心線を守る。
横蹴りや回し蹴りへ変化させない。
静4、動6。
```

## 04 横蹴り

```text
次は04だけを、新しい独立した一枚として生成してください。

マネキン資料の04と同じ右後方45度カメラ。
軸足を接地し、蹴り脚を横方向へ伸ばす。
蹴り足の踵または足刀を接触部位にする。
骨盤を横へ開き、膝と靴のつま先を同じ方向へそろえる。
上体を倒しすぎず、両手は守りとバランスに使う。
舞踊的な大開脚にしない。
静3、動7。
```

## 05 円掌受け

```text

片方の掌で円を描くように相手の力を受け流す。
肩甲骨、背骨、骨盤の回転が読める構図にする。
反対の手は胸または顔の近くへ残す。
首だけをカメラへ振り返らせない。
視線は受け流す方向へ向ける。
```

## 06 双掌押し

```text

マネキン資料の06と同じ左後方45度カメラ。
両掌を前方へ押し出す。
左右の掌に自然な前後差と遠近差を付ける。
両肘を伸ばし切らない。
後足から骨盤、背骨、両肩、両掌へ力をつなげる。

静3、動7。
```

## 07 退歩引化

```text
次は07だけを、新しい独立した一枚として生成してください。

マネキン資料の07と同じ左側面カメラ。
相手の力を引き込みながら一歩後退する動作。
後脚へ重心を移し、前脚を軽く引く。
両手は相手の力を下または斜め後ろへ導くように配置する。
前進攻撃に見せない。
背骨を保ち、腰だけを引かない。
静6、動4。
```

## 08 旋身換歩

```text
次は08だけを、新しい独立した一枚として生成してください。

マネキン資料の08と同じ左前方45度カメラ。
軸足を接地し、身体を回転させながら足位置を入れ替える途中。
骨盤、胸郭、肩、顔と視線を同じ回転方向へつなげる。
片手は守り、片手は次の攻撃または受けへ移る位置。
前垂れと腰布は身体の回転より少し遅れて自然に流す。
跳躍や舞踊ポーズにしない。
静3、動7。
```

## 生成後チェック

- [ ] マネキンと同じ動作、支持脚、重心、カメラ方向か。
- [ ] 8枚が独立画像として生成されているか。
- [ ] 顔、体格、髪、衣装、レギンス、靴が統一されているか。
- [ ] 攻撃していない手が防御位置へ残っているか。
- [ ] 掌、拳、踵、足刀の接触部位が明確か。
- [ ] 支持脚と蹴り脚の両方に白黒の中華武術シューズがあるか。
- [ ] 前垂れや腰布で関節、骨盤、支持脚が隠れていないか。
- [ ] 顔と視線が技の方向を向いているか。
- [ ] 派手なエフェクト、武器、敵、背景が混入していないか。
