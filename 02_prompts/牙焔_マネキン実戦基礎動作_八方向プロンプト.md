# 牙焔 マネキン実戦基礎動作 八方向プロンプト

牙焔をキャラクター化する前に、白い3Dデッサン人形で実戦的な打撃と蹴りの構造を確認するためのプロンプト。

## 使用する参考画像

添付する画像:

1. 承認済みの牙焔用体格マネキン四面図。
2. 必要に応じて、牙焔の基本格闘構えマネキン。

参考画像は体格、頭身、関節位置の正本として使用する。牙焔の衣装や炎はまだ使用しない。

## 一括生成プロンプト

```text
添付した牙焔用の白い3D体格マネキンを使い、実戦的な基本打撃と蹴り8種類を、一枚のポーズ資料として作成してください。

【目的】
牙焔の完成イラストや固有技を制作する前に、格闘動作の支持脚、重心、関節、身体の回転、カメラ方向を確認するための資料です。
決めポーズではなく、攻撃が相手へ届く直前または途中の実用的な動作を描いてください。

【出力形式】
・一枚の16:9横長モデルシート
・上段4パネル、下段4パネル
・上段左から01、02、03、04
・下段左から05、06、07、08
・各パネルに白い3Dデッサン人形を一体だけ配置
・一体につき一つの動作だけ
・人物同士を重ねない
・各パネル内に頭頂、両手、両足を完全に収める
・手足をパネル境界で切らない
・全パネルで同じ体格、頭身、関節構造
・細い境界線でパネルを区切る
・各パネル左上に小さく01から08の番号だけを表示
・技名、説明文、方向名は画像内へ入れない

【マネキン】
・承認済みの牙焔用体格マネキンと同じ成人男性型
・20代前半相当
・自然なアニメ調約7頭身
・高身長だが重量級ではない
・中肉中背、標準からやや引き締まった体格
・肩幅は標準からやや広め
・胸板、腕、胴、太腿を過剰に太くしない
・白または明るい灰色のマット樹脂素材
・頭部、胸郭、骨盤に薄い中心線
・肩、肘、手首、股関節、膝、足首を明確にする
・手は五本指を簡潔かつ明瞭に分ける
・両足は閉じた格闘靴型の簡略形状

【全動作共通】
・右利きを基本とする
・左脚を前、右脚を後ろにした構えを出発点とする
・攻撃の力を、接地足から骨盤、背骨、肩または股関節へつなげる
・支持脚を明確にする
・攻撃していない手は顔または身体の中心線を守る
・顔と視線はカメラではなく攻撃の到達方向を見る
・上体だけで攻撃せず、脚と骨盤を使う
・人体として実行可能な関節角度にする

【拳】
・四本の指を内側へ握る
・親指を四本の指の外側へ添える
・拳から指を突き出さない
・手首を前腕の延長上へ置く

【開いた掌】
・四本指と親指一本の合計五本
・掌の向きと親指の位置を一致させる
・指を極端に開かない
・手首を折りすぎない

【蹴り】
・支持脚を床へ完全に接地させる
・支持脚の膝を軽く緩める
・蹴り脚の股関節、膝、足首を自然につなげる
・膝と足先を技として正しい方向へそろえる
・足裏を内側へ反転させない
・前蹴り、膝蹴り、回し蹴りを混同しない

【背景】
・各パネルは均一な薄いグレー背景
・水平な薄い床面
・弱い接地影
・柔らかなスタジオ照明
・クリーンな3D人体デッサン教材風

【01 正面：左前手直突き】
カメラは人物の正面。
左脚を前、右脚を後ろに置く。
左拳を相手の中心へ肩から胸の高さで突き出す。
左肘を完全に伸ばし切らない。
右拳は顎の横へ残す。
正面からも両脚の前後差と骨盤の回転が分かるようにする。

【02 右前方45度：右後手直突き】
カメラは人物の右前方45度。
右拳を前方へ強く突き出す。
右後足、右膝、骨盤、背骨、右肩、右拳を一つの螺旋へつなげる。
左拳は顔を守る。
拳を前景へ出すが、肩と肘の接続を残す。

【03 右側面：左踏み込み掌底】
カメラは人物の右側面。
左足を前方へ踏み込んで接地した直後。
左掌根を相手の胸または顎の高さへ押し込む。
左指先は上向きまたは斜め上。
右拳は胸の近くへ残す。
側面から歩幅、骨盤移動、背骨、肩、掌底への力線を見せる。

【04 右後方45度：右フック】
カメラは人物の右後方45度。
右拳で顔の高さへの横鉤突きを放つ。
右肘を約90度に曲げる。
拳、肘、肩を同じ水平面へ置く。
軸足、骨盤、背骨、右肩を同じ回転方向へそろえる。
左拳は顔を守る。

【05 背面：左後ろ肘】
カメラは人物の真後ろ。
左後方へ左肘を短く鋭く打ち込む。
左拳を顔の近くへ畳み、左肘を接触点にする。
右拳は身体の中心線を守る。
背面から肩甲骨、背骨、骨盤、支持脚の連動を見せる。
首だけをカメラへ振り返らせない。

【06 左後方45度：右膝蹴り】
カメラは人物の左後方45度。
左脚を支持脚にする。
右膝を相手の腹部へ突き上げる。
右足は膝の下へ自然に畳む。
左足裏を完全に接地する。
骨盤を前へ押し出すが、上体を反らしすぎない。
両拳は顔の近くへ残す。

【07 左側面：左前蹴り】
カメラは人物の左側面。
右脚を支持脚にする。
左膝を先に引き上げ、左足の踵または足裏で前方を押す。
左股関節、膝、足首、踵を同じ進行面へ置く。
上体を立て、両拳で中心線を守る。
横蹴りや回し蹴りにしない。

【08 左前方45度：右中段回し蹴り】
カメラは人物の左前方45度。
左脚を軸足として回す。
右脚で相手の胴体へ中段回し蹴りを放つ。
右膝、右脛、足の甲を同じ回転面へ置く。
左軸足、骨盤、胸郭、右脚を同じ回転方向へそろえる。
蹴り脚の足裏を内側へ反転させない。
左拳を顔の近くへ残す。
完全な水平開脚や舞踊的な姿勢にしない。

【禁止事項】
髪、帽子、角、衣装、コート、手袋、炎、火花、オーラ、鎖、武器、敵、
キャラクターの顔や配色、完成イラスト風の衣装、
8体すべて同じポーズ、同じカメラ方向、方向の重複、
決めポーズ、威嚇姿勢、非実用的な大開脚、
余分な腕や脚、六本指、欠けた指、融合した指、
二重の手、折れた手首、逆関節、反転した足首、
浮いた支持脚、膝と足先の不一致、身体から切断して見える手足、
番号以外の文字、技名、ロゴ、透かし
```

## 生成後チェック

- [ ] 8体が同じ体格と頭身か。
- [ ] 8方向のカメラが重複していないか。
- [ ] 正面、背面、左右側面、四つの斜め方向があるか。
- [ ] 各動作の支持脚と重心が成立しているか。
- [ ] 攻撃していない手が防御へ残っているか。
- [ ] 拳、掌、肘、膝、足の接触部位が明確か。
- [ ] 肩、肘、手首が自然につながっているか。
- [ ] 股関節、膝、足首が自然につながっているか。
- [ ] 手は五本指で、親指の位置が正しいか。
- [ ] 前蹴りと回し蹴りが明確に区別されているか。
- [ ] 完成キャラクターの衣装や炎が混入していないか。

```text
16:9横長、全身、薄いグレー背景、床面と接地影。
他番号の動作を混合しない。
修正点: <具体的な修正内容>
```
